1. User A requests to run a transaction on the Polkadot ledger.
2. The API Client forwards the request to the API Server.
3. The API Server initiates the transaction on the Polkadot ledger through the Ledger Connector.
4. The Polkadot Ledger Connector processes the transaction request.
5. The API Server receives the response from the Polkadot Ledger Connector.
6. The API Server formats the response.
7. The API Client sends the formatted response back to User A.
