1. Waiter selects order to review.
2. Interface requests order details from Order Management System.
3. Order Management System fetches order items from Menu Database.
4. Menu Database returns order items to Order Management System.
5. Order Management System requests allergy warnings for items from Allergy Information System.
6. Allergy Information System looks up allergy info in Menu Database.
7. Menu Database returns allergy data to Allergy Information System.
8. Allergy Information System compiles allergy warnings.
9. Order Management System sends compiled warnings to Interface.
10. Interface displays allergy warnings to Waiter.
