1. Content Provider initiates submission.
2. Document Source collects document(s) and metadata.
3. Document Source submits document(s) and metadata (ITI-41) to Document Repository.
4. Document Repository processes the request.
5. Document Repository stores document(s).
6. Document Repository submits metadata (ITI-42) to Document Registry.
7. Document Registry validates metadata.
8. Document Registry sends an error response (ITI-42) to Document Repository.
9. Document Repository sends an error response (ITI-41) to Document Source.
