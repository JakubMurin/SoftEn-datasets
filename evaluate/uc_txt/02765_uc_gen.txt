1. Navigate to the Create Operation Request page.
2. Fill in the operation request information, including Patient ID, Doctor ID, Operation Type ID, Deadline, and Priority.
3. Submit the form.
4. Handle the form submission. (handleSubmit).
5. Create an operation request using the entered data.
6. Obtain authentication token.
7. Check the Doctor's role using the token.
8. Make a POST request to /api/OperationRequest/create-operation-request.
9. Receive the OperationRequestDto from the API.
10. Set success message.
11. Clear the form.
