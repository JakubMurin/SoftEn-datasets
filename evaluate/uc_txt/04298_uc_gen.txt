1. User navigates to 'Submit Session'.
2. Web Interface displays 'Session Submission Form'.
3. User fills in and submits the form.
4. Session details are saved in the database.
5. Session review process is triggered.
6. Session status: Under Review.
7. Notification to user is triggered.
8. User receives email notification: 'Session Submitted'.
9. User refreshes/checks email.
10. Session status updated: Approved/Rejected.
11. Notification to user triggered: 'Session Evaluated'.
12. User receives email notification: 'Session Approved' or 'Session Rejected'.
