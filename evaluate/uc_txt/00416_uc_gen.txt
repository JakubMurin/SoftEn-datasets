1. [Developer] Requests dataset instance upgrade information.
2. [Management System] Queries available dataset type versions.
3. [Version Control] Returns available versions.
4. [Management System] Displays versions with upgrade impacts.
5. [Developer] Selects new version for upgrade.
6. [Management System] Initiates upgrade process for dataset instance.
7. [Version Control] Checks application dependencies.
8. [AppDeployment] Confirms compatibility.
9. [Version Control] Prepares dataset instance upgrade.
10. [DB] Upgrades dataset instance to new version.
11. [Version Control] Updates pipeline configuration.
12. [Pipeline] Validated new dataset version compatibility in configuration.
13. [Management System] Notifies upgrade completion status to Developer.
14. [Developer] Receives upgrade success notification.
