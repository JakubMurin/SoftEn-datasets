1. Funder requests security details for specific project from DMPTool.
2. DMPTool assesses security measures and standards.
3. Security Assessment Module fetches relevant data security protocols from Project Data.
4. Project Data returns security protocols to Security Assessment Module.
5. Security Assessment Module verifies compliance with legal standards using Compliance Database.
6. Compliance Database returns compliance status to Security Assessment Module.
7. Report Generator compiles security assessment report.
8. Report Generator generates and formats the report.
9. DMPTool provides the report to the Funder on data security and compliance.
