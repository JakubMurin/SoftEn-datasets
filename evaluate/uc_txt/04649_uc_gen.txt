1. Administrator requests statistics from the UI.
2. UI fetches statistics from Backend System.
3. Backend System retrieves comprehensive stats from Statistics Module.
4. Statistics Module queries faculty work data from the Database.
5. Database returns queried data to Statistics Module.
6. Statistics Module processes data into statistics for Backend System.
7. Backend System generates digestible report using Report Generation Service.
8. Report Generation Service returns generated report to Backend System.
9. Backend System displays the report on UI.
10. UI shows comprehensive tracking statistics to the Administrator.
