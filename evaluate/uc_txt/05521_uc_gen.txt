1. Curator accesses the item/component page.
2. UI validates access.
3. Authorization is granted. UI accesses item/component page.
4. UI requests item/component details.
5. Retrieval Module queries the database for details and deletion date.
6. Database returns the requested data.
7. Retrieval Module displays item/component details and deletion date.
8. UI shows item/component page with deletion date to the Curator.
