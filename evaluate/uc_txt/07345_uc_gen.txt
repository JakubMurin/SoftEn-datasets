1. Depositor accesses the DOI Information Page.
2. Web Interface requests DOI Information.
3. Backend Server queries DOI Assignment Process in DOI Information Database.
4. DOI Information Database returns DOI Process Information to Backend Server.
5. Backend Server requests Required Metadata Fields from Metadata Service.
6. Metadata Service returns Metadata Fields to Backend Server.
7. Backend Server responds with DOI and Metadata Info to Web Interface.
8. Web Interface displays DOI and Metadata Information to Depositor.
