1. Entrant attends event by sending an Attend Event Request to Ticket.
2. Ticket receives the Attend Event Request and activates.
3. Ticket sends a Create Member Request to Member.
4. Member receives the Create Member Request and activates.
5. Member creates a new Member.
6. Member sends a Create Member Response to Ticket.
7. Ticket receives the Create Member Response and deactivates Member.
8. Ticket creates a Ticket.
9. Ticket sends an Attend Event Response to Entrant.
10. Entrant receives the Attend Event Response and deactivates Ticket.
