1. User post driver information {name, csp, driver} to Spider.
2. Spider receives driver information.
3. Spider stores driver information in spider db.
4. Spider responds with driver information to User.
5. User post region information {name, csp, region, zone} to Spider.
6. Spider receives region information.
7. Spider stores region information in spider db.
8. Spider responds with region information to User.
9. User post credential information {name, csp, credentials} to Spider.
10. Spider receives credential information.
11. Spider stores credential information in spider db.
12. Spider responds with credential information to User.
13. User post connection configuration {name, csp, driver, credential, region} to Spider.
14. Spider receives connection configuration.
15. Spider stores connection configuration in spider db.
16. Spider responds with connection configuration to User.
17. User post namespace information {name} to Tumblebug.
18. Tumblebug receives namespace information.
19. Tumblebug stores namespace information in tumblebug db.
20. Tumblebug responds with namespace information to User.
