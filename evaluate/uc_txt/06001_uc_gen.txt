1. User selects recyclable waste types.
2. Frontend application requests facilities for waste types.
3. Backend server queries facilities by waste type.
4. Database returns a list of facilities.
5. Backend server sends facilities and opening hours.
6. Frontend application displays facilities and opening hours to the user.
7. User requests optimal route and schedule.
8. Frontend application calculates optimal route based on user location and facility hours.
9. Backend server returns optimal route and schedule.
10. Frontend application displays optimal route and schedule to the user.
