1. User asks for a nickname.
2. User writes the nickname.
3. Client sends the join request with nickname, IP, and port to the server.
4. Server adds the nickname and IP to the chat room.
5. Server notifies the client that the user has joined.
6. Server updates the chat with the joined message.
7. Client enables the "Ready to Play" button.
8. Client enables the chat interface.
9. User acknowledges the successful join.
