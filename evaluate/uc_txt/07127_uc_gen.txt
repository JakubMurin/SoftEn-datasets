1. Attendee requests summary.
2. Workshop Platform fetches request for summary.
3. CMS queries workshop data from Database.
4. Database returns workshop data to CMS.
5. CMS generates summary content with Content Editor.
6. Content Editor submits summary to CMS.
7. CMS updates Database with summary.
8. Database confirms update to CMS.
9. CMS publishes summary to Summary Page.
10. Summary Page displays summary content to Attendee.
