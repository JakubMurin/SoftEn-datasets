1. Alice initiates adding process.
2. Edge Agent 2 provides instructions to Alice for adding the agent and selecting authorizations.
3. Alice adds agent with the selected authorizations.
4. Edge Agent 1 displays a provision code for adding Agent 2.
5. Alice shows the code to Agent 2.
6. Agent 2 sends add request and necessary data to Agent 1.
7. Agent 1 stores data in the wallet.
8. Alice confirms the addition of Agent 2.
9. Agent 1 adds keys and authorizations for Agent 2 to the policy registry.
10. Data and token sync between Agent 1 and Agent 2.
11. Agent 2 stores the data in the wallet.
