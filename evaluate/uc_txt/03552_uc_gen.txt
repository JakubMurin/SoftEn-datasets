1. Donor Representative accesses the website.
2. User Interface requests access from the Authorization System.
3. Authorization System authenticates the Donor Representative.
4. Donor Representative requests image descriptive information.
5. User Interface queries for images from File Management.
6. File Management fetches image metadata from Database.
7. Database returns metadata to File Management.
8. File Management displays descriptive information on User Interface.
9. User Interface shows descriptive information about images to Donor Representative.
