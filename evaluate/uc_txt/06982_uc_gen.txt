1. User searches and selects a restricted object.
2. Interface requests access details from ACS.
3. ACS displays "Contact Curator" option to Interface.
4. User clicks "Contact Curator" on Interface.
5. Interface generates an email template for Email.
6. User fills in and sends the access request via Email.
7. Email forwards the user request to Curator.
8. Curator reviews the request.
9. Curator updates access permissions on ACS if request is approved.
10. ACS notifies user of the decision via Email.
11. Email displays access decision notification to User.
