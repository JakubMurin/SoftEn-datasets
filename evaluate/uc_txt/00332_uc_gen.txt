1. Member selects to share a work.
2. Application retrieves file share form.
3. UIM displays the file share form.
4. Member views the file share form.
5. Application shows the file share form.
6. Member fills in content name, content type, and uploads file.
7. Member adds a file.
8. Member sends the form.
9. Application analyzes the file.
10. Application displays the file analysis result to Member.
11. Application asks for confirmation to send.
12. Member validates the send.
13. Application copies the file to "To moderate" directory.
14. Application generates a transaction number.
15. Application saves the transaction in the log file.
16. Application transmits the transaction and file to the server.
17. Server notifies the Librarian of the submitted work.
18. Application notifies Member of the waiting for moderation.
