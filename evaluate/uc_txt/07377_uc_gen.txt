1. Admin requests to view the list of advertisements.
2. List Advertisements View displays the list of advertisements.
3. Admin clicks on the edit button of an advertisement.
4. Redirects to Edit Advertisement View.
5. Edit Advertisement View shows the content of the advertisement.
6. Admin edits the advertisement data and clicks on save.
7. Sends the information to the AdsController.
8. AdsController checks the data.
9. AdsController saves the data.
10. AdsModel returns the saved data.
11. AdsController provides the result of saving the advertisement.
12. Edit Advertisement View displays the edited advertisement.
