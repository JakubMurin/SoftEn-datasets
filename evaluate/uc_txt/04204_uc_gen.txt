1. User enters credentials and selects "Remember Me" option.
2. Login Page sends credentials to Backend Server for authentication.
3. Backend Server verifies user's credentials with the Database.
4. Database returns the authentication result to Backend Server.
5. Backend Server sends the authentication result to the Login Page.
6. Login Page saves the "Remember Me" token in the Backend Server.
7. Backend Server stores the token in the Database.
8. Database confirms the token is saved in the Backend Server.
9. Backend Server confirms the "Remember Me" option is enabled in the Login Page.
10. Login Page displays the login confirmation to the User.
