1. User clicks 'Comenzar'.
2. App handles the event.
3. StartScreen handles the event.
4. StartButtonEventHandler handles the event.
5. StartScreen sets visible to false.
6. PlayersInputScreen sets visible to true.
7. User selects and confirms inputs.
8. App handles the input event.
9. PlayersInputScreen handles the input event.
10. ConfirmButton handles the input event.
11. ConfirmButton selects and sends data to Game.
12. Game loads received data.
13. PlayersNamesInputScreen is created.
14. App shows screen for player names input.
15. User enters and confirms player names.
16. App handles player names input event.
17. PlayersNamesInputScreen handles player names input event.
18. NamesButton executes player names input event.
19. NamesButton registers user names in Game.
20. Game creates a new player for each entered name.
21. QuestionScreen is created.
22. App shows screen with questions and options.
23. App displays questions for each player on QuestionScreen.
