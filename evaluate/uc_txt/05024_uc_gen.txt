2. Spider adds the driver to the spider database and sends a response to the user.
3. User sends a POST request to Spider to add a region.
4. Spider adds the region to the spider database and sends a response to the user.
5. User sends a POST request to Spider to add a credential.
6. Spider adds the credential to the spider database and sends a response to the user.
7. User sends a POST request to Spider to add a connection config.
8. Spider adds the connection config to the spider database and sends a response to the user.
9. User sends a POST request to Tumblebug to add a namespace.
10. Tumblebug adds the namespace to the tumblebug database and sends a response to the user.
