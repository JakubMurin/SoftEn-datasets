1. El usuario introduce sus credenciales.
2. La aplicación verifica las credenciales.
3. La aplicación solicita un código de verificación al sistema de autenticación.
4. El sistema de autenticación genera un código de verificación.
5. La aplicación solicita al usuario ingresar el código de verificación.
6. El servicio de correo electrónico envía el código de verificación al usuario.
7. El usuario ingresa el código de verificación.
8. La aplicación comprueba que el código de verificación sea correcto.
9. La aplicación notifica al usuario que la autenticación ha sido exitosa.
