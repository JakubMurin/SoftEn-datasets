1. Buyer updates profile information.
2. UI sends a request to update profile information to Web Server.
3. Web Server validates and saves the profile changes.
4. App Server updates the user profile in the Database.
5. App Server receives acknowledgment from Database.
6. App Server informs Web Server about the successful profile update.
7. Web Server updates UI with a confirmation of the profile update.
8. UI confirms to Buyer that the profile has been updated successfully.
