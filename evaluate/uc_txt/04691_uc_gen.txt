1. El usuario inicia sesión en la plataforma.
2. La plataforma verifica las credenciales del usuario.
3. La plataforma confirma el inicio de sesión al usuario.
4. El usuario solicita contenido personalizado.
5. La plataforma analiza el perfil del usuario.
6. El sistema de recomendación obtiene el historial de aprendizaje del usuario.
7. La base de datos envía el historial al sistema de recomendación.
8. El sistema de recomendación envía recomendaciones a la plataforma.
9. La plataforma muestra al usuario el contenido recomendado.
10. El usuario completa una actividad.
11. La plataforma guarda el progreso del usuario en la base de datos.
12. La base de datos confirma el guardado del progreso a la plataforma.
