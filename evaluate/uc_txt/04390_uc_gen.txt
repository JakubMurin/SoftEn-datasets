1. DataScientist requests to create a dataset instance.
2. UI parses request parameters, selects existing dataset type, and defines instance properties.
3. Dataset Management System verifies permissions and dataset type existence.
4. Dataset Management System creates a new dataset instance with specified properties.
5. Dataset Repository confirms the creation of the instance.

7. UI shows a confirmation/error message to DataScientist.
