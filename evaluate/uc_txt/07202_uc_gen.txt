1. Faculty Member navigates to the Repository Dashboard.
2. Repository Dashboard redirects to the Permission Service for permission update page.
3. Permission Service displays the permission update user interface.
4. Repository Dashboard displays the UI to the Faculty Member.
5. Faculty Member pastes a list of NetIDs.
6. Permission Service validates and processes the NetIDs.
7. Permission Service updates permissions for the provided NetIDs in the Repository DB.
8. Permission Service displays a confirmation message to the Repository Dashboard.
9. Repository Dashboard shows a confirmation that the NetIDs have been recognized by the DDR.
