1. User initiates changes configuration and data request.
2. App attempts to reserve device.
3. API relays reservation request to device actor.
4. Device actor relays request to device.
5. Device accepts request and relays port information.
6. Device connects and opens port.
7. Device actor accepts request and relays port information.
8. API relays port information to app.
9. App displays port information.
10. App frees device.
11. Device actor relays free request to device.
12. Device sends free signal.
