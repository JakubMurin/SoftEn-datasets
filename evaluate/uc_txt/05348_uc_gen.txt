1. Support Agent opens the refund page!.
2. UI displays the refund form for the Support Agent.
3. Support Agent fills and submits the refund request form with customer details.
4. Form submission triggers API call to create a refund request.
5. Controller processes the refund request and creates a new refund object.
6. Refund request data is saved in the database.
7. Admin notification service is triggered to notify admin of new refund request.
8. Admin is successfully notified.
9. UI displays success message to the Support Agent.
