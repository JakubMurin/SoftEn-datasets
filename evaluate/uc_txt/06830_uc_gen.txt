1. Designer requests design requirements from the Project Team.
2. Project Team queries User Research for user needs and preferences.
3. User Research provides insights to the Project Team.
4. Project Team requests brand guidelines from Design Guidelines.
5. Design Guidelines provide brand guidelines to the Designer.
6. Designer analyzes user insights and brand guidelines.
7. Designer creates a wireframe draft.
8. Designer submits the wireframe for feedback from the Project Team.
9. Project Team provides feedback to the Designer.
10. Designer revises the wireframe based on feedback.
11. Designer resubmits the revised wireframe to the Project Team.
12. Project Team approves the wireframe.
13. Designer starts design implementation based on the approved wireframe.
