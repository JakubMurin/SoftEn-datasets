1. DonorRepresentative logs in via secure authentication.
2. System validates donor credentials.
3. Display donor dashboard upon successful authentication.
4. DonorRepresentative selects "Supply Accruals" option.
5. System displays form to select collection or create a new one.
6. DonorRepresentative fills form with accrual details and selects an existing collection.
7. System checks for existing collection details.
8. DonorRepresentative confirms the collection to supply accruals.
9. DonorRepresentative uploads files and metadata.
10. System creates/update record for incremental accrual and generates unique identifiers for new files.
11. Accrual is recorded in the database.
12. System displays success message "Accrual added successfully".
13. DonorRepresentative logs out.
