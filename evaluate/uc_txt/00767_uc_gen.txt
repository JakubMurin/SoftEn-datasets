1. Admin requests data for the new user from FrontEnd.
2. FrontEnd sends data for new user to Admin.
3. Admin sends data for new user and token to FrontEnd.
4. FrontEnd sends data for new user and token of Admin to MD.
5. MD checks token of Admin.
6. MD validates data for new user.
7. MD adds new user.
8. MD sends message of success to FrontEnd.
9. FrontEnd shows success message to Admin.
