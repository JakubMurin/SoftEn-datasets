1. Admin requests to log in.
2. UI receives the login request.
3. Router processes the login request.
4. Controller handles the login operation.
5. Repository validates the user's credentials.
6. AdminModel fetches user data by username.
7. Database queries for admin data.
8. AdminModel receives admin data.
9. Repository gets the validation result.
10. Controller sends response to Router.
11. Router sends response to UI.
12. UI displays dashboard or error message based on response.
