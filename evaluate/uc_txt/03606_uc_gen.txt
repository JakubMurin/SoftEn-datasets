1. Le User appelle la méthode main().
2. Le Main initialise le Labyrinthe en chargeant le fichier "labySimple/laby0.txt".
3. Le Labyrinthe lit les données du fichier.
4. Le Labyrinthe crée une instance de Monstre avec les coordonnées (x, y).
5. Le Monstre se place initialement aux coordonnées spécifiées.
6. Le Monstre est retourné au Labyrinthe.
7. Le Labyrinthe est retourné au Main.
8. Le Main affiche le Labyrinthe.
