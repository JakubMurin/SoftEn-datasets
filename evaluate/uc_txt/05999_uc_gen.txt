1. User selects recyclable waste types.
2. Frontend Application requests facilities for waste types.
3. Backend Server queries facilities by waste type.
4. Database returns list of facilities.
5. Backend Server sends facilities and opening hours to Frontend Application.
6. Frontend Application displays facilities and opening hours to User.
7. User requests optimal route and schedule.
8. Frontend Application calculates optimal route based on user location and facility hours.
9. Backend Server returns optimal route and schedule to Frontend Application.
10. Frontend Application displays optimal route and schedule to User.
