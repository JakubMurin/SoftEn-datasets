1. O ator pretende listar pisos de edifícios com passagens para outros edifícios.
2. Container chama API para obter informações sobre pisos com passagens.
3. API responde com o código de status.
4. Em caso de código 200:.
5. Container devolve os pisos com passagens para o ator.
6. Em caso de código 400:.
7. Container devolve uma mensagem de erro para o ator.
