1. Librarian selects the "Late Returns" option.
2. Web Interface queries overdue books from the Database.
3. Database returns the list of overdue books to the Web Interface.
4. Web Interface displays the list of overdue books to the Librarian.
5. Librarian requests to send reminders.
6. Web Interface triggers the process of sending overdue notices via Notification System.
7. Notification System looks up borrower contact information from the Database.
8. Database returns the borrower contact information to the Notification System.
9. Notification System confirms that notifications have been sent to the Librarian.
