1. Archivist signs in to the system.
2. Authentication System verifies credentials and confirms success.
3. Archivist accesses the Upload Interface.
4. Archivist chooses inventory file(s).
5. Archivist provides metadata for the files.
6. Archivist defines access rights for the files.
7. Upload Interface saves metadata and access rights.
8. Archivist uploads the files.
9. File Storage stores files in the archive.
10. Upload Interface updates the inventory list.
11. Archival Collections adds new entries.
12. Database confirms successful update.
13. Upload Interface displays upload success message.
14. Notification System sends notification to donor representatives.
15. Notification System confirms notification sent.
