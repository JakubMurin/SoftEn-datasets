1. Manager sends a request to view product prices.
2. Dashboard receives the request from the Manager.
3. Dashboard initiates a GET request '/products/prices' to Backend.
4. Backend receives the GET request from the Dashboard.
5. Backend activates and fetches prices of products from the Database.
6. Database receives the fetch request from the Backend.
7. Database returns the prices of products to the Backend.
8. Backend sends product prices data to the Dashboard.
9. Dashboard receives the product prices data from the Backend.
10. Dashboard displays the product prices to the Manager.
