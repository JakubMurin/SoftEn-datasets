1. Alice chooses existing or add new device option in Edge Agent Start.
2. Edge Agent 2 provides instructions for adding Agent 2 and selecting authorizations.
3. Alice adds agent with selected authorizations.
4. Edge Agent 1 displays provisioning code for adding Agent 2.
5. Alice shows the code to Edge Agent 2.
6. Edge Agent 2 sends add request and necessary data.
7. Edge Agent 1 stores the verification key in the wallet.
8. Alice confirms Agent 2 addition.
9. Agent 1 adds keys and authorizations for Agent 2 to the policy registry.
10. Data is synchronized to Agent 2.
11. Agent 2 stores the data in the wallet.
