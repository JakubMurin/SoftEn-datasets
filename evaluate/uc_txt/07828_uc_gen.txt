1. Archivist signs in to the system.
2. Authentication system verifies credentials and grants access.
3. Archivist accesses the upload interface.
4. Archivist chooses inventory file(s).
5. Archivist provides metadata.
6. Archivist defines access rights.
7. Upload interface saves metadata and access rights.
8. Archivist uploads file(s).
9. File storage system stores files in the archive.
10. Upload interface updates inventory list.
11. Database adds new entries to the archival collections.
12. Upload success message is displayed to Archivist.
13. Notification system sends notifications to donor representatives.
