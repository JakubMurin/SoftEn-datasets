1. Manager accesses the Menu Management interface.
2. Manager requests the current menu items via the Management Interface.
3. Management Interface sends a request to Backend Server to fetch menu items.
4. Backend Server fetches the menu items data from the Database.
5. Backend Server displays the menu items via the Management Interface.
6. Manager selects items to update.
7. Manager submits the updated menu items.
8. Management Interface sends an update menu items request to Backend Server.
9. Backend Server updates the menu items in the Database.
10. Database confirms the successful update.
11. Backend Server confirms the successful update to the Management Interface.
12. Management Interface displays a success message to the Manager.
