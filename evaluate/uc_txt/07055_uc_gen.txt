1. RepoAdmin requests data synchronization from Symplectic Elements.
2. Symplectic Elements initiates bidirectional communication with Fedora Connector.
3. Fedora Connector verifies access to the hidden Fedora instance and provides confirmation data to Symplectic Elements.
4. Fedora Connector requests object collection from Fedora instance to Hydra.
5. Hydra collects objects from Fedora instance and provides them to Fedora Connector.
6. Fedora Connector updates Symplectic Elements with new and modified objects.
7. Symplectic Elements notifies RepoAdmin about the completed synchronization.
