1. Administrator requests access to a service.
2. Management UI receives access request.
3. Management API processes access request.
4. Inway receives access request.
5. Management API stores access request in the database.
6. Administrator views pending access requests.
7. Management UI retrieves pending access requests.
8. Management API retrieves pending access requests from the database.
9. Management API returns pending access requests to Management UI.
10. Management UI displays pending access requests to Administrator.
11. Administrator approves access request.
12. Management UI processes approval of access request.
13. Inway creates access grant.
14. Management API stores access grant in the database.
15. Administrator views access grants for the service.
16. Management UI retrieves access grants for the service.
17. Inway retrieves access grants for the service.
18. Management API retrieves access grants for the service from the database.
19. Management API returns access grants for the service to Inway.
20. Inway returns access grants for the service to Management API.
21. Management API returns access grants for the service to Management UI.
22. Management UI displays access grants for the service to Administrator.
