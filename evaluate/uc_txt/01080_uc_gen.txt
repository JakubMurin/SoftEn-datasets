1. Moderator navigates to Account Settings.
2. Web Interface receives request to view current account details.
3. Backend Server fetches account details from Account Database.
4. Backend Server displays current account details to Web Interface.
5. Web Interface shows account details form to Moderator.
6. Moderator inputs updated account details.
7. Web Interface submits updated account details to Backend Server.
8. Backend Server updates account details in Account Database.
9. Backend Server acknowledges successful update to Web Interface.
10. Web Interface displays update success message to Moderator.
