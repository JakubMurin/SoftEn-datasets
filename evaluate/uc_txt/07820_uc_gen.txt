1. Alice requests temporary object store credentials from Anonlink.
2. Anonlink creates restricted credentials in the Object Store.
3. Anonlink sends Credentials Response to Alice, who receives temporary object store credentials.
4. Alice uploads Binary Encoding Data to the Object Store.
5. Alice uploads data via "/clks" to Anonlink and receives the Upload Receipt.
6. After some time, Anonlink pulls/copies encoding data from the Object Store.
