1. Archivist inputs keyword.
2. UI sends search request with keyword to Search Controller.
3. Search Controller queries keyword to Indexing Service.
4. Indexing Service fetches matching files from Digital Archive Database.
5. Indexing Service formats matching files.
6. Results Formatter sends formatted results to Search Controller.
7. Search Controller displays search results to UI.
8. UI shows files matching keyword to Archivist.
