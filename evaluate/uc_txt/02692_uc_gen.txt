1. User queries for "www.example.com".
2. Root DNS Server receives the query.
3. Root DNS Server refers the query to TLD DNS Server (".com").
4. TLD DNS Server receives the query.
5. TLD DNS Server refers query to Authoritative DNS Server.
6. Authoritative DNS Server receives the query.
7. Authoritative DNS Server responds with IP address "93.184.215.14".
