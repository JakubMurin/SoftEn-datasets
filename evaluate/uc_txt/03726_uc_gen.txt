1. User sends a GET request to the Fishtext API for text generation.
2. Fishtext API processes the request and returns text.
3. JavaScript displays the generated text on the HTML DOM.
4. User interacts with the game by typing on the keyboard.
5. JavaScript checks the correctness of the input and updates the UI.
6. JavaScript stops the timer and calculates the score.
7. Server receives the game data and updates the database.
8. JavaScript displays end game window and generates speed data for the graph.
9. JavaScript draws the speed graph on the HTML DOM.
10. JavaScript saves the data to the localStorage.
