1. Administrator requests access to a service.
2. Management UI receives access request.
3. Management API processes access request.
4. Inway receives access request.
5. Management API stores access request in Management Database.
6. Administrator views pending access requests.
7. Management UI retrieves pending access requests.
8. Management API retrieves pending access requests from Management Database.
9. Management Database returns pending access requests to Management API.
10. Management API returns pending access requests to Management UI.
11. Administrator approves access request.
12. Management UI approves access request.
13. Inway creates access grant.
14. Management API stores access grant in Management Database.
15. Administrator views access grants for service.
16. Management UI retrieves access grants for service.
17. Inway retrieves access grants for service.
18. Management API retrieves access grants for service from Saas Organization.
19. Management Database returns access grants for service to Management API.
20. Management API returns access grants for service to Management UI.
21. Management UI displays access grants for service.
