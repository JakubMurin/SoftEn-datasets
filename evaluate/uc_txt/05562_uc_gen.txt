1. User configures image resolution for device.
2. Mobile App sets image resolution for device.
3. Mobile App sends device CID.
4. Server finds active TCP connection.
5. Server informs Mobile App about device unavailability.
6. Mobile App displays device unavailability message to User.

8. ESP32 presents configuration.
9. Server sends current device configuration to Mobile App.
10. Mobile App updates view.
11. Mobile App displays successful settings update message to User.
