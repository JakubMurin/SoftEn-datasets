1. El Sistema o Usuario inicia la construccion.
2. El Sistema o Usuario inicia la construccion del elemento construible.
3. La Fase de Construccion verifica si puede costear el elemento construible.
4. El Sistema o Usuario recibe confirmacion de costeo.
5. El Sistema o Usuario inicia una construccion tentativa.
6. La Construccion Tentativa es realizada.
7. El Sistema o Usuario recibe confirmacion de construccion tentativa.
8. El Sistema o Usuario comprueba si la construccion puede realizarse.
9. El Sistema o Usuario recibe un resultado negativo sobre la viabilidad de la construccion.
10. El Sistema o Usuario muestra un mensaje de error al usuario.
11. El Sistema o Usuario intenta mover el centro a una posicion determinada.
12. El Sistema o Usuario intenta mover el centro a una segunda posicion.
13. El Sistema o Usuario verifica si la construccion puede realizarse.
14. El Sistema o Usuario recibe un resultado positivo sobre la viabilidad de la construccion.
15. El Sistema o Usuario procede a construir.
