1. Manager selects "Manage Book Categories" on UI.
2. UI validates Manager role.
3. UI grants access.
4. UI Displays Book Categories for Manager to select.
5. Manager selects categories to delete.
6. UI requests Category Deletion from Book Category Management.
7. Book Category Management deletes Category Records from Database.
8. Database confirms Deletion.
9. Book Category Management displays Success Message on UI.
10. UI Shows Update to Manager.
