1. User searches and selects a restricted object.
2. Interface requests access details from ACS.
3. ACS displays "Contact Curator" option.
4. User clicks "Contact Curator".
5. Interface generates email template.
6. User fills in and sends the access request.
7. Email forwards the user request to Curator.
8. Curator reviews the request.
9. Curator updates access permissions if approved.
10. ACS notifies the user of the decision.
11. Email displays access decision notification to the user.
