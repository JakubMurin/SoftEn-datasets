1. Business User registers as a listener for Fault Management notifications with the Hub.
2. Business User requests SOF to create a Fault Management job.
3. Fault Management Job is created.
4. Fault Management Job state set to inProgress.
5. Fault Management Job generates data.
6. Fault Management Report is created.
7. Fault Management Report state set to inProgress.
8. Fault Management Report collects data.
9. Fault Management Report state set to completed.
10. Fault Management Job state set to completed or scheduled.
11. Business User requests SOF to get the Fault Management report.
12. Fault Management Report is retrieved.
