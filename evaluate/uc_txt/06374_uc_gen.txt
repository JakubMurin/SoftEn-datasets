1. User requests to display all data of an application.
2. Display Application Data UI does Show.
3. Display Application Data Controller fetches all applications.
4. Application Repository returns Iterable of job applications.
5. Display Application Data Controller sends List of job applications to UI.
6. Display Application Data UI displays List of Applications to User.
7. User selects an application.
8. Display Application Data UI sends selected application to Controller.
9. Display Application Data Controller fetches data for selected application.
10. Application Repository returns job application data.
11. Display Application Data Controller sends job application data to UI.
12. Display Application Data UI displays Application Data to User.
