1. Administrator requests statistics.
2. Admin UI fetches statistics from the Backend System.
3. Backend System retrieves comprehensive statistics from the Statistics Module.
4. Statistics Module queries faculty work data from the Database.
5. Database returns the queried data to the Statistics Module.
6. Statistics Module processes the data into statistics.
7. Report Generation Service generates a digestible report.
8. Report Generation Service returns the generated report to the Backend System.
9. Backend System displays the report on the Admin UI.
10. Admin UI shows the comprehensive tracking statistics to the Administrator.
