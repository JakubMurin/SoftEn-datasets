1. User selects "Link Email" on UI interface.
2. UI interface requests to link email.
3. User Profile Service checks if email already exists.
4. If email not found, notify the User Profile Service.
5. User Profile Service links the email to the user account.
6. User Profile Service sends confirmation email.
7. Email Service sends confirmation email to the user.
8. User Profile Service shows success message on UI.
9. UI interface displays "Email linked successfully" to the user.
