1. User requests text generation from Fishtext API.
2. Fishtext API returns text to JavaScript.
3. JavaScript displays the text on the HTML DOM.
4. User presses a key on the keyboard.
5. Keyboard outputs the character to the DOM.
6. JavaScript receives the entered character.
7. JavaScript checks the length of all entered characters.
8. JavaScript checks the correctness of the entered character.
9. JavaScript stops the timer for the attempt.
10. JavaScript calculates the points earned in the game.
11. JavaScript creates an object with game data.
12. JavaScript sends the game data to the Server.
13. Server updates user and attempt data in the Database.
14. JavaScript displays the end game window on the DOM.
15. JavaScript generates speed data for the graph.
16. JavaScript draws the speed diagram on the DOM.
17. JavaScript saves the game data in localStorage.
