1. User sends an HTTP request to the Finance App.
2. Finance App forwards SQL request to the Database.
3. Database responds with SQL data to Finance App.
4. Finance App sends HTTP response back to User.
5. User sends an authentication request to Finance App.
6. If valid credentials, Finance App responds with authentication success to User.
7. If invalid credentials, Finance App responds with authentication failure to User.
8. Finance App sends authentication response to User.
