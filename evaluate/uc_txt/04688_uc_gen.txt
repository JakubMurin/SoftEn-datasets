1. El usuario inicia sesión en la plataforma.
2. La plataforma verifica las credenciales del usuario en la base de datos.
3. La plataforma confirma al usuario el inicio de sesión.
4. El usuario solicita contenido personalizado a la plataforma.
5. La plataforma analiza el perfil del usuario con el Sistema de Recomendación IA.
6. El Sistema de Recomendación IA obtiene el historial de aprendizaje del usuario en la Base de Datos.
7. La Base de Datos envía el historial al Sistema de Recomendación IA.
8. El Sistema de Recomendación IA envía recomendaciones a la plataforma.
9. La plataforma muestra al usuario el contenido recomendado.
10. El usuario completa una actividad en la plataforma.
11. La plataforma guarda el progreso del usuario en la Base de Datos.
12. La Base de Datos confirma a la plataforma el guardado exitoso.
