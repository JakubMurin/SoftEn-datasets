1. Waiter selects the order to review.
2. Interface requests order details from the Order Management System.
3. Order Management System fetches order items from the Menu Database.
4. Menu Database returns order items to the Order Management System.
5. Order Management System requests allergy warnings for items from the Allergy Information System.
6. Allergy Information System looks up allergy info in the Menu Database.
7. Menu Database returns allergy data to Allergy Information System.
8. Allergy Information System compiles allergy warnings for items.
9. Order Management System sends compiled allergy warnings to the Waiter Interface.
10. Waiter Interface displays allergy warnings to the Waiter.
