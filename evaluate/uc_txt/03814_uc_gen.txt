1. User starts the app.
2. Client asks the user for a nickname.
3. User writes the provided nickname.
4. Client sends a request to the server to join with the given nickname, IP, and port.
5. Server checks if the nickname is valid.
6. If the nickname is valid, the server adds the new user's nickname and informs the client that the user joined.
7. Server updates the chat room with a message about the user who joined.
8. Client enables the ready-to-play button.
9. Client enables the chat functionality.
10. Client notifies the user that it successfully joined.
