1. Manager accesses inventory management.
2. UI requests current inventory list from Backend.
3. Backend queries inventory items from Database.
4. Database returns inventory items to Backend.
5. Backend displays inventory items to UI.
6. Manager selects item to update.
7. Manager enters new inventory details.
8. UI sends update inventory request with details to Backend.
9. Backend updates item in the database.
10. Database confirms the update.
11. Backend confirms success to Manager.
12. UI displays update confirmation to Manager.
