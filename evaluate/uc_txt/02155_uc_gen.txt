1. Curator chooses 'Move Item'.
2. User Interface requests form to move item.
3. Collection Management System displays form.
4. Curator submits form (Item ID, Source Collection, Target Collection).
5. User Interface validates and processes move request.
6. Collection Management System fetches item from Source Collection.
7. Database provides Item Details to Collection Management System.
8. Collection Management System removes item from Source Collection.
9. Database confirms removal.
10. Collection Management System adds item to Target Collection.
11. Database confirms addition.
12. Collection Management System displays success message.
13. User Interface confirms item moved.
