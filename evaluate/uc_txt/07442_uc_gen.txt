1. Library Staff accesses the collection creation form on the Web Interface.
2. Web Interface requests the collection creation form from the Server Application.
3. Server Application returns the collection creation form to Web Interface.
4. Library Staff fills and submits the form with custom metadata fields.
5. Browser Interface submits the collection creation request with custom metadata to Server Application.
6. Server Application validates and saves the collection data in the Database.
7. Database confirms the collection data has been saved.
8. Server Application displays a success message to Library Staff.
9. Browser Interface shows the success message and collection details to Library Staff.
