1. User clicks on the garbage bin under "financial goal" in the GUI Financial Account Details View.
2. User enters the amount to be deleted.
3. GUI sends a delete request with the financial-goal-id to the Server.
4. Server deletes the financial goal with the provided financial-goal-id from the Database.
5. Server returns the updated financial account data to the GUI.
6. GUI displays the updated financial account details to the User.
