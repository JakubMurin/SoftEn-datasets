1. Curator requests the Dashboard View.
2. Dashboard fetches recent submissions from SubmissionService.
3. SubmissionService queries the Database for recent submissions.
4. Database returns submission data to SubmissionService.
5. SubmissionService processes submission summary and sends it to Dashboard.
6. Dashboard prepares the Collection Widget with recent submissions.
7. Collection Widget displays the recent submissions on the Dashboard.
8. Dashboard shows the Dashboard with the Collection Widget to the Curator.
