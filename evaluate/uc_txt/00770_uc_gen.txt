1. Admin requests to add a new user.
2. FrontEnd requests data for the new user.
3. Admin sends data for the new user to FrontEnd.
4. FrontEnd sends data for the new user and token of Admin to MD.
5. MD checks the token of Admin.
6. MD validates the data for the new user.
7. MD adds the new user.
8. MD sends a message of success to FrontEnd.
9. FrontEnd shows success message to Admin.
