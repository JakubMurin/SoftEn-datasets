1. Archivist uploads a search image.
2. UI sends the image data to the WebServer.
3. WebServer initiates the image search process.
4. Image Search Service queries the Database for similar images.
5. Database returns matching images to the Image Search Service.
6. Results Handler prepares the search results.
7. Results Handler sends the results back to UI.
8. UI displays the search results to Archivist.
