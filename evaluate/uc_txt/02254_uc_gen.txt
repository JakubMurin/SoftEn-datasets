1. App Developer deploys new app version with updated dataset type.
2. Deployment System checks for updated dataset type.
3. Dataset Management deploys new version of the dataset type.
4. Dataset Type confirms successful deployment of the new version.
5. Dataset Management updates instances to the new dataset type version.
6. Dataset Instances confirm successful update of instances.
7. Dataset Management confirms successful deployment of the app and dataset.
8. Deployment System sends deployment success confirmation to App Developer.
