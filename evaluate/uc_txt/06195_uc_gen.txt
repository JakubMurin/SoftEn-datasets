1. Repository Manager accesses Replication Policy Settings.
2. EDRMS Interface retrieves Current Policy Settings.
3. Replication Policy Module queries Existing Policies.
4. EDRMS Database returns Policies.
5. Replication Policy Module shows Current Policies.
6. EDRMS Interface displays Current Policies to Repository Manager.
7. Repository Manager defines/updates Policy for Collection.
8. EDRMS Interface processes Policy Specifications.
9. Replication Policy Module updates Policy in DB.
10. EDRMS Database confirms Update.
11. Replication Policy Module initiates Policy Replication Update with Remote Storage Service.
12. Remote Storage Service acknowledges Replication Policy Update.
13. Replication Policy Module confirms Policy Update to EDRMS Interface.
14. EDRMS Interface shows Update Confirmation to Repository Manager.
