1. User completes client service and ticket information.
2. CMO review completed.
3. Professional ticket gets CMO report.
4. Professional ticket obtains category.
5. Ticket Transitions sends message to User to select a category if Category is 0.
6. Ticket Transitions sends message to User to enter report accuracy comments if Category is not 1 and Accuracy Comments are empty.
7. Professional ticket calculates report quality.
8. Ticket Transitions sends message to User to enter Report Quality Comments if Report quality is not 100 and no comments.
9. Professional ticket retrieves additional fact findings from CMO Report.
10. Ticket Transitions sets CMO user, report quality, and time.
11. Client Rest Service saves completed ticket, Dicom Study, and Professional Ticket updates.
12. Ticket Transitions generates PRCMO response letter and sends message to User.
