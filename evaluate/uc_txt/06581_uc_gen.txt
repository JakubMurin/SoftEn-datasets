1. Admin requests to view the list of advertisements.
2. List Advertisements View displays the list of advertisements.
3. Admin clicks on edit button of an advertisement.
4. Edit Advertise View redirects to the edit advertisement page.
5. Edit Advertise View shows the content of the advertisement.
6. Admin edits the advertisement data and clicks save.
7. Edit Advertise View sends the information to AdsController.
8. AdsController checks the data.
9. AdsController saves the data in AdsModel.
10. AdsModel returns the saved data to AdsController.
11. AdsController returns the result of saving the advertisement to Edit Advertise View.
12. Edit Advertise View displays the edited advertisement.
