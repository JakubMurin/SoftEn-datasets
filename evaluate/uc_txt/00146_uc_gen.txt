1. The archivist requests embargo metadata for ETDs from the Web Interface.
2. The Web Interface fetches the embargo metadata from the EDRMS Backend.
3. The EDRMS Backend queries the Repository Database for embargo metadata.
4. The Repository Database returns the embargo metadata to the EDRMS Backend.
5. The EDRMS Backend displays the embargo metadata to the Web Interface.
6. The Web Interface shows the embargo metadata to the archivist.
7. The archivist requests the download of the embargo metadata from the Web Interface.
8. The Web Interface fetches detailed metadata from the EDRMS Backend.
9. The EDRMS Backend queries the Repository Database for detailed embargo data.
10. The Repository Database returns the detailed data to the EDRMS Backend.
11. The EDRMS Backend prepares the download file on the File Server.
12. The File Server provides a download link to the EDRMS Backend.
13. The EDRMS Backend provides the download link to the Web Interface.
14. The Web Interface provides the download link to the archivist.
15. The archivist downloads the file from the File Server using the provided link.
16. The File Server sends the metadata file to the archivist.
