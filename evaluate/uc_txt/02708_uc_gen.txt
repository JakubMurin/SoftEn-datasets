1. Depositor accesses the deposit interface.
2. Web Interface requests authentication.
3. Web Interface authenticates depositor.
4. Web Interface shows dataset deposit form to Depositor.
5. Depositor fills in the form including metadata and files.
6. Dataset Management System processes the deposit form.
7. Dataset Management System checks if dataset metadata aligns with Pure requirements.
8. Dataset Management System deposits dataset and metadata to Bath Data Archive.
9. Archive confirms deposit and generates DOI.
10. Dataset Management System links dataset DOI with Pure records.
11. Pure updates Pure records.
12. Dataset Management System confirms dataset deposit and provides DOI to Depositor.
13. Web Interface displays confirmation to Depositor including DOI link.
