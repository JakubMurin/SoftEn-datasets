1. Member navigates to the Reserve Book page via web interface.
2. Web Interface displays the search interface for the member.
3. Member inputs search parameters to find desired book(s).
4. Reservation System queries the database for books matching search parameters.
5. Database returns a list of available books based on the search.
6. Reservation System displays the available books to the member.
7. Member chooses a book to reserve through the web interface.
8. Selected book reservation request is processed by the Reservation System.
9. Database updates reservation status upon successful booking.
10. Reservation System confirms successful reservation to the member.
11. Member receives a success message with a confirmation of the booking.
