1. User opens the mobile app.
2. App loads the system message module.
3. App requests system message data retrieval.
4. System message module checks for unread messages and retrieves them.
5. App displays unread message notification to the user.
6. User taps to view the messages.
7. User views message details.
8. App requests message content.
9. System message module fetches the message content.
10. App displays the content to the user.
