1. Admin requests to view the list of advertisements.
2. List Advertisements View displays the list of advertisements.
3. Admin clicks on the edit button of an advertisement.
4. List Advertisements View redirects to Edit Advertise View.
5. Edit Advertise View shows the content of the advertisement.
6. Admin edits the advertisement data and clicks on the save button.
7. Edit Advertise View sends the information to AdsController.
8. AdsController validates the data and saves it.
9. AdsModel saves the data and returns the saved data to AdsController.
10. AdsController returns the result of saving the advertisement to Edit Advertise View.
11. Edit Advertise View displays the edited advertisement to Admin.
