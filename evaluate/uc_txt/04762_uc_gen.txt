1. User clicks on the story cover.
2. View requests API for story details.
3. Story processes GetStoryDetail().
4. Story queries data from the database.
5. Database returns data to Story.
6. Message Service returns response.
7. View displays the Story Detail Page.
8. View requests API for related authors.
9. Author processes GetStoryRelateAuthor().
10. Author queries data from the database.
11. Database returns data to Author.
12. Message Service returns response.
13. View requests API for story chapters.
14. Chapter processes GetStoryChapters().
15. Chapter queries data from the database.
16. Database returns data to Chapter.
17. Message Service returns response.
18. View requests API for story comments.
19. Comment processes GetStoryComments().
20. Comment queries data from the database.
21. Database returns data to Comment.
22. Message Service returns response.
