1. Admin creates a new Operation Type with its attributes.
2. FrontEnd sends the request to Backoffice API to create the Operation Type.
3. Backoffice API validates the Operation Type attributes.
4. Backoffice API saves the new Operation Type in the database.
5. Backoffice API logs the creation of the Operation Type.
6. Backoffice API returns a confirmation to the FrontEnd.
7. FrontEnd returns the confirmation to the Admin.
