1. Moderator navigates to Account Settings.
2. Web Interface receives request to view current account details.
3. Backend Server fetches account details from Account Database.
4. Backend Server displays current account details to Web Interface.
5. Web Interface shows account details form to the Moderator.
6. Moderator inputs updated account details.
7. Web Interface submits the updated account details to the Backend Server.
8. Backend Server updates the account details in the Account Database.
9. Backend Server confirms the successful update.
10. Web Interface acknowledges the successful update to the Moderator.
11. Moderator displays the update success message.
