1. Curator accesses the item/component page through the UI Dashboard.
2. UI Dashboard validates access with the Authorization Module.
3. Authorization Module grants access.
4. UI Dashboard requests item/component details from the Retrieval Module.
5. Retrieval Module queries the Database for item/component details and deletion date.
6. Database returns the requested data to the Retrieval Module.
7. Retrieval Module displays item/component details and deletion date to the UI Dashboard.
8. UI Dashboard shows the item/component page with deletion date to the Curator.
