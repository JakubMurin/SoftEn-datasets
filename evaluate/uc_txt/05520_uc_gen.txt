1. Curator accesses the item/component page from UI Dashboard.
2. UI Dashboard validates access with Authorization Module.
3. Authorization Module grants access to UI Dashboard.
4. UI Dashboard requests item/component details from Retrieval Module.
5. Retrieval Module queries Database for details and deletion date.
6. Database provides requested data to Retrieval Module.
7. Retrieval Module displays item/component details and deletion date to UI Dashboard.
8. UI Dashboard shows the item/component page along with deletion date to the Curator.
