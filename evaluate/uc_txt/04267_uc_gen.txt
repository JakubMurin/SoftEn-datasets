1. Admin creates Operation Type with its attributes.
2. FrontEnd validates Operation Type attributes.
3. FrontEnd sends Operation Type creation request to Backoffice API.
4. Backoffice API validates Operation Type attributes.
5. Backoffice API saves New Operation Type in Database.
6. Backoffice API logs Operation Type creation.
7. Backoffice API returns confirmation to FrontEnd.
8. FrontEnd returns confirmation to Admin.
