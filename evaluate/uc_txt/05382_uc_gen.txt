1. User views User Details.
2. UserDetailsPage activates.
3. UserDetailsPage retrieves User Details via Controller.
4. Controller activates.
5. Controller fetches User Details via Model.
6. Model activates.
7. Model queries Database for User Details.
8. Database returns User details to Model.
9. Model returns User details to Controller.
10. Model deactivates.
11. Controller returns User details to UserDetailsPage.
12. Controller deactivates.
13. UserDetailsPage displays User Details.
14. UserDetailsPage deactivates.
