1. Funder requests security details for a specific project from DMP.
2. DMP assesses security measures and standards by involving SAM.
3. SAM fetches relevant data security protocols from PD.
4. PD returns security protocols to SAM.
5. SAM verifies compliance with legal standards by consulting CD.
6. CD returns compliance status to SAM.
7. SAM compiles a security assessment report and sends it to RG.
8. RG generates and formats the report.
9. DMP provides the report on data security and compliance to the Funder.
