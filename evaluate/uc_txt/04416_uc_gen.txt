1. Player starts the game from the main menu.
2. StartGameMenu processes the user's selection.
3. Player presses the exit button in StartGameMenu.
4. StartGameMenu initiates the game.
5. GameFrame handles the game for the player.
6. If the player loses, GameFrame directs to GameEndMenu.
7. GameEndMenu offers options to return to StartGameMenu or start a new game in GameFrame.
8. StartGameMenu deletes the game results from the database.
9. ResultsFrame displays the best game results to the player.
