1. Digital Records Archivist requests embargo metadata for ETDs from the Web Interface.
2. Web Interface fetches embargo metadata from EDRMS Backend.
3. EDRMS Backend queries embargo metadata from Repository Database.
4. Repository Database returns the embargo metadata.
5. EDRMS Backend displays the metadata to the Web Interface.
6. Web Interface shows embargo metadata to the Digital Records Archivist.
7. Digital Records Archivist requests download of embargo metadata from Web Interface.
8. Web Interface fetches detailed metadata from EDRMS Backend.
9. EDRMS Backend negotiates with the Repository Database to get detailed embargo data.
10. Repository Database returns the detailed embargo data.
11. EDRMS Backend contacts the File Server to prepare the download file.
12. File Server generates a download link.
13. EDRMS Backend provides the download link to the Web Interface.
14. Web Interface sends the download link to the Digital Records Archivist.
15. Digital Records Archivist downloads the file from the File Server using the provided link.
16. File Server delivers the metadata file to the Digital Records Archivist.
