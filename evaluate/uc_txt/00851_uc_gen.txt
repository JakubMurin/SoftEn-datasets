1. User initiates point contract deployment.
2. Application layer deploys PointController using JAVA_SDK.
3. Application layer records contract address in configuration file.
4. Application layer retrieves platform admin user information from MySQL.
5. Java_SDK executes PointController.addDAO.
6. System initializes integral total amount.
7. System performs annual integral distribution for recyclers.
8. User registers an account.
9. User adds batteries for integral payment.
10. User registers vehicle battery information.
11. User receives integral as a consumer.
12. User provides battery disassembly information for recycling.
13. User applies for integral trade.
