1. AccreditedRepresentative accesses POA Request List View.
2. Frontend shows List of VSO POA Requests.
3. Frontend sends GET request to Backend for POA Requests.
4. Backend verifies AccreditedRepresentative Authentication.
5. Backend queries VAGovDB for POA Request List Data.
6. VAGovDB returns POA Request List Data to Backend.
7. Backend returns POA Request List Response to Frontend.
8. Frontend displays POA Request List to AccreditedRepresentative.
