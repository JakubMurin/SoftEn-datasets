1. Batch Submitter navigates to EDRMS Documentation Section.
2. Batch Submitter provides credentials for authentication.
3. Authentication Service authenticates the user.
4. EDRMS Website requests Ingest Process Docs.
5. Documentation Server retrieves the documentation.
6. Documentation Server sends the documentation link to EDRMS Website.
7. EDRMS Website displays the documentation link to Batch Submitter.
8. Batch Submitter clicks the documentation link.
9. EDRMS Website redirects to Ingest Process Documentation.
10. Ingest Process Documentation displays the Ingest Process Documentation to Batch Submitter.
