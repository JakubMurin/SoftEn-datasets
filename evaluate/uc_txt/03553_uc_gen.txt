1. DonorRepresentative accesses the website through the User Interface.
2. User Interface requests access from the Authorization System.
3. Authorization System authenticates the DonorRepresentative.
4. DonorRepresentative requests descriptive information about images.
5. User Interface queries the File Management system for images.
6. File Management system fetches image metadata from the Database.
7. Database returns the metadata to the File Management system.
8. File Management system displays the descriptive information on the User Interface.
9. User Interface shows the descriptive information about the images to the DonorRepresentative.
