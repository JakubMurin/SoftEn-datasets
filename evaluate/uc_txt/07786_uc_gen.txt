1. User clicks 'Comenzar' button.
2. App handles the event to start the game.
3. StartScreen displays and StartButtonEventHandler processes the event.
4. The StartButtonEventHandler hides the StartScreen and displays PlayersInputScreen to select inputs.
5. User selects and confirms inputs.
6. The event is handled to display the screen for selecting player names.
7. User inputs and confirms player names.
