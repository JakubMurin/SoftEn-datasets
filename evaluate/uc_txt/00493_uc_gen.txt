1. Inviter create new connection by sending data.
2. Invitee Agent generates new Peer DID.
3. Invitee Agent creates connection state record.
4. Invitee receives connection record.
5. Inviter packs connection invitation into QR code.
6. Inviter sends invitation to Invitee using OOB mechanism.
7. Inviter checks connection state.
8. Invitee parses the invitation message.
9. Invitee reviews the invitation.
10. Invitee accepts the invitation.
11. Invitee Agent creates pairwise DID.
12. Invitee Agent creates connection state record.
13. Invitee receives connection record.
14. Invitee Agent sends "ConnectionRequest" message to Inviter Agent.
15. Inviter Agent sends response to Invitee Agent.
16. Inviter Agent sends "ConnectionResponse" message to Invitee Agent.
17. Invitee Agent sends response to Inviter Agent.
