1. User opens the application.
2. Homepage screen displays "Account Settings".
3. User navigates to Account Settings.
4. Modify Password Screen is displayed.
5. User is prompted to enter the updated password.
6. User enters the updated password and presses "Modify" button.
7. Updated password is submitted to AuthService.
8. API call is made to modify the password.
9. Forwarded updated password is validated.
10. Password update validation fails.
11. Validation error is displayed to the user.
12. Password update validation succeeds.
13. Password is updated in the database.
14. Confirmation of password update is sent.
15. Update success message is displayed.
16. User is navigated back to the Homepage.
17. Confirmation message is displayed to the user.
