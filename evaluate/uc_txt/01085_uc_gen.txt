1. Moderator Navigates to Account Settings.
2. Web receives Request and forwards to Server.
3. Server communicates with Database to retrieve account details.
4. Database sends account details back to Server.
5. Server presents updated account details to Web.
6. Web displays account details form to Moderator.
7. Moderator Inputs updated account details in Web Interface.
8. Updated details are submitted by Web to the Server.
9. Server updates information in the Database.
10. Database confirms successful update back to Server.
11. Server acknowledges successful account update to Web.
12. Web shows update success message to the Moderator.
