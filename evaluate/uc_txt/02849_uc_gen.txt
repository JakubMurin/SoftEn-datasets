1. User starts the script in main.py.
2. Main generates embeddings for each requirement using OpenAI API.
3. Main stores the embeddings in the Pinecone vector database.
4. Main queries the stored embeddings from Pinecone.
5. Main groups the embeddings into clusters using K-Means Clustering.
6. Main creates a scatter plot and textual output using the Visualization Module.
7. Visualization Module displays the results to the User.
