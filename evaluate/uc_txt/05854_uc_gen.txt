1. Owner requests a sales report.
2. Frontend Interface initiates an API call to Backend System for sales data.
3. Backend System triggers Reporting Module to generate a sales report.
4. Reporting Module fetches sales data from Database.
5. Reporting Module generates the sales report.
6. Backend System sends the sales report to Frontend Interface.
7. Frontend Interface displays the sales report data.
8. Owner views the displayed sales report.
