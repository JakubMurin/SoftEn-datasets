1. User clicks on the 'Add account note' button in the header.
2. User navigates to the 'Add note' page.
3. User inputs data for the note.
4. User clicks on the 'Save Note' button.
5. Note is created.
6. Note data is stored in the Fines database using JPA.
7. Created note data is returned in JSON format.
8. User navigates back to the 'View account' page.
