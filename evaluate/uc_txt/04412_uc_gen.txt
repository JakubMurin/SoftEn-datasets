1. Player starts the game.
2. Player navigates through the start game menu.
3. Player exits the game by pressing the exit button.
4. Player begins the game in the game frame.
5. If the player loses, the game redirects to the game end menu.
6. Player chooses to return to the start game menu from the game end menu.
7. Player decides to start a new game from the game end menu.
8. Start game menu deletes game results from the database.
9. Results frame shows the best game results.
10. Results frame retrieves game results from the database.
