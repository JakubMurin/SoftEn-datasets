2. System requests the Customer Manager to input a job reference.

4. System shows Job Opening details and asks the Customer Manager if they want to update the description.
5. Customer Manager inputs their choice (yes/no).
6. System asks if the Customer Manager wants to update the address.
7. Customer Manager inputs their choice (yes/no).
8. System asks if the Customer Manager wants to update the Mode.
9. Customer Manager inputs their choice (yes/no).
10. System asks if the Customer Manager wants to update the Contract Type.
11. Customer Manager inputs their choice (yes/no).
12. System asks if the Customer Manager wants to update the job title.
13. Customer Manager inputs their choice (yes/no).
14. System asks if the Customer Manager wants to update the number of vacancies.
15. Customer Manager inputs their choice (yes/no).
16. System requests the Customer Manager for new values to update.
17. Customer Manager inputs the new values.
18. System displays the updated Job Opening and shows the operation success.
