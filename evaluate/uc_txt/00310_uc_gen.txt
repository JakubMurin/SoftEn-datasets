1. User decides to pay with USDT.
2. UI requests payment information from Gateway.
3. Gateway provides USDT deposit address, amount, and QR code to User.
4. User transfers USDT via Blockchain.
5. Monitor detects transaction on Blockchain.
6. Monitor requests transaction confirmation from Blockchain.
7. Blockchain updates transaction status.
8. Gateway receives final transaction approval notification from Monitor.
9. Gateway stores transaction and user information in Database.
10. Database confirms payment and converts to points for User.
