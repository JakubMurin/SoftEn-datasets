1. Content Editor enters search query (keywords/tags).
2. CMS Search Interface queries for content.
3. CMS Database returns search results.
4. CMS Search Interface displays search results to Content Editor.
5. Content Editor selects content from the results.
6. CMS Search Interface requests selected content details from CMS Database.
7. CMS Database retrieves content details.
8. CMS Search Interface displays selected content in the editor.
9. Content Editor edits the content.
10. Content Page saves the content changes to CMS Database.
11. CMS Database confirms that changes are saved.
12. Content Page displays confirmation to Content Editor (edit successful).
