1. User goes to the post page.
2. User displays post details and comment form.
3. User enters comment content (text).
4. User submits comment.
5. UI creates comment using CommentDTO.
6. CommentController validates and creates the comment.
7. CommentService saves the comment.
8. CommentRepo inserts comment data into CommentsDB.
9. UI displays "Comment created successfully" message.
10. UI updates post comments section.
