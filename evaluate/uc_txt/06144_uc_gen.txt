1. Admin requests to add a new other service.
2. UI displays the form to add the service.
3. Admin submits the new service data.
4. UI sends service details to be added.
5. Router triggers the Controller to add the service.
6. Controller adds the service via Repository.
7. Repository saves the new service data.
8. Service data is created in the OtherServiceModel.
9. Data is inserted into the Database.
10. Service added successfully message is returned.
