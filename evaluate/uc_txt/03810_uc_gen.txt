1. User starts the app.
2. User asks for a nickname.
3. User writes the nickname.
4. Client joins the server with the provided nickname, IP, and port.
5. Server adds the nickname and IP, then notifies the client that it has joined.
6. Server updates the chat by informing of the new user joining.
7. Client enables the "Ready to Play" button and the chat feature.
8. User acknowledges success.
9. In case of an error with the nickname, the server sends an error message to the client.
10. Client notifies the user of the error.
