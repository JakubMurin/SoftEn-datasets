1. User inputs a number.
2. Game receives the input and sets the guess number.
3. User clicks "Adivinar".
4. Game parses the guess number to an integer and sets a message based on conditions.
5. Game displays the message to the User.
6. User clicks "Reiniciar".
7. Game generates a new target number, resets the guess, and message.
8. User clicks "Back to Home".
9. User invokes onBackToHome function.
