1. Member requests to share work.
2. Application retrieves file share form.
3. UI manager displays the file share form.
4. Member views the file share form.
5. Member fills in work details and attaches the file.
6. Member adds file to the form.
7. Member submits the form.
8. Application analyzes the file.
9. Application displays the analysis result to the member.
10. Application requests confirmation for submission.
11. Member validates the submission.
12. Application copies the file to the "Moderation" directory.
13. Application generates a transaction number.
14. Application logs the transaction.
15. Application sends transaction and file to the server.
16. Server notifies librarian of the work submission.
17. Application notifies member of work moderation wait.
