1. User requests a secret key generation.
2. SecretKey object calls SecureRandom constructor.
3. SecureRandom object is created.
4. SecretKey object requests SecureRandom to generate random bytes.
5. SecureRandom object calls NativeCrypto to generate random bytes.
6. NativeCrypto generates random bytes and returns to SecureRandom.
7. SecureRandom returns the generated bytes to SecretKey.
8. SecretKey object provides the generated secret key to the User.
