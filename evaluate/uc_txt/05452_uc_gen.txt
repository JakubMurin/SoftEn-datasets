1. Sign In to Authentication System with Archivist credentials.
2. Access and open the Upload Interface section.
3. Select one or more Inventory Files to upload.
4. Provide necessary Metadata about the Files.
5. Define the Access Rights for the uploaded Files.
6. Store the Metadata and Access Rights into the Database.
7. Successfully upload the selected Files to the File Storage system.
8. Update the Inventory List with the newly added entries.
9. Display a success message confirming the completion of the upload process.
10. Send Notifications to the Donor Representatives via the Notification System.
