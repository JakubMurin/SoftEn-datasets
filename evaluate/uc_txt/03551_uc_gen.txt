1. Donor Representative accesses the website.
2. User Interface receives the access request.
3. Authorization System authenticates the Donor Representative.
4. Donor Representative requests image descriptive information.
5. User Interface queries for images.
6. File Management fetches image metadata.
7. Database returns metadata for the images.
8. File Management displays the descriptive information.
9. User Interface shows the descriptive information about the images to the Donor Representative.
