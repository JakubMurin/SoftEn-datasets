1. User configures image resolution for the device.
2. Mobile App sets the image resolution for the device.
3. Mobile App sends device CID to Server.
4. Server searches for active TCP connection.
5. Server informs Mobile App of device unavailability.
6. Mobile App displays device unavailability.
7. Server sets configuration for ESP32.
8. ESP32 presents the configuration.
9. Server sends current device configuration to Mobile App.
10. Mobile App updates the view.
11. Mobile App displays successful settings update message to User.
