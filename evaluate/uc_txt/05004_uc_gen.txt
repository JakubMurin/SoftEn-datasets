1. Archivist uploads search image.
2. UI sends image data to the Web Server.
3. Web Server initiates image search with the Image Search Service.
4. Image Search Service queries the Database for similar images.
5. Database returns matching images to the Image Search Service.
6. Image Search Service prepares search results using the matching images.
7. Results Handler sends the search results back to the UI.
8. UI displays the search results to the Archivist.
