1. User initiates editing of a task.
2. System fetches task data based on task ID.
3. User populates the task edit modal fields.
4. User submits the task edit form to save.
5. System processes the task data submission.
6. Hierarchy constraints of the task data are validated.
7. System checks for circular reference in task hierarchy.
8. Task is added or updated in the database.
9. Save event is logged for auditing purposes.
