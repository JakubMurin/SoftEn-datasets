1. Faculty Member enters credentials.
2. Credentials are validated by Authentication Service.
3. Search Interface displays search options.
4. Faculty Member inputs search criteria for collection.
5. Database queries with the search criteria.
6. Search Interface displays list of matching collections.
7. Faculty Member selects desired collection.
8. Search Interface requests selected collection details.
9. Collection details are retrieved from the Database.
10. Faculty Member accesses the Collection.
11. Collection Content is displayed to Faculty Member.
