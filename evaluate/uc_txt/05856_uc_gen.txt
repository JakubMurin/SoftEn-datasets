1. Owner requests a sales report.
2. Frontend Interface initiates API call to Backend System for sales data.
3. Backend System queries Reporting Module to generate the sales report.
4. Reporting Module fetches the required sales data from the Database.
5. Database provides the sales data to the Reporting Module.
6. Reporting Module creates the sales report and sends it to the Backend System.
7. Backend System displays the generated sales report data on the Frontend Interface.
8. Frontend Interface shows the sales report to the Owner.
