1. O usuário solicita a página dos andares.
2. O SPA retorna a página dos andares ao usuário.
3. O usuário seleciona um andar da tabela.
4. O SPA mostra os dados do andar selecionado ao usuário.
5. O SPA realiza uma requisição à API, solicitando os edifícios disponíveis.
6. A API retorna a lista de edifícios em uma caixa de seleção para o SPA.
7. O SPA exibe os dados do andar selecionado.
8. O usuário insere um arquivo com o floormap do andar selecionado.
9. O SPA envia os dados do floormap do andar escolhido para a API para atualização.
10. A API processa a atualização do floormap com o novo arquivo enviado pelo SPA.
11. A API confirma a conclusão da atualização.
12. O SPA informa ao usuário o sucesso da operação.
