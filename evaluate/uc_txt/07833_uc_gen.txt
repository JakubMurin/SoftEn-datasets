1. RM initiates Update by starting the process.
2. DIMS Interface receives the Update Initiation.
3. DIMS Interface submits the Update Request to Central Repository Info Management.
4. Central Repository Info Management checks Current Repository Information.
5. Central Repository Info Management updates the Repository Information in DAMS.
6. DAMS retrieves the Updated Repository Information.
7. DAMS Propagates Updates to Relevant Areas.
8. DIMS Interface receives Update Success Notification.
