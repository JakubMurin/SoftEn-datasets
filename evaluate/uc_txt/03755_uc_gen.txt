1. Archivist submits credentials to Authentication System.
2. Authentication System validates credentials with User Management System.
3. User Management System provides authentication success details to Authentication System.
4. Authentication System grants access to Archivist.
5. Archivist requests access to X-drive from File Management System.
6. File Management System verifies permissions with Access Restriction Controls.
7. Access Restriction Controls retrieves Archivist permissions from User Management System.
8. User Management System provides permission details to Access Restriction Controls.
9. Access Restriction Controls permits access to File Management System.
10. File Management System fetches files from XDrive.
11. XDrive returns requested files to File Management System.
12. File Management System displays files to Archivist.
