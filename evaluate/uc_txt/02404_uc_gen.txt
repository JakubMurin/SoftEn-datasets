1. Faculty Data Steward accesses the DMPTool.
2. DMPTool Interface displays the Dashboard.
3. Faculty Data Steward navigates to the Costing Section.
4. DMPTool Interface requests Costing Data.
5. Costing Module fetches Costing Information from Project Data.
6. Project Data returns the Costing Data to Costing Module.
7. Costing Module displays the Costing Information.
8. DMPTool Interface shows the Costing Section.
