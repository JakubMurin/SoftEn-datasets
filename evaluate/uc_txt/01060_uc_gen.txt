1. User opens the Add Event Page.
2. User enters event details.
3. User selects an associated contact.
4. User submits the event form.
5. Add Event Page sends event details to the server.
6. Server validates the details.
7. Server saves the event in the database.
8. Database confirms the event is saved.
9. Server sends a success response to the Add Event Page.
10. Add Event Page shows a success message to the user.
