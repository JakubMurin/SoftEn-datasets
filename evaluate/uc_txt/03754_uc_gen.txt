1. Archivist submits credentials to Authentication System.
2. Authentication System validates credentials.
3. User Management System retrieves Archivist permissions.
4. Access Restriction Controls verify permissions.
5. Access Restriction Controls permit access to File Management System.
6. File Management System fetches files from XDrive.
7. XDrive returns requested files to File Management System.
8. File Management System displays files to Archivist.
