1. User clicks 'Comenzar' button.
2. App handles the event.
3. StartScreen handles the event.
4. StartButtonEventHandler handles the event and sets visibility.
5. PlayersInputScreen is displayed for input selection.
6. User selects and confirms inputs.
7. PlayersInputScreen handles the event.
8. ConfirmButton handles the event and requests data selection.
9. Game receives and loads the data.
10. PlayersNamesInputScreen is created and displayed.
11. User inputs and confirms player names.
12. PlayersNamesInputScreen handles the event.
13. NamesButton handles the event and executes.
14. Game registers user names.
15. Player objects are created and added to the game.
16. QuestionScreen is created and displayed.
17. App shows questions and options.
18. QuestionScreen shows questions for each player.
