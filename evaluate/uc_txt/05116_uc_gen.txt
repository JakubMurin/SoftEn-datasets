1. Publisher requests preview of Data Package through UI Interface.
2. UI Interface verifies Publisher's identity with Publisher Account.
3. Publisher Account confirms Publisher's identity to UI Interface.
4. UI Interface prepares Data Package for preview using Data Package Service.
5. Data Package Service validates the Data Package with Validation Service.
6. Validation Service provides validation results to Data Package Service.
7. Data Package Service sends validated Data Package to Preview Service.
8. Preview Service generates a preview link.
9. Data Package Service sends the preview link to UI Interface.
10. UI Interface displays the preview link to Publisher.
