1. Archivist uploads a search image to the UI.
2. UI sends the image data to the Web Server.
3. Web Server initiates image search with Image Search Service.
4. Image Search Service queries the Database for similar images.
5. Database responds to Image Search Service with matching images.
6. Image Search Service prepares search results.
7. Results Handler sends the search results back to UI.
8. UI displays the search results to Archivist.
