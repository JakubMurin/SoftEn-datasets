1. Publisher requests Preview of Data Package from UI Interface.
2. UI Interface verifies Publisher Identity with Publisher Account.
3. Publisher Account confirms Publisher Identity.
4. UI Interface prepares Data Package for Preview with Data Package Service.
5. Data Package Service validates Data Package with Validation Service.
6. Validation Service provides Validation Results to Data Package Service.
7. Data Package Service sends Validated Data Package to Preview Service.
8. Preview Service generates Preview Link and sends it to Data Package Service.
9. Data Package Service provides Preview Link to UI Interface.
10. UI Interface displays Preview Link to Publisher.
