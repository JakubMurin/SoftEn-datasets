1. Researcher logs in to DMPTool.
2. Researcher navigates to project DMP.
3. DMPTool fetches DMP record details.
4. Researcher selects "Add Publication".
5. Researcher enters publication details.
6. DMPTool checks if publication is in Repository.
7. DMPTool links publication with DMP record.
8. Researcher requests research output report.
9. DMPTool fetches all publications linked to DMP.
10. Repository returns publications data to DMPTool.
11. Report Module generates report based on publications.
12. Report Module sends the generated report to DMPTool.
13. DMPTool displays/download the report to Researcher.
