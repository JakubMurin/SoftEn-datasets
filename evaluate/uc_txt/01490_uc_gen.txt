1. User posts a Slack command to launch a rocket.
2. APP receives the `/rocker` event triggered command from SLACK.
3. APP posts an ephemeral message to the user in a channel.
4. User clicks on the button `launch rocket`.
5. APP receives the `interaction` event triggered command from SLACK.
6. APP posts a message to the user.
7. SLACK displays a Rocket Countdown Message 3 to the user.
8. APP posts a message to the user.
9. SLACK displays a Rocket Countdown Message 2 to the user.
10. APP posts a message to the user.
11. SLACK displays a Rocket Countdown Message 1 to the user.
