1. Batch Submitter requests authentication by navigating to the EDRMS Documentation Section on the EDRMS Website.
2. EDRMS Website prompts Batch Submitter to authenticate by requesting authentication from the Authentication Service.
3. Batch Submitter provides credentials for authentication.
4. Authentication Service authenticates Batch Submitter.
5. EDRMS Website requests Ingest Process Documentation from the Documentation Server.
6. Documentation Server retrieves Ingest Process Documentation from the Ingest Process Documentation.
7. Documentation Server returns Ingest Process Documentation to the EDRMS Website.
8. EDRMS Website sends a link to the Ingest Process Documentation to Batch Submitter.
9. Batch Submitter displays the link to the Ingest Process Documentation.
10. Batch Submitter clicks on the link to access the Ingest Process Documentation.
11. EDRMS Website redirects Batch Submitter to the Ingest Process Documentation.
12. Ingest Process Documentation displays the Ingest Process Documentation to Batch Submitter.
