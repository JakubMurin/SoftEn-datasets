1. User searches for a restricted object and selects it.
2. Interface requests access details from ACS.
3. ACS displays "Contact Curator" option to the user.
4. User clicks on "Contact Curator" option.
5. Interface generates an email template.
6. User fills in and sends the access request email.
7. Email forwards the user request to the Curator.
8. Curator reviews the access request.
9. Curator updates access permissions in ACS (if approved).
10. ACS notifies the user of the decision via Email.
11. Email displays the access decision notification to the user.
