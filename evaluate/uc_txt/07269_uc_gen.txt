1. The Authenticated User initiates the process to record a new activity.
2. The Client prepares the necessary data regarding the new activity.
3. The Client communicates the data related to the new activity to the Server.
4. The Server sends the information about the new activity to the Database for storage.
5. The Database successfully adds the new activity data.
6. The Server confirms to the Client that the data has been saved.
7. The Client notifies the Authenticated User about the successful recording of the new activity.
