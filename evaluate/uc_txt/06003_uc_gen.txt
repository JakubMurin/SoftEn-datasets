1. User selects recyclable waste types.
2. Frontend Application requests facilities for waste types from Backend Server.
3. Backend Server queries facilities from Database by waste type.
4. Database returns a list of facilities to Backend Server.
5. Backend Server sends facilities and opening hours to Frontend Application.
6. Frontend Application displays facilities and opening hours to the user.
7. User requests optimal route and schedule.
8. Frontend Application calculates optimal route based on user location and facility hours.
9. Backend Server returns the optimal route and schedule to Frontend Application.
10. Frontend Application displays the optimal route and schedule to the user.
