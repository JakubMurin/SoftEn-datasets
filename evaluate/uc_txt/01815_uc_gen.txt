1. Researcher requests creation of proxy Data Package.
2. DPIE App identifies authoritative data sources.
3. Data Sources return a list of sources.
4. DPIE App selects appropriate sources based on reliability and quality.
5. DPIE App generates proxy Data Package specification.
6. Data Package Tooling stores proxy Data Package.
7. Data Package Tooling returns proxy Data Package URL and metadata.
8. DPIE App provides proxy Data Package details and access instructions to Researcher.
