1. Administrator requests statistics.
2. Admin UI fetches statistics from Backend System.
3. Backend System retrieves comprehensive stats from Statistics Module.
4. Statistics Module queries faculty work data from Database.
5. Database returns the queried data to Statistics Module.
6. Statistics Module processes data into statistics.
7. Report Generation Service generates a digestible report.
8. Report Generation Service returns the generated report to Backend System.
9. Backend System displays the report through Admin UI.
10. Admin UI shows comprehensive tracking statistics to Administrator.
