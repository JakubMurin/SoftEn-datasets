1. Manager selects the "Monthly Popular Dishes" report.
2. UI Dashboard requests for most frequently ordered food of the month.
3. Backend Server queries ordered food data of the month.
4. If data available.
5. Database returns ordered food data of the month.
6. Backend Server processes data to find most frequently ordered dishes.
7. Backend Server sends report of most frequently ordered food.
8. UI Dashboard displays most frequently ordered food report.
9. If no data available.
10. Database returns empty result.
11. Backend Server sends no data message.
12. UI Dashboard displays no data available message.
