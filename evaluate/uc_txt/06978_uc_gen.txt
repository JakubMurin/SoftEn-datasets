1. User searches and selects a restricted object.
2. Repository Interface requests access details from the Access Control System.
3. Access Control System displays the "Contact Curator" option to the Repository Interface.
4. User clicks on the "Contact Curator" option.
5. Repository Interface generates an email template to contact the curator.
6. User fills in the email template and sends the request to the curator.
7. Email Notification System forwards the user request to the Collection Curator.
8. Collection Curator reviews the request.
9. Collection Curator updates access permissions in the Access Control System (if approved).
10. Access Control System notifies the user of the decision via Email Notification System.
11. Email Notification System displays the access decision notification to the User.
