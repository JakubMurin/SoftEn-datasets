1. End-User requests help from CS and provides NIN.
2. CS sends NIN to Application.
3. CS informs End-User to start Swedish BankID process.
4. Application initiates Signicat API call to authenticate the user via BankID.
5. End-User starts BankID process.
6. End-User authenticates with BankID via IDP.
7. IDP provides authentication information to Signicat.
8. Signicat processes the authentication.
9. Application receives information on successful authentication from Signicat.
10. CS provides authentication result to the end-user.
11. CS assists the authenticated end-user as needed.
