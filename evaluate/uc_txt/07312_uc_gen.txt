1. Consumer inputs search keywords in the search interface.
2. Search Interface requests matching data packages from Data Registry.
3. Data Registry returns a list of matching data packages to Search Interface.
4. Search Interface displays the search results to the Consumer.
5. Consumer selects a data package based on the description.
6. Search Interface requests details of the selected data package from Data Registry.
7. Data Registry returns the details of the selected package to Search Interface.
8. Search Interface displays the details of the selected data package to the Consumer.
