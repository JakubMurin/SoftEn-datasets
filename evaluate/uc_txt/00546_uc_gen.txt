1. Student opens login page.
2. Monitor displays login page.
3. Student logs in with username and password.
4. Monitor verifies login details.
5. Authentication confirms the login success.
6. Monitor displays library page.
7. Student submits book details.
8. Monitor performs book search using provided details.
9. BookManagement retrieves data of the book.
10. Book updates the book data received.
11. BookManagement lists the available book.
12. Monitor displays the list of books.
13. Student clicks on reserve book.
14. Monitor initiates the book reservation process.
15. Reservation confirms the successful book reservation.
16. Monitor notifies the student of the reservation confirmation.
