1. User speaks into the device.
2. Audio input is captured by the UI interface.
3. Speech is processed and converted to text by the Speech Recognition Module.
4. Converted text is sent to the Content Query Processor.
5. Content Query Processor queries the Content Database for matching items.
6. Content Database returns matched content items.
7. Search results are displayed to the user by the UI interface.
8. User is shown the search results.
