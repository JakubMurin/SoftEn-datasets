1. O Atendente inicia o empréstimo para um leitor.
2. O Sistema identifica o leitor utilizando o ID fornecido.
3. O Leitor é identificado pelo sistema.
4. O Sistema cria um objeto de empréstimo.
5. O objeto de empréstimo é criado com sucesso.
6. O Sistema associa o empréstimo ao leitor.
7. A associação entre o empréstimo e o leitor é concluída.
8. O Sistema informa o início do empréstimo ao Atendente.
