1. Depositor accesses deposit interface.
2. Web Interface requests authentication.
3. Web Interface authenticates depositor.
4. Web Interface shows dataset deposit form.
5. Depositor fills in form (Including metadata, files).
6. Dataset Management System processes deposit form.
7. Dataset Management System deposits dataset and metadata.
8. Archive confirms deposit and generates DOI.
9. Dataset Management System links dataset DOI with Pure records.
10. Dataset Management System updates Pure records.
11. Dataset Management System confirms dataset deposit and provides DOI to depositor.
12. Web Interface displays confirmation (Including DOI link).
