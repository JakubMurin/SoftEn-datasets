1. Visitor inputs the Book Title in the Library System interface.
2. System Interface receives Book Title and starts to search for the book in the library database.
3. Book Details based on Title are requested by System Interface from Book Management Module.
4. Book Management Module queries Book Condition for the given Book ID.
5. Library Database sends back Book Condition on query.
6. The Book Condition is displayed on the Library System Interface.
7. System Interface shows Book Condition to the Visitor.
