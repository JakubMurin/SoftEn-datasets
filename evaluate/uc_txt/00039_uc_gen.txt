1. Chef selects ingredient to update.
2. UI requests current expiry date from Backend.
3. Backend queries ingredient expiry date from Database.
4. Database returns expiry date to Backend.
5. Backend shows current expiry date to UI.
6. Chef inputs new expiry date.
7. UI submits updated expiry date to Backend.
8. Backend updates expiry date in database.
9. Database confirms update to Backend.
10. Backend confirms successful update to UI.
11. UI displays update confirmation to Chef.
