1. Developer edits code.
2. Developer pushes changes to the /dev repository branch.
3. Developer creates a pull request from /dev to /calibration repository branch.
4. Reviewer reviews the pull request.
5. Reviewer approves the review.
6. Developer merges the pull request.
7. Changes are deployed to the calibration server.
8. Developer checks the results.
9. Developer creates a pull request from /dev to /main repository branch.
10. Reviewer approves the pull request.
11. Developer merges the pull request.
12. Changes are deployed to the mainnet server.
13. Developer checks the results on the mainnet server.
