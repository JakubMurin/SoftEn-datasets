1. Librarian accesses book editing section.
2. Library Management Interface requests book details.
3. Book Management System queries book details from Database.
4. Database returns book details to Book Management System.
5. Book Management System shows book details to Library Management Interface.
6. Library Management Interface displays book details to Librarian.
7. Librarian edits book and location information using Library Management Interface.
8. Library Management Interface sends updated details to Book Management System.
9. Book Management System updates book details and location in Database.
10. Database confirms the success of the update to Book Management System.
11. Book Management System confirms the success of the update to Library Management Interface.
12. Library Management Interface displays update confirmation to Librarian.
