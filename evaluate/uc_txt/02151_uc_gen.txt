1. Curator chooses 'Move Item' option.
2. User Interface requests form to move the item.
3. Collection Management System displays the move item form.
4. Curator submits the form with Item ID, Source Collection, and Target Collection.
5. User Interface validates and processes the move request.
6. Collection Management System fetches the item from Source Collection.
7. Database provides Item Details to Collection Management System.
8. Collection Management System removes the item from Source Collection.
9. Collection Management System confirms the removal.
10. Collection Management System adds the item to Target Collection.
11. Collection Management System confirms the addition.
12. Collection Management System displays success message.
13. User Interface confirms that the item has been moved.
