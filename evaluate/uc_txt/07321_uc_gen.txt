1. Faculty Member selects data to upload in the Upload Interface.
2. Upload Interface uploads the data file to the Repository System.
3. Repository System stores the data file in the Database.
4. Repository System requests metadata entry form from Metadata Service.
5. Metadata Service displays the form to the Faculty Member.
6. Faculty Member enters metadata, including publication details, in the Metadata Service.
7. Metadata Service updates metadata in the Database.
8. Repository System confirms data and publication association to the Faculty Member.
