1. Manager requests waiter schedules through the Manager Interface.
2. Manager Interface fetches waiter schedules from the Server Back-end.
3. Server Back-end queries the database for waiter schedules.
4. Database returns the waiter schedules to the Server Back-end.
5. Server Back-end returns the waiter schedules data to the Manager Interface.
6. Manager Interface displays the waiter schedules to the Manager.
