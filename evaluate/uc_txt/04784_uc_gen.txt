1. User clicks on the garbage bin icon next to the financial goal they want to delete.
2. User enters the needed information (amount) for deletion in the GUI.
3. GUI sends a request to the server to delete the financial goal with the provided ID.
4. Server forwards the deletion request to the database and removes the financial goal from the records.
5. Server sends back the updated financial account data to the GUI.
6. GUI displays the updated financial account details to the user.
