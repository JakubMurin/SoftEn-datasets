1. Archivist requests to restrict file.
2. Web Interface validates Archivist credentials.
3. Authorization Service confirms credentials valid.
4. Web Interface displays restriction options form.
5. Archivist submits restriction details.
6. Database updates file metadata with restriction details.
7. Database confirms metadata update.
8. Web Interface applies access restrictions through FileManagementSystem.
9. FileManagementSystem confirms restriction application.
10. Web Interface displays success message.
