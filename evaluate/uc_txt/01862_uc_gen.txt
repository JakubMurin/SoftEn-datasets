1. Manager selects "Monthly Popular Dishes" report.
2. UI Dashboard requests for the most frequently ordered food of the month.
3. Backend Server queries ordered food data of the month from Database.
4. Database returns ordered food data of the month to Backend Server.
5. Backend Server processes data to find the most frequently ordered dishes.
6. Backend Server sends the report of most frequently ordered food to UI Dashboard.
7. UI Dashboard displays the most frequently ordered food report to the Manager.
