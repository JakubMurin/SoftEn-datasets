1. Team member requests email templates from Recruitment Coordinator (RC).
2. Recruitment Coordinator checks existing templates in the Template Database (DB).
3. Template Database informs RC if templates are not available.
4. RC requests specific details from TM for template creation.
5. TM provides specific details to RC.
6. RC creates template draft.
7. RC submits template draft to Approval Authority (AA) for approval.
8. AA reviews template draft.
9. If revisions are needed, AA requests revisions from RC.
10. RC revises the template based on AA's feedback.
11. RC resubmits the revised template to AA for approval.
12. Upon approval, AA approves the template.
13. RC stores the new template in the database.
14. RC informs TM about the availability of the newly created template.
15. If templates are available, DB provides existing templates to RC.
16. RC shares existing templates with TM.
17. TM selects an appropriate template for customization.
18. TM customizes the selected template as per the current recruitment need.
19. TM initiates recruitment using the customized template.
