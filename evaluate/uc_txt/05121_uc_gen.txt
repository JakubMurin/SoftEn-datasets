1. Publisher requests preview of Data Package from UI Interface.
2. UI Interface verifies Publisher's identity with Publisher Account.
3. Publisher Account confirms Publisher's identity.
4. UI Interface prepares Data Package for Preview using Data Package Service.
5. Data Package Service validates Data Package with Validation Service.
6. Validation Service provides validation results to Data Package Service.
7. Data Package Service sends validated Data Package to Preview Service.
8. Preview Service generates preview link and sends it to UI Interface.
9. UI Interface displays the preview link to Publisher.
