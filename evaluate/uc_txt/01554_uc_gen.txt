1. Admin requests to view list of advertisements.
2. List Advertise View shows the list of advertisements.
3. Admin clicks on the edit button for an advertisement.
4. List Advertise View redirects to the edit advertise page.
5. Edit Advertise View displays the advertisement details.
6. Admin edits the advertisement data and clicks save.
7. Edit Advertise View sends the information to Ads Controller.
8. Ads Controller checks the data.
9. Ads Controller saves the data.
10. Ads Controller returns the saved data.
11. Edit Advertise View shows the result of saving the advertisement.
