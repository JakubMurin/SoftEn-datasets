1. Developer sends a JavaScript query to the JavaScript Engine.
2. JavaScript Engine activates to process the query.
3. JavaScript Engine forwards the query to the Query Engine.
4. Query Engine processes the query and returns the results to the JavaScript Engine.
5. JavaScript Engine returns the results to the Developer.
6. For each document in the query results:.
7. Developer operates on the document.
8. JavaScript Engine requests the document from the Query Engine.
9. Query Engine retrieves the document from the Database.
10. Database sends the document to the Query Engine.
11. Query Engine sends the document back to the JavaScript Engine.
