1. Faculty Data Steward accesses the DMPTool through the UI Interface.
2. The UI Interface requests login credentials from the Authentication Service.
3. The Authentication Service successfully authenticates the Faculty Data Steward.
4. Faculty Data Steward selects "View Roles" in the UI Interface.
5. The UI Interface fetches Roles data from the Role Management Module.
6. The Role Management Module requests Roles information from the DMPTool.
7. The DMPTool queries the Database for Roles data.
8. The Database provides Roles data to the DMPTool.
9. The DMPTool returns Roles information to the Role Management Module.
10. The Role Management Module displays the Roles information on the UI Interface.
11. The UI Interface shows the Roles sections to the Faculty Data Steward.
