1. User sends a GET request to Fishtext API for text generation specifying type and quantity.
2. Fishtext API returns the status and generated text upon success.
3. JavaScript displays the text on the HTML DOM.
4. User presses a key on the keyboard.
5. The keyboard outputs the pressed character on the DOM.
6. JavaScript receives the entered character.
7. JavaScript checks the length of all entered characters.
8. JavaScript starts a timer if the length is less than 2.
9. JavaScript records the current date if the length is 2 or more.
10. JavaScript checks the time elapsed and updates speed if a second has passed.
11. JavaScript calls the checkLetter function to validate the input.
12. JavaScript updates the DOM with appropriate classes based on input correctness.
13. JavaScript stops the timer for the attempt.
14. JavaScript calculates the points earned during the game.
15. JavaScript creates an object with game data.
16. JavaScript sends a POST request to the Server with game data.
17. Server retrieves the last user ID from the Database.
18. Server updates user data in the Database.
19. Server adds the attempt data to the Attempts table in the Database.
20. JavaScript updates the UI with end game information.
21. JavaScript generates speed data arrays for the graph.
22. JavaScript draws the speed diagram on the DOM.
23. JavaScript stores the data in localStorage.
