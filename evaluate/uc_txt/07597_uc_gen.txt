1. DonorRepresentative accesses the website.
2. User Interface requests access from the Authorization System.
3. Authorization System authenticates the DonorRepresentative.
4. DonorRepresentative requests image descriptive information.
5. User Interface queries File Management for images.
6. File Management fetches image metadata from the Database.
7. Database returns the metadata to File Management.
8. File Management displays the descriptive information to User Interface.
9. User Interface shows the descriptive information about images to DonorRepresentative.
