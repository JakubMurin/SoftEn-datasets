1. User creates an IncreasingOrder object.
2. User creates a DecreasingOrder object.
3. User initializes the scanner for looping.
4. Scanner sends a message to the user.
5. User inputs data into the scanner.
6. Scanner creates a switch object.
7. Switch redirects to IncreasingOrder or DecreasingOrder based on input.
8. SortingTemplate object is activated.
9. SortingTemplate processes the data using the Template Method pattern.
10. SortingTemplate sorts data in increasing or decreasing order using respective methods.
