1. El usuario encargado selecciona la opción "Cargar Nueva Categoría" en el menú.
2. El sistema muestra el menú de Categorías al usuario encargado.
3. El usuario encargado solicita cargar una nueva categoría.
4. El sistema solicita los datos necesarios para la nueva categoría al usuario encargado.
5. El usuario encargado ingresa los datos requeridos.
6. El sistema valida los datos ingresados por el usuario encargado.
7. El sistema carga la nueva categoría en el repositorio correspondiente.
8. El sistema confirma la operación de carga de la nueva categoría.
9. El sistema muestra un mensaje de confirmación al usuario encargado.
