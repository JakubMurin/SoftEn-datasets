1. Manager requests the inventory list.
2. Interface activates to retrieve the data.
3. Interface sends a request to the Inventory Management Module to retrieve the inventory data.
4. Inventory Management Module activates and queries the Database for the inventory list.
5. Database retrieves the inventory list data.
6. Database deactivates and sends the inventory data back to the Inventory Management Module.
7. Inventory Management Module receives the data and passes it on to the Interface.
8. Interface deactivates and displays the inventory list to the Manager.
