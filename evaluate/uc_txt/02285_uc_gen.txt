1. User requests property update for dataset.
2. Dataset Management System validates property update requirements.
3. Dataset Instance checks current version.
4. Version Control provides current version details.
5. Dataset Instance confirms migration required.
6. Dataset Management System confirms migration required to proceed.
7. User confirms migration.
8. Dataset Management System initiates migration.
9. Migration Tool performs migration (update properties, version).
10. Dataset Instance reports migration success.
11. Migration Tool reports migration completed.
12. Dataset Management System reports migration successful, properties updated.
