1. User sends a GET request to the Fishtext API for text generation specifying type and quantity.
2. Fishtext API returns the generated text upon successful request.
3. JavaScript displays the text on the HTML DOM.
4. User presses a key on the keyboard.
5. The keyboard input is displayed on the DOM.
6. JavaScript receives the entered character.
7. JavaScript checks the length of all entered characters.
8. JavaScript executes functionality based on character input correctness.
9. JavaScript stops the timer on attempt completion and calculates points.
10. JavaScript sends a POST request to the server with attempt data.
11. Server updates user data in the database.
12. Server adds the attempt data to the Attempts table.
13. JavaScript displays the end game window on the DOM.
14. JavaScript prepares speed data for the graph representation.
15. JavaScript renders the speed diagram on the DOM.
16. JavaScript stores the game data in the localStorage.
