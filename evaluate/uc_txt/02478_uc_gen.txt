1. Researcher accesses DMPTool Interface.
2. DMPTool Interface requests resource requirements input from IT Resources Estimation.
3. Researcher inputs project details (data volume, processing power, etc.) to IT Resources Estimation.
4. IT Resources Estimation returns resource estimates to DMPTool Interface.
5. DMPTool Interface initiates cost calculation with Cost Calculation Module.
6. Cost Calculation Module requests institutional pricing data from Institutional Systems.
7. Institutional Systems returns pricing data to Cost Calculation Module.
8. Cost Calculation Module returns cost estimate to DMPTool Interface.
9. DMPTool Interface displays cost estimate and resource needs to Researcher.
