1. Assigned User selects "Categories" from the menu.
2. Category Repository is activated.
3. Category Repository shows the category menu to the Assigned User.
4. Assigned User selects "Add New Category".
5. Category Repository requests data for the new category.
6. Assigned User enters the required data.
7. Category Repository validates the entered data.
8. Category Repository loads the new category.
9. Category Repository confirms the successful loading.
10. Assigned User confirms the operation.
11. Assigned User receives a confirmation message.
