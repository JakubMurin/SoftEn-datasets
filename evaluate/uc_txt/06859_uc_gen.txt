1. AppDeveloper initiates deployment.
2. Deployment Module checks for existing versions.
3. Version Control retrieves current versions of dataset type.
4. Dataset Management lists current versions.
5. Deployment Module confirms existing versions.
6. Deployment Module creates new version.
7. Version Control registers new dataset type version.
8. Dataset Management confirms version registration.
9. Version Control indicates a new version is created.
10. Deployment Module includes new dataset type in app artifact.
11. App Artifact Repository updates the app artifact.
12. Deployment is successful, and AppDeveloper is informed.
