1. CS receives help request from End-User.
2. CS provides NIN to Application.
3. Application sends NIN to Signicat.
4. CS informs End-User to start Swedish BankID.
5. End-User starts BankID.
6. End-User authenticates with BankID via IDP.
7. IDP informs Signicat about authentication.
8. Signicat processes the authentication.
9. Application requests authentication result from Signicat.
10. Application receives authentication success from Signicat.
11. CS provides authentication result to End-User.
12. CS assists the authenticated End-User.
