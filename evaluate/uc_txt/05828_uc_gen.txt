1. Buyer requests to update profile information.
2. UI sends update profile request to Web server.
3. Web server forwards the request along with data to App server for validation.
4. App server validates and saves the profile changes to the database.
5. Database updates user profile information.
6. App server receives confirmation from the database.
7. Web server records profile update completion.
8. UI confirms the successful profile update to the Buyer.
