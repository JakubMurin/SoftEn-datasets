1. User displays the main menu.
2. User selects Menu 1 or 2 and chooses Aku or Dia.
3. Aku displays his profile.
4. User enters the quantity.
5. Quantity information is printed.
6. User returns to the main menu.
7. If not selected, go back to the main menu.
8. Dia displays his profile.
9. User enters the quantity.
10. Quantity information is printed.
11. User returns to the main menu.
12. If not selected, go back to the main menu.
