1. Manager selects "Manage Book Categories".
2. UI Interface validates Manager Role.
3. Authorization confirms Access Granted.
4. UI Interface displays Book Categories.
5. Manager selects categories to delete.
6. UI Interface requests Category Deletion from Book Category Management.
7. Book Category Management deletes Category Records in Database.
8. Database confirms deletion.
9. Book Category Management displays Success Message on UI Interface.
10. UI Interface provides a success update to Manager.
