1. Agency User submits data to Web Interface for processing.
2. Web Interface initiates data submission to Broker Backend.
3. Broker Backend validates submission data with Data Validator.
4. Data Validator passes validation.
5. Data Deriver derives data elements.
6. Data Deriver processes derivation rules.
7. Data Deriver saves derived elements to Database.
8. Database confirms saved elements.
9. Broker Backend acknowledges derivation success.
10. Web Interface displays submission success message.
11. Agency User sees success message.
