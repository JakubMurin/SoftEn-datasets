1. Researcher defines criteria for participant selection.
2. Recruitment Platform queries potential participants based on the criteria.
3. User Database returns a list of potential participants.
4. Recruitment Platform sends recruitment invitations to participants.
5. Participant responds to the invitation.
6. Researcher receives a list of participating users.
7. Researcher conducts user research sessions with participants.
8. Participant provides feedback and data to the researcher.
9. Researcher updates project status to "Complete".
10. Researcher inputs user research data into the Research Analysis System.
11. Analysis System analyzes and summarizes research findings.
12. Analysis System provides synthesized results to the researcher.
13. Researcher archives project data and findings in the Analysis System.
