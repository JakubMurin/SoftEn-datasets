1. The user navigates to the financial goal section and clicks on the garbage bin icon visible only to the goal owner.
2. The user enters the amount to be deleted.
3. The GUI sends a delete request with the financial goal ID to the server.
4. The server processes the delete request and sends a delete request to the database.
5. The database deletes the financial goal using the provided ID.
6. The database returns the updated financial account data to the server.
7. The server returns the updated financial account data to the GUI.
8. The updated financial account details are displayed to the user.
