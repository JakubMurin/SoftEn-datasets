1. Curator requests item referrer data from EDRMS Interface.
2. EDRMS Interface queries for item ID from Database.
3. Database returns item ID to EDRMS Interface.
4. EDRMS Interface requests referrer stats for item ID from Analytics Engine.
5. Analytics Engine queries referrer logs from Database.
6. Database returns referrer data to Analytics Engine.
7. Analytics Engine provides summarized referrer info to EDRMS Interface.
8. EDRMS Interface displays referrer data to Curator.
