1. FacultyMember navigates to the dashboard.
2. Repository Dashboard requests permission update page from Permission Service.
3. Permission Service displays permission update UI.
4. Repository Dashboard displays UI to FacultyMember.
5. FacultyMember pastes a list of NetIDs.
6. Repository Dashboard validates and processes NetIDs.
7. Permission Service updates permissions for NetIDs in Repository DB.
8. Permission Service displays confirmation message to Repository Dashboard.
9. Repository Dashboard shows confirmation that DDR recognizes the NetIDs to FacultyMember.
