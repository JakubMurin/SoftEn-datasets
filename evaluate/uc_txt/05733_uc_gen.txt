1. User opens the mini program.
2. Mini program loads the system messages module.
3. Mini program requests the system messages module to fetch system messages.
4. System messages module queries and returns the unread message list to the mini program.
5. Mini program displays a notification for unread messages.
6. User clicks to view messages.
7. User views the message details.
8. Mini program requests the system messages module to fetch the message content.
9. System messages module queries and returns the message content to the mini program.
10. Mini program displays the message content.
