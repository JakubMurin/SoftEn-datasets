1. User inserts card into ATM.
2. User enters PIN.
3. ATM verifies PIN with Bank.
4. ATM verifies and approves PIN.
5. User selects Balance Inquiry or Withdrawal or Cancel.
6. If Balance Inquiry selected, ATM displays balance.
7. If Withdrawal selected, ATM requests withdrawal amount.
8. ATM asks for confirmation to withdraw.
9. If Sufficient Funds, ATM processes withdrawal and dispenses cash.
10. If Insufficient Funds, ATM displays Insufficient Funds message.
11. If Cancel selected, ATM cancels transaction.
12. User removes card from ATM.
