1. Owner requests a sales report.
2. Frontend Interface sends API call to Backend System for sales data.
3. Backend System queries Reporting Module to generate sales report.
4. Reporting Module fetches sales data from Database.
5. Sales data is sent from Database to Reporting Module.
6. Sales report is generated by Reporting Module.
7. Backend System displays sales report data to Frontend Interface.
8. Frontend Interface shows sales report to the Owner.
