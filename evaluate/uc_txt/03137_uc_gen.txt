1. O User requisita a página dos andares ao SPA.
2. O SPA retorna a página dos andares ao User.
3. O User seleciona um andar da tabela no SPA.
4. O SPA mostra os dados do andar selecionado chamando a API.
5. A API mostra os edifícios numa select box para o SPA.
6. O SPA mostra os dados do andar selecionado.
7. O User insere um ficheiro com o floormap do andar selecionado no SPA.
8. O SPA faz um patch na API para atualizar o /api/floorMap com os dados do ficheiro.
