1. User requests property update for dataset.
2. Dataset Management System validates property update requirements.
3. Dataset Instance checks current version.
4. Version Control provides current version details.

6. Dataset Management System requests confirmation to proceed with migration.

10. Dataset Instance reports migration success.
11. Migration Tool notifies completion of migration.
