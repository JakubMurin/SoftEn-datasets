1. End-User requests help.
2. End-User provides NIN.
3. CS receives NIN from End-User.
4. CS informs End-User to start Swedish BankID.
5. Application sends NIN to Signicat.
6. Signicat receives NIN.
7. End-User starts BankID.
8. End-User authenticates with BankID via IDP.
9. IDP provides authentication information to Signicat.
10. Signicat processes the authentication.
11. Application requests authentication result from Signicat.
12. Signicat provides authentication result to Application.
13. CS provides the authentication result to End-User.
14. CS helps the authenticated End-User.
