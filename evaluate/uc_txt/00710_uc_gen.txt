1. Repository Manager selects an object to list components.
2. DIMS UI receives the request to list components.
3. DIMS Backend queries the DAMS Database for components of the object.
4. DAMS Database returns components data to DIMS Backend.
5. DIMS Backend sorts components in their appropriate order.
6. DIMS Backend shows sorted components to DIMS UI.
7. DIMS UI displays components in order to the Repository Manager.
