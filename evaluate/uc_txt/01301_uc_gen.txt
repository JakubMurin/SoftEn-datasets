1. User writes username in the username input field.
2. PacManController updates the GUI.
3. PacManController retrieves the username from the input field.
4. PacManController receives the username from the input field.
5. PacManController validates the username.
6. PacManUser confirms the username validation.
7. PacManController enables the start button.
8. User clicks on the start button.
9. Start button click is handled by PacManController.
10. PacManController sets the username in the game.
11. PacManController sets the initial score.
12. Game starts.
13. Game is over.
14. PacManController saves the highscore for the user.
15. PacManPersistence accesses the file reader.
16. FileReader retrieves the scores data.
17. PacManPersistence creates a new user with username and score.
18. PacManUser receives the new user data.
19. PacManPersistence accesses the file writer.
20. FileWriter updates the scores data with the new user.
