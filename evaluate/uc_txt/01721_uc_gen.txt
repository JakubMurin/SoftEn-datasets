1. Customer selects "My Orders" from the UI interface.
2. UI requests a list of orders from Backend.
3. Backend fetches customer orders from the Database.
4. Database returns orders to Backend.
5. Backend displays orders on UI.
6. UI shows the list of orders to Customer.
7. Customer selects a dish from an order.
8. UI requests dish details from Backend.
9. Backend fetches dish details from Database.
10. Database returns dish details to Backend.
11. Backend displays dish details on UI.
12. UI shows the dish details to Customer.
13. Customer enters a comment and submits it.
14. UI posts the comment to Backend.
15. Backend saves the comment to the dish in Database.
16. Database confirms the successful save to Backend.
17. Backend displays a success message on UI.
18. UI shows a confirmation of the comment submission to Customer.
