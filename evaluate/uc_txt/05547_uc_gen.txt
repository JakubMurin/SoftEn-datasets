1. User inserts their card details to proceed with the ticket purchasing.
2. User selects the show date from the available options.
3. Theater Portal offers the user available seating options.
4. User submits the ticket order confirming the selection.
5. Theater Portal submits the order to the Theater Server for processing.
6. Theater Server confirms the order with the Theater Portal.
7. Theater Portal confirms the ticket order with the User.
