1. Member requests to share a work.
2. Application retrieves file share form.
3. UIM displays file share form.
4. Member views the form.
5. Application shows the form.
6. Member fills in content details and adds file.
7. Member sends the form.
8. Application analyzes the file.
9. Application displays file analysis result to Member.
10. Application requests confirmation for submission.
11. Member validates the submission.
12. Application copies file to "To moderate" directory.
13. Application generates transaction number.
14. Application logs the transaction.
15. Application transmits transaction and file to server.
16. Server notifies Librarian of work submission.
17. Application notifies Member of moderation wait.
