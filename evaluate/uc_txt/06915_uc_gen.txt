1. Content Editor enters search query with keywords/tags.
2. CMS Search Interface queries CMS Database for content.
3. CMS Database retrieves search results.
4. CMS Search Interface displays search results to Content Editor.
5. Content Editor selects content from search results.
6. CMS Search Interface requests selected content from CMS Database.
7. CMS Database retrieves content details for selected content.
8. CMS Search Interface displays content in editor on Content Page.
9. Content Editor edits the content.
10. Content Page saves changes to CMS Database.
11. CMS Database confirms changes are saved.
12. Content Page displays confirmation message of successful edit.
