1. Player opens the game interface.
2. Game interface loads the map and robot positions.
3. Game controller initializes the game state.
4. Game controller displays the initial map and robot positions.
5. Player clicks on the edit instructions button.
6. Game interface enters the instruction editing interface.
7. Player selects an instruction.
8. Game interface adds the selected instruction.
9. Player continues selecting and adding instructions.
10. Game interface requests to continue selecting and adding instructions.
11. Player selects an instruction.
12. Game interface adds the selected instruction.
13. Player decides whether to continue selecting and adding instructions.
14. Game interface completes the instruction editing.
15. Player clicks on the start button.
16. Game interface starts the game.
17. Game logic begins executing the instruction sequence.
18. Game controller updates the game status.
19. Player exits the game.
20. Game interface handles the game exit.
