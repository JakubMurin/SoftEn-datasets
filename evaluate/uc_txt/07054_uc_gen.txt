1. RepoAdmin requests data synchronization.
2. Symplectic Elements initiates bidirectional communication.
3. Fedora Connector verifies access to hidden Fedora instance.
4. Fedora Connector receives confirmation and data provision from Fedora Instance.
5. Fedora Connector requests object collection from Hydra.
6. Hydra collects objects from Fedora Instance.
7. Hydra provides collected objects to Fedora Connector.
8. Fedora Connector updates Symplectic Elements with new and modified objects.
9. Symplectic Elements completes synchronization.
