1. Admin requests to view the list of advertisements.
2. List Advertisements View displays the list of advertisements.
3. Admin clicks on the edit button of an advertisement.
4. Redirected to the Edit Advertisement View.
5. Edit Advertisement View displays the content of the advertisement.
6. Admin edits the advertisement data and clicks save button.
7. Edit Advertisement View sends the information.
8. Ads Controller checks the data.
9. Ads Controller saves the data.
10. Ads Controller returns the saved data.
11. Edit Advertisement View displays the edited advertisement.
