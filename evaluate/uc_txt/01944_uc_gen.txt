1. DataManager requests data type information.
2. DMPTool Interface queries stored projects metadata.
3. Metadata Repository returns metadata including data types.
4. DMPTool Interface displays data types to DataManager.
5. DataManager selects a data type.
6. DMPTool Interface queries required support services for the selected data type.
7. Support Services Module returns a list of basic support services and functionalities.
8. DMPTool Interface displays support services and functionalities to DataManager.
9. DataManager confirms understanding and requirements.
10. DMPTool Interface logs interaction and requirements confirmation with Metadata Repository.
