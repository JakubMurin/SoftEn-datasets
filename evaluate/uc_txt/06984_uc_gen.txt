1. User searches for the restricted object in the Repository Interface.
2. Interface requests Access Details from the Access Control System.
3. ACS displays the "Contact Curator" option on the Interface.
4. User clicks on "Contact Curator".
5. Interface generates an Email Template.
6. User fills in the Template and sends the access request to the Email Notification System.
7. Email forwards the User's request to the Collection Curator.
8. Curator reviews the request.
9. If approved, Curator updates access permissions on ACS.
10. ACS notifies the User of the access decision through Email.
11. Email displays the access decision notification to the User.
