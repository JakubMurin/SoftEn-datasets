1. Curator accesses the item/component page.
2. UI Dashboard validates the access.
3. Authorization Module grants access.
4. UI Dashboard requests item/component details.
5. Item Retrieval Module queries the Database for details and deletion date.
6. Database returns the requested data.
7. Item Retrieval Module displays item/component details and deletion date.
8. UI Dashboard shows the item/component page with deletion date to the Curator.
