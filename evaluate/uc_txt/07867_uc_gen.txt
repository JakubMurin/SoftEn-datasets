1. Send reminder to return book (1 day before due date) to Reader.
2. If overdue, send immediate reminder to return book to Reader.
3. Reader puts book for scanning or enters book code.
4. System checks book code in Book Database.
5. If returned on time, display "Book returned successfully" to Reader.
6. If returned late, display number of days overdue and penalty amount to Reader.
7. Display Library's QR code for payment to Reader.
8. Reader makes late-fee payment via Payment Gateway.
9. Payment Gateway confirms payment completion to System.
10. Display "Book return completed" message to Reader.
11. Update borrowing history in Book Database (Book name, on-time/late status).
12. System confirms history update with Book Database.
13. Switch to voting screen for Reader.
