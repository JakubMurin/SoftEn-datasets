1. User opens BLE Mailbox Central.
2. User requests scan.
3. BLE Mailbox Central requests permissions.
4. Permissions granted by the user.
5. BLE Mailbox Central scans for devices.
6. User selects BLE Mailbox Peripheral.
7. BLE Mailbox Central requests connection.
8. Connection handshaking between Central and Peripheral.
9. BLE Mailbox Central requests state of locker.
10. Response received with locker state from Peripheral.
11. User signs in.
12. BLE Mailbox Central signs in with Peripheral.
13. Peripheral authenticates user.
14. BLE Mailbox Central requests state of authentication.
15. Response received with authentication state from Peripheral.
16. User requests to change state of locker.
17. BLE Mailbox Central requests to change state of locker.
18. State of locker successfully changed by Peripheral.
19. BLE Mailbox Central requests state of locker.
20. Response received with updated locker state from Peripheral.
