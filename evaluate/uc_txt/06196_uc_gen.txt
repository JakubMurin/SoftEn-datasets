1. Repository Manager accesses Replication Policy Settings.
2. EDRMS Interface retrieves Current Policy Settings.
3. Replication Policy Module queries Existing Policies in EDRMS Database.
4. Replication Policy Module shows Current Policies to EDRMS Interface.
5. EDRMS Interface displays Current Policies to Repository Manager.
6. Repository Manager defines/updates Policy for Collection.
7. EDRMS Interface processes Policy Specifications.
8. Replication Policy Module updates Policy in EDRMS Database.
9. Replication Policy Module initiates Policy Replication Update with Remote Storage Service.
10. Remote Storage Service acknowledges Replication Policy Update.
11. Replication Policy Module confirms Policy Update to EDRMS Interface.
12. EDRMS Interface shows Update Confirmation to Repository Manager.
