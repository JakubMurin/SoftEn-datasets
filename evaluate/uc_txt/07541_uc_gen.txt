1. O User requisita a página dos andares ao SPA.
2. O SPA retorna a página dos andares ao User.
3. O User seleciona um andar da tabela no SPA.
4. O SPA mostra os dados do andar selecionado chamando a API.
5. A API responde com os dados do andar selecionado.
6. O SPA solicita a lista de edifícios à API.
7. A API retorna a lista de edifícios em uma select box para o SPA.
8. O SPA mostra os dados do andar selecionado.
9. O User insere um ficheiro com o floormap do andar selecionado no SPA.
10. O SPA envia o ficheiro para a API para atualização.
