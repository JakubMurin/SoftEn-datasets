1. Donor Representative accesses the website.
2. User Interface requests access to the Authorization System.
3. Authorization System authenticates the Donor Representative.
4. Donor Representative requests descriptive information about the image.
5. User Interface queries for images from the File Management system.
6. File Management system fetches image metadata from the Database.
7. Database returns the image metadata to the File Management system.
8. File Management system displays the descriptive information about the images on the User Interface.
9. User Interface shows the descriptive information about the images to the Donor Representative.
