1. Member selects to Share artwork.
2. Application retrieves File Share Form.
3. UI Manager shows the File Share Form.
4. Member views the Form.
5. Member fills in content name, content type, and selects file.
6. Member adds the file.
7. Member sends the form.
8. Application analyzes the file.
9. Application displays the file analysis result to Member.
10. Application requests confirmation for the submission.
11. Member confirms the submission.
12. Application copies the file to the "To moderate" directory.
13. Application generates a transaction number.
14. Application logs the transaction.
15. Application transmits the transaction and file to the Server.
16. Server notifies the Librarian of the submission.
17. Application notifies the Member of the waiting time for moderation.
