1. Manager accesses the Message Schedule option in the UI Dashboard.
2. UI Dashboard requests a Message Input Form from the Messaging System.
3. Messaging System displays the Input Form to the UI Dashboard.
4. Manager submits the Message Details including Timeframe and Content.
5. UI Dashboard schedules the Message Display with the Schedule Manager.
6. Schedule Manager saves the Scheduled Message in the Repository Database.
7. Repository Database acknowledges the save operation to the Schedule Manager.
8. Schedule Manager confirms the Message Scheduled with the Messaging System.
9. UI Dashboard shows a Confirmation to the Manager.
10. UI Dashboard displays a Success Confirmation to the Manager.
