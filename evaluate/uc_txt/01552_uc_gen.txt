1. Admin requests to view the list of advertisements.
2. List advertisements view shows the list of advertisements.
3. Admin clicks on the edit button on an advertisement.
4. Redirects to the edit advertisement page.
5. Edit advertisement view displays the advertisement content.
6. Admin edits the advertisement data and clicks save.
7. Edit advertisement view sends data to AdsController.
8. AdsController checks the data.
9. AdsController saves the data to AdsModel.
10. AdsModel returns the saved data.
11. AdsController sends the result to edit advertisement view.
12. Edit advertisement view displays the edited advertisement.
