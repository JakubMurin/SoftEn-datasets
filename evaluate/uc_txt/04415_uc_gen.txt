1. Player launches the game.
2. Player interacts with the start game menu.
3. Player presses the exit button in the menu.
4. Start game menu checks the user's choice.
5. Start game menu starts the game.
6. Game frame is displayed.
7. If the player loses, Game end menu is shown.
8. Player returns to the menu from Game end menu.
9. Player starts a new game from Game end menu.
10. Start game menu deletes game results from the database.
11. Database confirms deletion.
12. Player views the best game results in Results frame.
13. Results frame retrieves results from the database.
14. Database displays the results to Results frame.
15. Player returns to the menu from Results frame.
