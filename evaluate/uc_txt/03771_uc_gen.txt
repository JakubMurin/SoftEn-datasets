1. Receptionist requests to create a booking.
2. UI displays booking form.
3. Receptionist fills in booking details.
4. UI sends booking data to create a booking.
5. Router forwards booking request to Controller.
6. Controller creates a new booking in the system.
7. Repository creates a new booking in the system.
8. Room availability is checked.
9. If room is available, a new booking is created.
10. If room is not available, an error message is displayed.
