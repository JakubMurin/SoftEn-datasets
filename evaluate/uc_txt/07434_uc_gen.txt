1. Curator chooses 'Move Item' option.
2. User Interface requests form to move the item.
3. Collection Management System displays the move item form.
4. Curator submits the form with information:
            - Item ID
            - Source Collection
            - Target Collection .
5. User Interface validates and processes the move request.
6. CMS fetches the item from the Source Collection.
7. Database provides item details to CMS.
8. CMS removes the item from the Source Collection.
9. CMS confirms the removal of the item.
10. CMS adds the item to the Target Collection.
11. Database confirms addition of the item to the Target Collection.
12. CMS displays a success message indicating the successful move.
13. UI confirms to Curator that the item has been successfully moved.
