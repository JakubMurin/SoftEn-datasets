1. User requests scan.
2. BLE mailbox central requests permissions.
3. User grants permissions.
4. BLE mailbox central scans for devices.
5. User selects BLE mailbox peripheral.
6. BLE mailbox central requests connection.
7. Connection handshaking between central and peripheral.
8. BLE mailbox central requests state of locker.
9. Peripherals responds with locker state.
10. User signs in.
11. BLE mailbox central signs in.
12. Peripheral authenticates user.
13. BLE mailbox central requests state of authentication.
14. Peripheral responds with authentication state.
15. User requests change in locker state.
16. BLE mailbox central requests locker state change.
17. Peripherals changes locker state.
18. BLE mailbox central requests state of locker.
19. Peripheral responds with locker state.
