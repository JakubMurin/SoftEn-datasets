1. AccreditedRepresentative has to access POA Request List View on the vets-website React Frontend.
2. Frontend displays a list of VSO POA Requests.
3. AccreditedRepresentative is verified to be logged in.
4. Backend sends request to VAGovDB to fetch AR user accounts for VSOs with specified conditions.
5. Backend retrieves and processes POA Request List data and returns to Frontend.
6. Frontend receives the POA Request List response and displays it.
