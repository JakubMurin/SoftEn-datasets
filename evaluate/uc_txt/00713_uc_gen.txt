1. The Repository Manager selects an object to list components.
2. The DIMS UI sends a request to the DIMS Backend to list components.
3. The DIMS Backend queries the DAMS Database for components of the object.
4. The DAMS Database returns the components data to the DIMS Backend.
5. The DIMS Backend sorts components in their appropriate order.
6. The DIMS Backend shows the sorted components to the DIMS UI.
7. The DIMS UI displays the components in order to the Repository Manager.
