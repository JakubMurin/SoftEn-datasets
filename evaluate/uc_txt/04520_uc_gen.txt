1. Administrator enters the book ID for deletion.
2. Web Client sends a DELETE request to the Server for the specified book ID.
3. Server searches for the book in the Database based on the ID.
4. Database finds the book based on the ID.
5. Server deletes the book from the Database.
6. Database confirms successful deletion of the book.
7. Server notifies the Web Client that the book was successfully deleted.
8. Web Client informs the Administrator about the successful deletion.
