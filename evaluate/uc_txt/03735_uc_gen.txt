1. User requests text generation from Fishtext API.
2. Fishtext API returns text to JavaScript.
3. JavaScript displays text on HTML DOM.
4. User presses a key on the keyboard.
5. Keyboard inputs the character into HTML DOM.
6. JavaScript receives the inputted character.
7. JavaScript checks the length of all entered characters.
8. JavaScript updates input status based on correctness (correct, incorrect, backspace).
9. JavaScript stops the timer and calculates game points.
10. JavaScript creates an object with game data and sends it to the Server.
11. Server processes the current attempt and updates the database.
12. JavaScript displays end game window on HTML DOM.
13. JavaScript prepares data arrays for speed graph and draws the diagram on HTML DOM.
14. JavaScript stores game data in localStorage.
