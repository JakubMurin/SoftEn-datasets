1. User clicks Court Details link on Account Details.
2. Account Details navigates to Court Details.
3. Local Justice Areas API (Get Local Justice Areas) retrieves Local Justice Areas records from Fines Database using JPA.
4. Court Details populates Local Justice Areas auto-complete.
5. Fines API Get Courts retrieves Courts records from database using JPA.
6. Court Details populates Courts auto-complete.
7. User enters Court details and clicks Return to Account Details on Court Details.
8.  Court Details navigates back to Account Details.
