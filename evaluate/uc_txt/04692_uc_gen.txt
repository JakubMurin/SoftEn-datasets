1. Usuario initiates session.
2. Plataforma verifies credentials.
3. Plataforma confirms login to Usuario.
4. Usuario requests personalized content.
5. Plataforma analyzes user profile.
6. Sistema de Recomendación IA obtains learning history from Base de Datos.
7. Base de Datos sends history to Sistema de Recomendación IA.
8. Sistema de Recomendación IA sends recommendations to Plataforma.
9. Plataforma displays recommended content to Usuario.
10. Usuario completes activity.
11. Plataforma saves progress in Base de Datos.
12. Base de Datos confirms progress saving to Plataforma.
