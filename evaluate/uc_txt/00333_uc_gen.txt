1. Member requests to share a work.
2. Application gets file sharing form.
3. User Interface Manager displays file sharing form.
4. Member views file sharing form.
5. Member fills form with content name, type, and file.
6. Member adds file.
7. Member sends filled form.
8. Application analyzes the file.
9. Application displays file analysis result to Member.
10. Application asks Member for confirmation to send.
11. Member confirms send.
12. Application copies file to "To moderate" directory.
13. Application generates transaction number.
14. Application logs transaction.
15. Application transmits transaction and file to Server.
16. Server notifies Librarian of work submission.
17. Application notifies Member of wait for moderation.
