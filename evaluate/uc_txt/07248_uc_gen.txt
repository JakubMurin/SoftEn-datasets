1. User opens the game.
2. Game Interface loads the map and robot's position.
3. Game Controller initializes the game state.
4. Game Controller displays the initial map and robot's position.
5. User clicks on the command editing button.
6. Game Interface enters the command editing interface.
7. User selects a command.
8. Game Interface adds the command.
9. User continues selecting and adding commands.
10. Game Interface requests to continue selecting and adding commands.
11. Game Interface allows the user to continue selecting and adding commands.
12. User selects a command.
13. Game Interface adds the command.
14. User continues selecting and adding commands.
15. Game Interface asks if the user wants to continue selecting and adding commands.
16. Game Interface completes the command editing.
17. User clicks on the start button.
18. Game Interface starts the game.
19. Game Logic begins executing the command sequence.
20. Game Controller updates the game state.
21. User decides to exit the game.
22. Game Interface initiates the game exit.
23. Game Logic cleans up the game state.
24. Game Logic confirms successful game exit.
25. Game Controller confirms successful game exit.
