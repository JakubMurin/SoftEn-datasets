1. User clicks on "Submit" question button.
2. View executes "Submit Question".
3. Route executes quiz().
4. View validates the submission.
5. Questions submit the response.
6. Model retrieves student proficiency.
7. Model saves proficiency and student data to the database.
8. Model retrieves item responses from proficiency.
9. Adaptive Testing updates student proficiency.
10. Model corrects the response and answer.
11. Views display the result to the user.
12. Route directs to the result page.
13. View renders the result page.
