1. Employee accesses the "Absences" section in the navigation menu.
2. The system routes the request to the controller for processing.
3. The controller fetches all absences associated with the Employee's ID.
4. The controller queries the database for the absences data.
5. The database returns the rows of absences data to the controller.
6. The controller renders the "Absences List" view with the retrieved data.
7. The system displays the HTML view of the absences list to the Employee.
