1. Player launches the game.
2. Player interacts with the start game menu.
3. Player presses the exit button in the start game menu.
4. Start game menu initiates the game.
5. Game frame is displayed.
6. If the player loses, Game end menu is shown.
7. Player returns to the start game menu from the game end menu.
8. Player starts a new game from the game end menu.
9. Start game menu deletes game results from the database.
10. Results frame displays the best game results.
11. Player returns to the start game menu from the results frame.
