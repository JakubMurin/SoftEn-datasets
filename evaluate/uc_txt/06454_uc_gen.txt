1. User starts the application by invoking the main function.
2. The main function triggers the execution of managing department information.
3. Company, HR, IT, Development, QA, and all employee objects are instantiated.
4. Employees with their respective budgets are added to HR, Development, and QA departments.
5. The system prints the name and budget of the company.
6. The system displays the total number of workers in the company.
