1. User opens the Add Event Page.
2. User enters the event details.
3. User selects the associated contact.
4. User submits the event form.
5. Add Event Page sends event details to the server.
6. Server validates the details.
7. Server saves the event.
8. Database reports event as saved to the server.
9. Server sends a success response to the Add Event Page.
10. Add Event Page shows a success message to the User.
11. Add Event Page sends an error response to the server.
12. Add Event Page shows an error message to the User.
13. Add Event Page prompts the User to re-enter event details.
