1. User opens the "Messages Page".
2. "Messages Page" triggers "Get Messages by inbox ID" activity.
3. "Get Messages by inbox ID" activity sends request to the Message database to get messages.
4. Message database returns messages to the "Get Messages by inbox ID" activity.
5. "Get Messages by inbox ID" activity provides messages to the "Messages Page".
6. "Messages Page" displays the conversations to the user.
