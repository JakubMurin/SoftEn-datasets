1. User inputs a number.
2. GuessNumber sets the guess to the input number.
3. User clicks "Adivinar".
4. GuessNumber parses the guess number.
5. GuessNumber sets message based on guess vs targetNumber.
6. GuessNumber displays the message to the User.
7. User clicks "Reiniciar".
8. GuessNumber generates a new random target number.
9. GuessNumber resets the guess to an empty string.
10. GuessNumber resets the message to an empty string.
11. User clicks "Back to Home".
12. User invokes onBackToHome.
