1. Researcher logs into the EDRMS system.
2. System authenticates the researcher.
3. Researcher navigates to My Deposited Items.
4. System fetches the researcher's deposited items.
5. System displays the deposited items to the researcher.
6. Researcher selects an item to edit.
7. System checks if the researcher has permission to edit the item.
8. Permission is granted to the researcher.
9. Researcher edits the metadata of the selected item.
10. System updates the metadata of the item in the Repository Database.
11. Repository Database saves the changes.
12. System confirms the successful update of metadata to the researcher.
