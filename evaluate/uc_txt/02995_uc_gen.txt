1. User opens BLE Mailbox Central.
2. User requests a scan for devices.
3. BLE Mailbox Central requests permissions.
4. Permissions are granted by the user.
5. BLE Mailbox Central scans for devices.
6. User selects BLE Mailbox Peripheral.
7. BLE Mailbox Central requests connection to Peripheral.
8. Connection handshaking between Central and Peripheral.
9. BLE Mailbox Central requests state of the locker.
10. BLE Mailbox Peripheral responds with the state of the locker.
11. User signs in to the locker.
12. BLE Mailbox Central signs in to the Peripheral.
13. BLE Mailbox Peripheral authenticates the user.
14. BLE Mailbox Central requests state of authentication.
15. BLE Mailbox Peripheral responds with the state of authentication.
16. User requests to change the state of the locker.
17. BLE Mailbox Central requests to change the state of the locker.
18. BLE Mailbox Peripheral changes the state of the locker.
19. BLE Mailbox Central requests state of the locker.
20. BLE Mailbox Peripheral responds with the state of the locker.
