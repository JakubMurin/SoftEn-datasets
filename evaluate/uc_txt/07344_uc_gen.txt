1. Depositor accesses the DOI Information Page.
2. Web Interface receives the request to access DOI Info.
3. Backend Server queries the DOI Assignment Process in the DOI Information Database.
4. DOI Information Database returns the DOI Process Info to Backend Server.
5. Backend Server requests required Metadata Fields from Metadata Service.
6. Metadata Service returns the required Metadata Fields to Backend Server.
7. Backend Server responds with DOI and Metadata Info to Web Interface.
8. Web Interface displays the DOI and Metadata Information to Depositor.
