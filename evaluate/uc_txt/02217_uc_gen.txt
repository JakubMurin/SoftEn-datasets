1. Library Staff accesses the collection creation form.
2. Browser Interface receives the request for the collection creation form from Library Staff.
3. Server Side Application returns the collection creation form to the Browser Interface.
4. Library Staff fills and submits the form with custom metadata fields.
5. Browser Interface submits the collection creation request with custom metadata to Server Side Application.
6. Server Side Application validates the data and saves the collection in the Database.
7. Database confirms the successful saving of the collection to the Server Side Application.
8. Server Side Application displays a success message to the Browser Interface.
9. Browser Interface shows the success message and collection details to the Library Staff.
