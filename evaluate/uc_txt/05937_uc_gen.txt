1. Faculty Member enters credentials.
2. Credentials are validated by Authentication Service.
3. Authentication is successful.
4. Faculty Member accesses the Search Interface.
5. Search Interface displays search options.
6. Faculty Member inputs search criteria for the collection.
7. Search Interface queries the database with the criteria.
8. Database returns search results.
9. Search Interface displays list of matching collections.
10. Faculty Member selects desired collection.
11. Search Interface requests selected collection details from the database.
12. Database retrieves collection details.
13. Search Interface displays the selected collection content.
