1. User initializes the main application.
2. Main instantiates the Labyrinthe object by reading from a file.
3. Labyrinthe reads the file to set up the labyrinth.
4. Labyrinthe requests the Monstre initialization with given coordinates.
5. Monstre sets its initial position to the given coordinates.
6. Monstre returns back to Labyrinthe.
7. Labyrinthe responds to Main.
8. Main displays the labyrinth.
