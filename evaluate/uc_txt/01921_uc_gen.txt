1.  The Logged-in User clicks on "+ add order" under "recurring transactions" .
2.  The GUI Fin. Acc. Details View opens the Recurring Transaction Order Create Form .
3.  The Logged-in User enters recurring transaction order details .
4.  The GUI Recurring Transaction Order Create Form creates a new recurring transaction order based on user inputs .
5.  The Server inserts the new recurring transaction order into the database .
6.  The GUI Recurring Transaction Order Create Form reloads the view .
7.  Show success message .
