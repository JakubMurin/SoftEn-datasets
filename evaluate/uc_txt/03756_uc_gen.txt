1. Archivist submits credentials to the Authentication System.
2. Authentication System validates the credentials.
3. User Management System retrieves Archivist permissions.
4. Access Restriction Controls verify permissions.
5. Access Restriction Controls permit access to FileManager.
6. FileManager fetches the requested files from XDrive.
7. XDrive returns the requested files to FileManager.
8. FileManager displays the files to the Archivist.
