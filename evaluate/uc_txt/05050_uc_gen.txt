1. User clicks 'Comenzar' button.
2. App handles the event.
3. StartScreen handles the event.
4. StartButtonEventHandler handles the event.
5. StartScreen sets visibility to false.
6. PlayersInputScreen is displayed to select and confirm inputs.
7. User selects and confirms inputs.
8. PlayersInputScreen handles the event.
9. ConfirmButton handles the event.
10. PlayersInputScreen requests to select data.
11. ConfirmButton selects data.
12. Game loads received data.
13. PlayersNamesInputScreen is created.
14. App shows the screen asking for player names.
15. User enters and confirms player names.
16. PlayersNamesInputScreen handles the event.
17. NamesButton handles the event.
18. PlayersNamesInputScreen executes the event.
19. Game registers users with entered names.
20. For each entered name, a new Player is created and added to the game.
21. QuestionScreen is created.
22. App shows the screen with questions and options.
23. App shows questions for each player.
