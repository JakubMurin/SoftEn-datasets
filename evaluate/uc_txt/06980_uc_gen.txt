1. User searches and selects a restricted object.
2. User requests access details from the Repository Interface.
3. Repository Interface displays "Contact Curator" option.
4. User clicks "Contact Curator".
5. Repository Interface requests Access Control System to display contact option.
6. Access Control System displays the contact option to the User.
7. User generates an email template.
8. User fills in and sends the request email.
9. Email Notification System forwards the User request to the Collection Curator.
10. Collection Curator reviews the User request.
11. Collection Curator updates access permissions in the Access Control System if approved.
12. Access Control System notifies the User of the decision via email.
13. Email Notification System displays the access decision notification to the User.
