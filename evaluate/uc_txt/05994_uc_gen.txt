1. User selects recyclable waste types.
2. Frontend application requests facilities for selected waste types.
3. Backend server queries facilities by waste type.
4. Database returns a list of facilities.
5. Backend server sends facilities and opening hours to frontend.
6. Frontend displays facilities and opening hours to the user.
7. User requests optimal route and schedule.
8. Frontend calculates optimal route based on user location and facility hours.
9. Backend returns optimal route and schedule to frontend.
10. Frontend displays optimal route and schedule to the user.
