1. Publisher logs in via GitHub or Google.
2. User Account Management authenticates the login successfully.
3. Publisher initiates the data package creation.
4. Dataset Publishing prepares package metadata.
5. Dataset Publishing sets package visibility to private.
6. Access Control verifies team access rights.
7. Dataset Publishing imports the dataset.
8. Dataset Publishing validates the dataset.
9. Dataset Publishing saves the dataset package.
10. Dataset Publishing notifies team members.
11. Dataset Publishing publishes the package to the team.
