1. User selects 'Change' button on View Account (Payment Terms tab) page.
2. User navigates to Change Payment Terms page.
3. User inputs data and clicks 'Save changes' button on Change Payment Terms page.
4. Call API with Defendant Account ID and Payload (JSON).
5. API creates Defendant Account Payment Terms.
6. Created Defendant Account Payment Terms are displayed on the View Account page.
7. Display Payment Terms tab on the View Account page.
