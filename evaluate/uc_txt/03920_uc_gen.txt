1. User selects treatment, date, and enters contact info.
2. UI creates a booking for the User.
3. Core saves the booking to file.
4. Core notifies UI that the booking is created.
5. User submits the booking.
6. UI sends booking information to REST endpoint.
7. REST adds the booking.
8. Core saves the booking to file.
9. Core notifies UI that the booking is added.
10. UI confirms the booking to the User.
