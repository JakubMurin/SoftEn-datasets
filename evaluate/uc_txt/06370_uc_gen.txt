1. Depositor initiates dataset deposit.
2. ERDMP saves dataset information.
3. Dataset Repository confirms dataset saved.
4. ERDMP prompts Depositor to link with DMP.
5. Depositor provides DMP reference to ERDMP.
6. ERDMP requests DMP details from DMPT.
7. DMPT queries DMP by reference from DMPRepo.
8. DMPRepo returns DMP details to DMPT.
9. DMPT provides DMP details to ERDMP.
10. ERDMP links dataset with DMP using DatasetRepo.
11. DatasetRepo updates DMP with dataset link.
12. DMPRepo confirms linkage with DatasetRepo.
13. DatasetRepo confirms dataset-DMP linkage with ERDMP.
14. ERDMP confirms dataset linked with DMP to Depositor.
