1. Admin requests to view advertisements list.
2. List advertises view shows the list of advertisements to Admin.
3. Admin clicks on edit button of an advertisement.
4. List advertises view redirects to edit advertise view.
5. Edit advertise view displays the content of the advertisement to Admin.
6. Admin edits the advertisement data and saves changes.
7. Edit advertise view sends data to AdsController.
8. AdsController checks the data validity.
9. AdsController saves the data in AdsModel.
10. AdsModel confirms the saved data back to AdsController.
11. AdsController returns data about the successful save to edit advertise view.
12. Edited advertisement is displayed to Admin.
