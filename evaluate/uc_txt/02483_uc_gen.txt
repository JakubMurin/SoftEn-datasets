1. Researcher accesses the DMPTool Interface.
2. Researcher inputs project details like data volume and processing power.
3. DMPTool requests resource requirements input from ITRE.
4. ITRE inputs project details and returns resource estimates to DMPTool.
5. DMPTool initiates cost calculation with CCM.
6. CCM requests pricing data from IS.
7. IS returns pricing data to CCM.
8. CCM calculates the cost estimate and returns it to DMPTool.
9. DMPTool displays the cost estimate and resource needs to the Researcher.
