1. Researcher accesses the digital archive through the Web Interface.
2. Web Interface sends a request to the Search Service to view an item.
3. Search Service queries item metadata from the Archive Database.
4. Archive Database returns metadata, including the digitization date, to the Search Service.
5. Search Service displays the item details to the Web Interface.
6. Web Interface shows the item details, including the digitization date, to the Researcher.
