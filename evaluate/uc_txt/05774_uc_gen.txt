1. Librarian selects "Late Returns" option.
2. Web Interface queries overdue books from Database.
3. Database returns list of overdue books to Web Interface.
4. Web Interface displays the list of overdue books to the Librarian.
5. Librarian requests Web Interface to send reminders.
6. Web Interface triggers the Notification System to send overdue notices.
7. Notification System looks up borrower contact information from Database.
8. Database returns borrower contact information to Notification System.
9. Notification System confirms notifications sent to the Librarian.
