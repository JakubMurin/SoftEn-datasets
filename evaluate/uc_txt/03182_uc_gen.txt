1. User presses "Select language" button.
2. System displays language selection window to User.
3. User selects language flag.
4. System displays tickets window.
5. System retrieves ticket prices.
6. System displays ticket type selection window to User.
7. User selects ticket type and quantity, presses "Buy ticket" button.
8. System displays payment confirmation window to User.
9. User confirms purchase.
10. System displays payment method selection window to User.
11. User selects payment method.
12. System displays payment window to User.
13. User pays for the ticket.
14. System sends transaction report.
15. System prints tickets.
16. System dispenses change.
17. System displays purchase confirmation to User.
