1. Attendee requests summary from the Workshop Platform.
2. Workshop Platform fetches the request for summary from CMS.
3. CMS queries the workshop data from the Database.
4. Database returns the workshop data to CMS.
5. CMS generates summary content using the Content Editor.
6. Content Editor submits the summary to CMS.
7. CMS updates the Database with the generated summary.
8. Database confirms the update to CMS.
9. CMS publishes the summary on the Summary Page.
10. Summary Page displays the summary content to the Attendee.
