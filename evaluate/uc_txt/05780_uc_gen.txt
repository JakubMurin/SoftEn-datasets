1. Librarian selects the "Late Returns" option in the Web Interface.
2. Web Interface queries the database for overdue books.
3. Database returns a list of overdue books to the Web Interface.
4. Web Interface displays the list of overdue books to the Librarian.
5. Librarian requests to send reminders via the Web Interface.
6. Web Interface triggers notifications through the Notification System.
7. Notification System looks up borrower contact information in the database.
8. Database returns the contact information to the Notification System.
9. Notification System confirms that the notifications have been sent to the Librarian.
