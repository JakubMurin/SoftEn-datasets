1. Curator chooses 'Move Item'.
2. User Interface requests form to move item.
3. Collection Management System displays the form.
4. Curator submits form (Item ID, Source Collection, Target Collection).
5. User Interface validates and processes the move request.
6. Database fetches the item from Source Collection.
7. Collection Management System confirms removal of item from Source Collection.
8. Collection Management System adds the item to the Target Collection.
9. Collection Management System displays success message.
10. User Interface confirms that the item has been successfully moved.
