1. User accesses Order Details Page.
2. Order Information Page requests Order Details Page.
3. Controller fetches Order Details.
4. Model queries Order Information from Database.
5. Database returns Order Information to Model.
6. Model sends Order Details to Controller.
7. Controller displays Order Details on Order Information Page.
8. User selects Re-Buy Product.
9. Order Information Page requests Re-Buy Product.
10. Controller processes Re-Buy Request.
11. Model updates Order Information in Database.
12. Model confirms Re-Buy.
13. Controller displays Re-Buy Confirmation.
14. User selects Give Feedback.
15. Order Information Page requests Feedback Form.
16. Controller fetches Feedback Form from Model.
17. Model retrieves Product Information from Database.
18. Model sends Feedback Form to Controller.
19. Controller displays Feedback Form on Order Information Page.
20. User submits Feedback.
21. Order Information Page submits Feedback Form to Controller.
22. Controller saves Feedback in Model.
23. Model stores Feedback in Database.
24. Model confirms Feedback Submission.
25. Controller displays Feedback Confirmation on Order Information Page.
26. User selects Update Order.
27. Order Information Page requests Update Order.
28. Controller fetches Order Information from Model.
29. Model retrieves Order Details from Database.
30. Model redirects to Cart Details Page.
31. Controller displays Cart Details Page on Order Information Page.
32. User selects Cancel Order.
33. Order Information Page requests Cancel Order.
34. Controller processes Cancel Order in Model.
35. Model updates Order Status in Database.
36. Model confirms Order Cancellation.
37. Controller displays Order Cancellation Confirmation on Order Information Page.
