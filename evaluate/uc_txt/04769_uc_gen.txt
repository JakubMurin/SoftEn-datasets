1. User creates Taproot Assets Address.
2. Client sends request to LN Node through Relay.
3. Relay sends request to LN Link Node.
4. LN Link Node receives the request and forwards to Tapd.
5. Tapd processes the request and returns the Address.
6. LN Link Node returns the Address to Relay.
7. Relay returns the Address to Client.
8. Client shows the Address to User.
