1. Buyer updates profile information.
2. Profile information is sent to Web Server through UI.
3. Web Server validates and saves the profile changes.
4. App Server updates the user's profile in the Database.
5. Acknowledgment of profile update is received by App Server from DB.
6. App Server confirms profile update completion to Web Server.
7. Web Server notifies UI about successful profile update.
8. UI confirms profile update success to the Buyer.
