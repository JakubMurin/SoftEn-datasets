1. Archivist inputs keyword.
2. UI sends search request with the keyword to Search Controller.
3. Search Controller queries the keyword to Indexing Service.
4. Indexing Service fetches matching files from Digital Archive Database.
5. Indexing Service formats the matching files.
6. Search Controller receives the formatted results from Results Formatter.
7. UI displays the search results to the Archivist.
8. Archivist views the files matching the keyword.
