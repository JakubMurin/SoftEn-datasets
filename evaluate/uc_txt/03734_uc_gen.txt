1. User initiates text generation request from Fishtext API.
2. Upon success, JavaScript displays generated text on HTML DOM.
3. User interacts with keyboard by pressing keys.
4. Javascript receives input and starts input validation.
5. Javascript calculates typing speed, errors, and creates end game object.
6. Call for end game data submission to server in backend.
7. Server updates user data in database with result.
8. JS renders end game window on HTML DOM.
9. JavaScript records speed data for drawing graph.
