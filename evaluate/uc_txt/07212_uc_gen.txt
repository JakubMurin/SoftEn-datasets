1. Batch Submitter navigates to the EDRMS Documentation Section.
2. Batch Submitter requests authentication.
3. Authentication service authenticates the user.
4. Batch Submitter provides credentials.
5. The user is authenticated.
6. EDRMS Website requests Ingest Process Docs.
7. Documentation Server retrieves the documentation.
8. Ingest Process Documentation is returned.
9. Documentation Server sends the documentation link.
10. EDRMS Website displays the documentation link.
11. Batch Submitter clicks the documentation link.
12. EDRMS Website redirects the submitter to the documentation.
13. Ingest Process Documentation is displayed to the Batch Submitter.
