1. Researcher defines criteria for participant selection.
2. Recruitment Platform queries potential participants based on criteria.
3. User Database returns list of potential participants.
4. Recruitment Platform sends recruitment invitations.
5. Participant responds to invitation.
6. Recruitment Platform provides list of participating users to Researcher.
7. Researcher conducts user research sessions with Participant.
8. Participant provides feedback and data to Researcher.
9. Researcher inputs user research data to Analysis System.
10. Analysis System analyzes and summarizes findings.
11. Analysis System provides synthesized results to Researcher.
12. Researcher updates project status to "Complete" in Recruitment Platform.
13. Researcher archives project data and findings in Analysis System.
