1. The Archive Manager requests data volume information from the DMPTool.
2. The DMPTool queries planned data submissions from the Repository API.
3. The Repository API returns data submission estimates to the DMPTool.
4. The DMPTool requests the current storage capacity from the IT System.
5. The IT System returns current storage capacity details to the DMPTool.
6. The DMPTool requests additional data preservation details from the Data Manager.
7. The Data Manager provides data format, size, and period information to the DMPTool.
8. The DMPTool shows a comprehensive storage needs analysis to the Archive Manager.
9. The Archive Manager plans for storage expansion with the IT System.
10. The IT System confirms the storage infrastructure plan to the Archive Manager.
11. The Archive Manager confirms completion of the storage planning to the DMPTool.
