1. User searches and selects the restricted object.
2. User requests access details through the Repository Interface.
3. Access Control System displays "Contact Curator" option to the user.
4. User clicks on "Contact Curator" option.
5. Email Notification System generates an email template.
6. User fills in and sends the access request via email to the Collection Curator.
7. Collection Curator forwards the user request to the Access Control System.
8. Collection Curator reviews the user request.
9. Collection Curator updates access permissions in the Access Control System if the request is approved.
10. Access Control System notifies the user of the decision via email.
11. Email Notification System displays the access decision notification to the user.
