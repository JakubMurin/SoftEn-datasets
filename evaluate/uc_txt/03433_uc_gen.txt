1. User creates a new course.
2. UI creates course.
3. UI asks for course information.
4. User provides name, description, and edition.
5. UI creates course with provided information.
6. Controller creates a new course entity.
7. Controller sets the rest of the information.
8. Builder creates the course entity.
9. Domain creates the course entity.
10. Builder sends the course entity back to Controller.
11. Controller fetches repositories.
12. Controller creates a new Course object.
13. Factory creates a new Course object.
14. Factory creates a new Course Repository.
15. Controller saves the Course entity.
16. Controller sends the Course back to UI.
