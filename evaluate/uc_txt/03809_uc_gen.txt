1. User asks the client for a nickname.
2. User writes the chosen nickname.
3. Client sends the join request with the nickname, IP, and port to the server.
4. Server checks if the nickname is available.
5. Server adds the nickname to the list of users with the corresponding IP.
6. Server notifies the client that the user has successfully joined.
7. Server updates the chat with the joined user's nickname.
8. Client enables the "Ready to Play" button.
9. Client enables the chat functionality.
10. User acknowledges the successful join.
