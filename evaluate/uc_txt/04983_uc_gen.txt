1. Manager accesses the message schedule option in the UI Dashboard.
2. UI Dashboard sends a request for the message input form to the Messaging System.
3. Messaging System displays the message input form to the UI Dashboard.
4. Manager submits the message details, including timeframe and content, in the UI Dashboard.
5. UI Dashboard initiates the scheduling of the message display with the Schedule Manager.
6. Schedule Manager saves the scheduled message in the Repository Database.
7. Repository Database acknowledges the save operation.
8. Schedule Manager confirms that the message is successfully scheduled to the Messaging System.
9. UI Dashboard shows the confirmation to the Manager.
