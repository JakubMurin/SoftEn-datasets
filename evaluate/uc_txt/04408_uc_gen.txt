1. Manager selects "Delete Employee".
2. Web Interface requests deletion of employee.
3. Server checks if employee exists.
4. Database checks if employee exists.
5. Employee found.
6. Server asks for confirmation.
7. Manager confirms deletion.
8. Web Interface sends delete command with confirmation.

10. Database confirms deletion.

12. Web Interface shows deletion confirmation to Manager.
13. Employee not found.
