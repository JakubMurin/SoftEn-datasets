1. Archivist submits credentials to Authentication System.
2. Authentication System validates credentials with User Management System.
3. User Management System provides permissions details to Access Restriction Controls.
4. Access Restriction Controls verify permissions and permit access to File Management System.
5. File Management System fetches files from XDrive.
6. XDrive returns the requested files to File Management System.
7. File Management System displays files to the Archivist.
