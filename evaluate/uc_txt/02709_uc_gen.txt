1. Depositor accesses the deposit interface on the Web Interface.
2. Web Interface requests authentication from Pure Integration Module.
3. Web Interface authenticates the Depositor.
4. Depositor is shown the dataset deposit form.
5. Depositor fills in the dataset deposit form (including metadata and files).
6. Web Interface processes the deposit form.
7. Dataset Management System checks if dataset metadata aligns with Pure requirements.
8. Dataset Management System deposits the dataset and metadata in the Bath Data Archive.
9. Dataset Management System confirms deposit and generates DOI.
10. Pure Integration Module links dataset DOI with Pure records.
11. Pure Integration Module updates Pure records.
12. Dataset Management System confirms dataset deposit and provides DOI to the Depositor.
13. Depositor is displayed confirmation including the DOI link.
