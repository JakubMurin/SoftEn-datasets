1. User decides to pay with USDT through the Website UI.
2. Website UI requests payment information from the Payment Gateway.
3. Payment Gateway provides the USDT deposit address, amount, and includes QR code to the User.
4. User transfers USDT to the Blockchain Interface.
5. Transaction Monitoring detects the transaction.
6. Blockchain Interface requests transaction confirmation from Transaction Monitoring.
7. Transaction Monitoring updates transaction status based on 6 confirmations received.
8. Transaction Monitoring notifies the Gateway of the final transaction approval.
9. Gateway stores transaction and user information in the Database and Backend.
10. Database and Backend confirms the payment and converts it to points.
