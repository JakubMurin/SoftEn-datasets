1. User clicks 'Comenzar' button.
2. App handles the event to start the game.
3. StartScreen handles the event to show start button.
4. StartButtonEventHandler handles the event to hide start screen and display input screen.
5. PlayersInputScreen is displayed for selecting and confirming inputs.
6. App handles input selection and confirmation.
7. ConfirmButton selects and sends input data to the Game.
8. Game loads received data and creates PlayersNamesInputScreen.
9. PlayersNamesInputScreen displays screen for entering players' names.
10. App handles names input and verifies.
11. NamesButton handles the event, registers users, and creates Player instances.
12. Game creates players based on entered names.
13. QuestionScreen is created and displayed to show questions and options.
14. App displays questions to players.
