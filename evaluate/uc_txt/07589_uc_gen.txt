1. The Faculty Member selects the data file for upload.
2. The Web Browser Interface initiates the file upload process.
3. The File Upload Service stores the data file in the Digital Repository.
4. The Database saves the file metadata including uploader information.
5. The Repository confirms the metadata storage.
6. The Upload Service confirms the file storage and displays upload success.
7. The Upload Service provides the Faculty Member with the file ID and upload confirmation.
8. The Faculty Member chooses the licensing option (CC-By or CC0) in the Web Browser Interface.
9. The License Service receives the license choice and file ID from the Web Browser Interface.
10. The DRM Service applies the chosen license to the file.
11. The Repository updates the file metadata with the license information.
12. The Database updates the file record with license metadata.
13. The Repository confirms the license update.
14. The Repository confirms the license application to the file.
15. The License Service displays the license application success to the Web Browser Interface.
16. The Web Browser Interface confirms to the Faculty Member that the license has been set.
