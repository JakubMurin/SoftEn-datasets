1. Dataset Developer requests Archetype Tool to create dataset package.
2. Archetype Tool generates dataset template and provides it to the Developer.
3. Dataset Developer defines dataset type (Schemas, Configurations, Code).
4. Dataset Developer uploads dataset package to Build System.
5. Build System validates package integrity and compatibility.
6. Build System stores validated dataset package in Dataset Repository and acknowledges storage to Developer.
7. Build System notifies Developer about successful packaging and storage.
8. Dataset Developer deploys dataset type for application integration in the Application Ecosystem.
9. Application Ecosystem integrates the dataset type and confirms successful deployment and integration to Developer.
