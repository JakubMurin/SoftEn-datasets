1. O Gestor de Logística inicia o processo de registo de um percurso.
2. O Sistema EletricGo solicita os dados necessários para o registo ao Gestor de Logística.
3. O Gestor de Logística introduz os dados solicitados no EletricGo.
4. O Sistema EletricGo solicita a confirmação dos dados de registo ao Gestor de Logística.
5. O Gestor de Logística confirma os dados do registo no EletricGo.
6. O Sistema EletricGo informa o sucesso da operação ao Gestor de Logística.
