1. User decides to pay with USDT.
2. Website UI requests payment information.
3. Payment Gateway provides USDT deposit address and amount, including QR code, to the User.
4. User transfers USDT via Blockchain.
5. Transaction Monitoring detects the transaction.
6. Blockchain requests confirmation of transaction.
7. Blockchain updates transaction status after 6 confirmations.
8. Transaction Monitoring notifies Gateway of final transaction approval.
9. Payment Gateway stores transaction and user information in the Database.
10. Database confirms payment reception and converts to points.
