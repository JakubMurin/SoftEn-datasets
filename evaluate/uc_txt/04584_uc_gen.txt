1. User opens the application.
2. Homepage screen displays "Account Settings".
3. User navigates to Account Settings.
4. Delete Account Dialog is displayed.
5. User gets prompted for confirmation.
6. User confirms account cancellation.
7. Account cancellation initiated.
8. API Call for Account Cancellation sent.
9. Forward cancellation request to Users Service.
10. Account marked for deletion in the Database.
11. Account marked for deletion confirmed.
12. Account cancellation confirmed.
13. Cancellation confirmation sent to User.
14. Cancellation confirmation displayed to User.
15. Home screen navigated back to.
16. Account cancellation confirmation message displayed.
17. User logs out and session data cleared.
