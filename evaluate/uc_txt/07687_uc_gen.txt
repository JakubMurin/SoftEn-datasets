1. Navigate to 'Submit Session'.
2. Display 'Session Submission Form' to AuthenticatedUser via Web Interface.
3. Fill in and Submit Form by AuthenticatedUser via Web Interface.
4. Save the submitted Session Details to the Session Database.
5. Trigger Session Review Process by Session Review System based on the saved details.
6. Update Session Status as 'Under Review' in the Session Database.
7. Trigger Notification to User by Notification Service.
8. Check for Email Notification of 'Session Submitted' by AuthenticatedUser.
9. Update the Session Status as 'Approved' or 'Rejected' in the Database after the review.
10. Trigger Notification to User with 'Session Approved' or 'Session Rejected' message by Notification Service.
