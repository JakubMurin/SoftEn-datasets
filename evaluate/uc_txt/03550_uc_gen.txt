1. DonorRepresentative accesses the website.
2. User requests access to the website.
3. Authorization System authenticates the user.
4. DonorRepresentative requests Image Descriptive Info.
5. User Interface queries for Images.
6. File Management fetches Image Metadata from Database.
7. Database returns Image Metadata to File Management.
8. File Management displays Descriptive Info.
9. User Interface shows Descriptive Information about Images to DonorRepresentative.
