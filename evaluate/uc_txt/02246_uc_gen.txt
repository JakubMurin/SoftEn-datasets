1. End-User needs help.
2. End-User provides NIN to CS.
3. CS forwards NIN to Application.
4. CS informs End-User to start Swedish BankID.
5. Application sends POST request to Signicat for authentication.
6. Signicat processes the authentication.
7. End-User starts BankID authentication.
8. End-User authenticates with BankID via IDP.
9. End-User authenticated successfully.
10. IDP sends information on authentication to Signicat.
11. Application requests authentication result from Signicat.
12. Signicat sends authentication result to Application.
13. CS provides authentication result to End-User.
14. CS assists the authenticated End-User.
