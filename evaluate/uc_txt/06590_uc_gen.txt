1. Admin requests to view the list of advertisements.
2. List of advertisements view displays the advertisement list to the Admin.
3. Admin clicks on the edit button of an advertisement.
4. Redirected to the edit advertisement page.
5. Edit advertisement page shows the content of the advertisement.
6. Admin edits the advertisement data and clicks the save button.
7. Edit advertisement view sends the information to AdsController.
8. AdsController checks the data.
9. AdsController saves the data in AdsModel.
10. AdsModel returns the saved data to AdsController.
11. AdsController returns the result of saving the advertisement to the edit advertisement view.
12. Edit advertisement view displays the edited advertisement to the Admin.
