1. El usuario inicia sesión en la plataforma.
2. Las credenciales del usuario son verificadas por la plataforma.
3. El usuario recibe confirmación de inicio de sesión.
4. El usuario solicita contenido personalizado.
5. La plataforma analiza el perfil del usuario a través del Sistema de Recomendación IA.
6. El Sistema de Recomendación IA obtiene el historial de aprendizaje del usuario desde la base de datos.
7. La base de datos envía al Sistema de Recomendación IA el historial de aprendizaje del usuario.
8. El Sistema de Recomendación IA envía recomendaciones de contenido personalizado a la plataforma.
9. La plataforma muestra al usuario el contenido recomendado.
10. El usuario completa una actividad de aprendizaje.
11. La plataforma guarda el progreso del usuario en la base de datos.
12. La plataforma recibe confirmación de guardado del progreso.
