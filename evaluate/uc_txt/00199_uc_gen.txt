1. O cliente marca uma consulta com a secretária.
2. A secretária confirma a consulta para o cliente.
3. O cliente informa os sintomas ao veterinário.
4. O veterinário registra os sintomas do cliente.
5. O veterinário fornece o diagnóstico e plano de tratamento para o cliente.
6. O cliente retorna para uma consulta.
7. O veterinário registra o pedido de exames para o cliente.
8. O veterinário fornece os resultados dos exames para o cliente.
9. O veterinário fornece receitas para o cliente.
