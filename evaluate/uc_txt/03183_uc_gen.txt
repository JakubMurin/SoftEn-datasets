1. User presses the "Select language" button.
2. System displays the language selection window.
3. User selects a language by pressing on the flag.
4. System fetches ticket prices.
5. System displays ticket type selection window.
6. User selects ticket type and quantity then presses "Purchase Ticket" button.
7. System shows payment confirmation window.
8. User confirms payment.
9. System displays payment method selection window.
10. User selects payment method.
11. System shows payment info window.
12. User pays for the ticket.
