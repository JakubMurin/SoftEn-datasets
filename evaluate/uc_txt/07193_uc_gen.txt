1. Curator selects items for permission change.
2. User Interface validates curator's credentials.
3. Item Management receives request for current permissions of selected items.
4. Database queries and returns current permissions of selected items.
5. Item Management displays current permissions to User Interface.
6. Curator inputs new permissions for items.
7. User Interface sends request for new permissions.
8. Item Management checks if curator has rights to change permissions.
9. Database updates permissions.
10. Item Management confirms permission update to User Interface.
11. User Interface displays success message to Curator.
