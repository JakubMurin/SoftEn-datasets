1. Admin requests to view the list of advertisements.
2. List Advertisements View displays the advertisement list.
3. Admin clicks the edit button on an advertisement.
4. List Advertisements View redirects to the edit advertisement page.
5. Edit Advertise View shows the content of the advertisement.
6. Admin edits the advertisement data and saves.
7. Edit Advertise View sends the information to AdsController for processing.
8. AdsController validates the data and saves it in AdsModel.
9. AdsModel returns the saved data to AdsController.
10. AdsController returns the result of saving the advertisement to Edit Advertise View.
11. Edit Advertise View displays the edited advertisement.
