1. Display "Account Settings" on HomepageScreen.
2. Navigate to Account Settings.
3. Display Modify Password Page on ModifyPasswordScreen.
4. Prompt User for updated password.
5. Enter updated password.
6. Press "Modify" button.
7. Submit updated password to AuthService.
8. API Call to Modify Password in UsersRouter.
9. Forward updated password to UsersService.
10. Validate updated password.
11. Update password in Database.
12. Confirm password updated.
13. Confirm update success in UsersRouter.
14. Send update confirmation to AuthService.
15. Display update success message on ModifyPasswordScreen.
16. Navigate back to Homepage on ModifyPasswordScreen.
17. Display confirmation message on HomepageScreen.
