1. Manager accesses Menu Management in the Management Interface.
2. Management Interface requests current menu items from the Backend Server.
3. Backend Server fetches menu items from the Database.
4. Backend Server displays menu items data to the Management Interface.
5. Manager selects items to update.
6. Manager submits updated menu items.
7. Management Interface sends an update menu items request to the Backend Server.
8. Backend Server updates menu items in the Database.
9. Database confirms the updated items to the Backend Server.
10. Backend Server confirms successful update to the Management Interface.
11. Management Interface displays a success message to the Manager.
