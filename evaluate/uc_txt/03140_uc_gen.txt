1. O User requisita a página dos andares ao SPA.
2. O SPA retorna a página dos andares ao User.
3. O User seleciona um andar da tabela no SPA.
4. O SPA mostra os dados do andar selecionado à API.
5. A API mostra os edifícios numa caixa de seleção de volta ao SPA.
6. O SPA mostra os dados do andar selecionado ao User.
7. O User insere um ficheiro com o floorplan do andar selecionado ao SPA.
8. O SPA faz um patch para a API com o floorplan inserido.
