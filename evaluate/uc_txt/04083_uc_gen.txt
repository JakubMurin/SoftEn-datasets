1. Admin accesses the Update Center Form through the UI.
2. UI displays the form to the Admin.
3. Admin inputs the Recycling Center data through the UI.
4. UI submits the new Center information to the Server.
5. Server validates and adds the information to the Database.
6. Database confirms the addition of the new Center information to the Server.
7. Server displays a success message to the UI.
8. UI shows a confirmation message to the Admin.
