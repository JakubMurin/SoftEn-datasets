1. User navigates to 'Submit Session' page.
2. Web Interface displays 'Session Submission Form' to the user.
3. User fills in and submits the session details.
4. Web Interface saves the session details in the database.
5. Session Database triggers the session review process in Session Review System.
6. Session Review System sets the Session Status as Under Review in the database.
7. Notification service triggers notification about the session submission.
8. User receives an email notification about the session submission.
9. User goes to check their email.
