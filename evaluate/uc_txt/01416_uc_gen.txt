1. Developer requests Archetype Tool to create dataset package.
2. Archetype Tool generates dataset template and provides it to Developer.
3. Developer defines the dataset type (Schemas, Configurations, Code).
4. Developer uploads the dataset package to Build System.
5. Build System validates the package integrity and compatibility.
6. Build System stores the validated dataset package in Dataset Repository and acknowledges storage.
7. Build System notifies Developer about successful packaging and storage, or returns error message if validation fails.
8. Developer deploys the dataset type for application integration.
9. Application Ecosystem integrates the dataset type and confirms successful deployment.
