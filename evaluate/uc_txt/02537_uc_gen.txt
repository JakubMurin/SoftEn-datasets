1. Manager accesses the Menu Management functionality.
2. Management Interface requests current menu items from Backend Server.
3. Backend Server fetches menu items from the Database.
4. Database provides menu items data to Backend Server.
5. Backend Server displays menu items to Management Interface.
6. Manager selects items to update.
7. Manager submits updated menu items.
8. Management Interface sends an update menu items request to Backend Server.
9. Backend Server updates menu items in the Database.
10. Backend Server confirms the successful update with the Database.
11. Backend Server returns the confirmation of a successful update to Management Interface.
12. Management Interface displays a success message to the Manager.
