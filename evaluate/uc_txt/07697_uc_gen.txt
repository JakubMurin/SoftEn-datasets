1. Data Scientist requests to create a dataset instance through the UI/CLI.
2. UI/CLI parses request parameters and selects existing dataset type while defining instance properties.
3. Dataset Management System verifies permissions and dataset type existence.
4. Dataset Management System creates a new dataset instance with the specified properties.
5. Dataset Management System displays a success message with instance details to the Data Scientist.
6. UI/CLI shows a confirmation/error message to the Data Scientist.
