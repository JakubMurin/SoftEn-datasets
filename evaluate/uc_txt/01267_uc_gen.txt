1. Manager requests inventory list.
2. Management Interface activates.
3. Management Interface retrieves inventory data.
4. Inventory Management Module activates.
5. Inventory Management Module queries inventory list from Database.
6. Database provides inventory list data to Inventory Management Module.
7. Inventory Management Module returns inventory list to Management Interface.
8. Management Interface deactivates.
9. Manager views and interacts with the displayed inventory list.
