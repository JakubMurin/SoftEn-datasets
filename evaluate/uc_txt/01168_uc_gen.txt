1. Depositor accesses the DOI Information Page.
2. Web Interface receives the request for DOI information.
3. Backend Server queries the DOI Assignment Process in the DOI Information Database.
4. DOI Information Database returns the DOI Process Information to the Backend Server.
5. Backend Server requests the required Metadata Fields from the Metadata Service.
6. Metadata Service returns the Metadata Fields to the Backend Server.
7. Backend Server responds with the DOI and Metadata Information to the Web Interface.
8. Web Interface displays the DOI and Metadata Information to the Depositor.
