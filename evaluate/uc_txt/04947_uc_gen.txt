1. Template Admin creates a new report template.
2. Report template factory initiates the creation of a report template.
3. Report template is created successfully.
4. Row template factory is called to create a row template.
5. Row template is successfully created.
6. Row template instance is returned to the row template factory.
7. Row template factory returns the row template instance to the report template.
8. Report template receives the row template instance.
9. Report template instance is returned to the report template factory.
10. Report template factory returns the report template instance to the Template Admin.
