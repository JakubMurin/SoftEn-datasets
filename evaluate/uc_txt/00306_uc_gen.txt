1. User decides to pay with USDT.
2. UI requests payment information from Gateway.
3. Gateway provides USDT deposit address and amount, including QR code, to the User.
4. User transfers USDT via Blockchain.
5. Monitor detects the transaction.
6. Blockchain requests transaction confirmation from Monitor.
7. Blockchain updates transaction status multiple times.
8. Monitor notifies Gateway of final transaction approval.
9. Gateway stores transaction and user information in Database.
10. Database confirms payment and converts it to points for the User.
