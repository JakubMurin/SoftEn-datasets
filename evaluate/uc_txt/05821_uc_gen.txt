1. Buyer updates profile information.
2. UI sends a PUT request to Web Server for updating user profile.
3. Web Server validates and saves the profile changes.
4. App Server updates the user profile in the Database.
5. Database acknowledges the update.
6. App Server confirms the profile update to Web Server.
7. Web Server notifies UI about the successful update.
8. UI informs the Buyer about the successful profile update.
