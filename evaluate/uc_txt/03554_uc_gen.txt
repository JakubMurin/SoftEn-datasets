1. Donor Respresentative accesses the website.
2. User Interface receives request for access.
3. Authorization System authenticates the user.
4. Donor Respresentative requests descriptive info about an image.
5. User Interface queries for images.
6. File Management queries the Database to fetch image metadata.
7. Database returns the image metadata to File Management.
8. File Management displays the descriptive info about the image.
9. User Intestate shows the descriptive info about the image to Donor Respresentative.
