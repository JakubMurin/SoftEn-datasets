1. Manager accesses the inventory management system.
2. UI requests the current inventory list.
3. Backend queries inventory items from the database.
4. Database returns inventory items to Backend.
5. Backend displays inventory items to UI.
6. Manager selects an item to update.
7. Manager enters new inventory details.
8. UI sends update inventory request with details to Backend.
9. Backend updates the item in the database.
10. Database confirms the update to Backend.
11. Backend confirms the success to Manager.
12. UI displays the update confirmation to Manager.
