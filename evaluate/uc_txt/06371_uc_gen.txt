1. Depositor initiates dataset deposit.
2. Enhanced Research Data Management Platform saves dataset information.
3. Enhanced Research Data Management Platform prompts depositor to link with DMP.
4. Depositor provides DMP reference.
5. Enhanced Research Data Management Platform requests DMP details from Data Management Plan Tool.
6. Data Management Plan Tool queries DMP Repository by reference.
7. DMP Repository returns DMP details to Data Management Plan Tool.
8. Data Management Plan Tool provides DMP details to Enhanced Research Data Management Platform.
9. Enhanced Research Data Management Platform links dataset with DMP in Dataset Repository.
10. Dataset Repository updates DMP with dataset linkage.
11. Dataset Repository confirms dataset-DMP linkage to Enhanced Research Data Management Platform.
12. Enhanced Research Data Management Platform confirms dataset linked with DMP to Depositor.
