1. User authenticates with the system.
2. Authentication Service validates user credentials with the Database.
3. Database confirms the validity of the credentials.
4. Authentication Service notifies the User of successful authentication.
5. User requests help links from the Help Link Service.
6. Help Link Service fetches contextual links from the Database.
7. Database returns the requested links to the Help Link Service.
8. Help Link Service displays the help links to the User.
