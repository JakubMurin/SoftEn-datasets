1. User places an order and initiates payment.
2. Merchant system generates platform order.
3. Merchant system requests order creation from WeChat Pay system.
4. WeChat Pay system generates prepay order.
5. WeChat Pay system returns prepay order identifier to Merchant system.
6. Merchant system generates signed payment information.
7. User initiates payment.
8. User triggers WeChat payment.
9. WeChat client sends payment request to WeChat Pay system.
10. WeChat Pay system verifies payment authorization.
11. WeChat Pay system returns payment authorization to User.
12. User confirms payment and enters password.
13. WeChat Pay system submits authorization.
14. WeChat Pay system verifies authorization.
15. WeChat Pay system notifies Merchant system of payment result.
16. Merchant system saves payment notification.
17. Merchant system informs WeChat Pay system of successful processing.
18. WeChat Pay system returns payment result to User and sends notification.
19. User queries payment status from Merchant system.
20. Merchant system checks payment status with WeChat Pay system.
21. WeChat Pay system returns payment status to Merchant system.
22. WeChat Pay system displays payment message to User.
