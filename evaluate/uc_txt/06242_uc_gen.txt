1. Member enters a search query for a book.
2. Search Interface fetches matching books from Library Database.
3. Search Interface displays list of matched books to the Member.
4. Member selects a book to check availability from Book Details.
5. Book Details requests availability status from Availability Checker.
6. Availability Checker checks current borrowings and reservations for the selected book.
7. Availability Checker shows availability status to the Member.
8. Member is presented if the book is "available" or "reserved" in the book details.
