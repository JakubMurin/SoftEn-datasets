1. Archivist submits credentials.
2. Authentication System validates Archivist's credentials.
3. Authentication System grants access to Archivist.
4. Archivist requests access to X-drive from File Management System.
5. File Management System verifies permissions with Access Restrictions controls.
6. User Management System retrieves Archivist permissions.
7. Access Restrictions Controls grants access to FileManager.
8. File Management System fetches files from XDrive.
9. XDrive returns requested files to FileManager.
10. FileManager displays files to Archivist.
