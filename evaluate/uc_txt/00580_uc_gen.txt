1. User decides to pay with USDT.
2. UI requests payment information from Gateway.
3. Gateway provides USDT deposit address and amount, including QR code.
4. User transfers USDT through Blockchain.
5. Monitor detects transaction.
6. Request transaction confirmation from Blockchain.
7. Blockchain updates transaction status iteratively 6 times.
8. Monitor notifies Gateway of final transaction approval.
9. Gateway stores transaction and user information in the Database.
10. Database confirms payment and converts to points for the User.
