1. Depositor accesses the repository information through the Web Interface.
2. Web Interface requests accepted items and formats from EDRMS.
3. EDRMS retrieves item types and formats from Format and Type Management.
4. Format and Type Management returns types and recommended formats to EDRMS.
5. EDRMS displays the information to the Web Interface.
6. Web Interface shows accepted item types and formats to the Depositor.
