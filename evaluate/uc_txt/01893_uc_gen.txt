1. Curator requests item referrer data from EDRMS.
2. EDRMS queries the Database for the item ID.
3. Database retrieves and returns the item ID.
4. EDRMS requests referrer statistics for the item from Analytics Engine.
5. Analytics Engine queries the Database for referrer logs.
6. Database retrieves and returns the referrer data.
7. Analytics Engine provides summarized referrer information to EDRMS.
8. EDRMS displays the referrer data to the Curator.
