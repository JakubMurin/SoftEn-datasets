1. Building Manager requests task from Phone App.
2. Phone App requests task from Robot Management System.
3. Robot Management System adds request to queue.
4. Robot Management System checks robot availability.
5. If robot is available:.
6. Robot Management System assigns task to Robot.
7. Robot cleans the room.
8. Robot Management System sends task assigned message to Phone App.
9. Phone App shows message to Building Manager.
10. If robot is not available:.
11. Robot Management System estimates time.
12. Robot Management System sends estimated wait time message to Phone App.
