1. Manager selects option to create a course.
2. System asks for the course creation data.
3. Manager inserts the course data.
4. System asks for confirmation.
5. Manager confirms the creation.
6. Manager provides course data for creation:.
7. CourseController creates the course.
8. CourseService creates the course in the system.
9. CourseFactory generates a new course.
10. PersistenceContext fetches repositories.
11. New course is handled by Repository Factory to save courses.
12. Check if particular course exists with given data.
13. If the course does not exist, create a new course.
14. Otherwise, fetch existing course from repository.
15. Service concludes the operation successfully.
