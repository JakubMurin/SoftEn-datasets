1. The archivist selects an embargoed item.
2. The UI requests embargo details from the Embargo Management System.
3. The Embargo Management System retrieves embargo info from the Database.
4. The Database returns the embargo info to the Embargo Management System.
5. The Embargo Management System displays embargo details to the UI.
6. The archivist inputs a new embargo length.
7. The UI sends an update embargo length request to the Embargo Management System.
8. The Embargo Management System updates embargo info in the Database.
9. The Database confirms the update to the Embargo Management System.
10. The Embargo Management System displays a success message to the UI.
11. The UI shows a confirmation message to the archivist.
