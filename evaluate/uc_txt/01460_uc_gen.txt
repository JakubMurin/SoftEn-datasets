1. Faber requests to create Faber DID using FaberIndyWallet.
2. FaberAnonCreds creates Faber DID using ED25519 and stores it in FaberIndyWallet.
3. Faber creates SCHEMA using Faber DID.
4. FaberAnonCreds stores SCHEMA in FaberIndyWallet.
5. Faber requests to create Diem Identifier and ED25519 keys using FaberPythonDiem.
6. FaberPythonDiem stores Diem Identifier in FaberMiniWallet.
7. Faber publishes identifier to Ledger in DiemLedger.
