1. Moderator navigates to Account Settings.
2. Web interface receives a request to view current account details.
3. Backend server fetches account details from the database.
4. Backend server displays current account details to Web interface.
5. Web interface shows account details form to the Moderator.
6. Moderator inputs updated account details.
7. Web interface submits updated account details to the Backend server.
8. Backend server updates account details in the database.
9. Web interface acknowledges successful update to the Moderator.
10. Moderator receives update success message.
