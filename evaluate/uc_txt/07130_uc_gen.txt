1. Attendee requests a summary report from Workshop Platform.
2. Workshop Platform fetches the request for a summary from CMS.
3. CMS queries workshop data from the Database.
4. Database returns workshop data to CMS.
5. CMS generates summary content using Content Editor.
6. Content Editor submits the generated summary to CMS.
7. CMS updates the Database with the summary.
8. Database sends confirmation to CMS.
9. CMS publishes the summary to the Summary Page.
10. Summary Page displays the summary content to the Attendee.
