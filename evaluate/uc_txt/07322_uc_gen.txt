1. FacultyMember selects data to upload on the Upload Interface.
2. Upload Interface uploads the data file to the Repository System.
3. Repository System stores the data file in the Database.
4. Repository System requests metadata entry form from Metadata Service.
5. Metadata Service displays the metadata entry form to FacultyMember.
6. FacultyMember enters metadata, including publication details, on Metadata Service.
7. Metadata Service updates metadata in the Database.
8. Repository System confirms data and publication association to FacultyMember.
