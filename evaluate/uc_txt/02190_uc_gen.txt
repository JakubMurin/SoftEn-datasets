1. Archivist requests to restrict file access through Web Interface.
2. Web Interface validates Archivist credentials with Authorization Service.
3. Authorization Service confirms the credentials are valid.
4. Web Interface displays the form for restriction options.
5. Archivist submits the restriction details.
6. Database updates file metadata with the restriction details.
7. Database confirms the metadata update.
8. File Management System applies the access restrictions.
9. File Management System confirms the restriction application.
10. Web Interface displays a success message to the Archivist.
