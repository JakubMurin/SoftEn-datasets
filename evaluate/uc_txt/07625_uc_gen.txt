1. Customer requests to create an account.
2. Portal displays the account creation form to the Customer.
3. Customer submits account details (username, email, password) to the Portal.
4. Portal validates the account details with the Authentication System.
5. If details are valid:.
6. Authentication System confirms the details are valid.
7. Portal creates a new user account in the Accounts Database.
8. Accounts Database confirms the account creation.
9. Portal informs the Customer about the successful account creation.
10. Portal initiates the first-time login sequence.
11. Authentication System requests authentication (login) from the Customer.
12. Customer provides credentials to the Authentication System.
13. Authentication System validates the provided credentials with the Accounts Database.
14. Accounts Database confirms the credentials are valid.
15. Authentication System notifies the Customer about successful authentication.
16. If details are invalid:.
17. Authentication System reports an error to the Portal.
18. Portal displays an error message to the Customer.
