1. The UI displays available vehicle DTOs to the Green Spaces Manager for selection.
2. Green Spaces Manager selects a vehicle for assignment.
3. UI sends the request to assign the selected vehicle to the agenda entry to the controller.
4. Controller assigns the vehicle to the agenda entry.
5. Controller updates the vehicle assignment in the Vehicle Repository.
6. Vehicle Repository updates the assignment.
7. Controller informs the UI about the successful assignment.
