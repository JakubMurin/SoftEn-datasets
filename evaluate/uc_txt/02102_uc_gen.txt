1. O atendente inicia o empréstimo para o leitor.
2. O sistema identifica o leitor com o ID fornecido.
3. O leitor é identificado.
4. O sistema cria o objeto de empréstimo.
5. O objeto de empréstimo é criado.
6. O sistema associa o empréstimo ao leitor.
7. A associação do empréstimo com o leitor é concluída.
8. O sistema informa o início do empréstimo ao atendente.
