1. Admin requests to view advertisement list.
2. List advertisements view displays the advertisement list.
3. Admin clicks on the edit button of an advertisement.
4. Redirect to the edit advertisement page.
5. Edit advertisement view displays the advertisement content.
6. Admin edits the advertisement data and clicks save.
7. Information is sent to AdsController.
8. AdsController checks the data and saves it in AdsModel.
9. AdsModel returns the saved data to AdsController.
10. AdsController returns the result to edit advertisement view.
11. Edit advertisement view displays the updated advertisement.
