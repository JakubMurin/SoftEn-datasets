1. User selects treatment, date, and enters contact info.
2. UI creates booking with name, email, treatment, and date.
3. Core saves booking to file.
4. Core confirms booking creation to UI.
5. User submits booking.
6. UI sends POST request to REST for /bookings.
7. REST adds booking.
8. Core saves booking to file.
9. Core confirms booking addition to REST.
10. REST confirms booking to UI.
11. UI displays confirmation message to User.
