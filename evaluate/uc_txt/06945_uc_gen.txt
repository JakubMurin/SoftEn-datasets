1. Archivist authenticates with the system.
2. System successfully authenticates the archivist.
3. Archivist accesses the User Management module in EDRMS Core.
4. EDRMS Core retrieves the list of users from the User Management DB.
5. User Management DB provides the user data to EDRMS Core.
6. EDRMS Core displays the list of users to the archivist.
7. Archivist selects the repository administrator(s) in EDRMS Core.
8. EDRMS Core requests the Access Control Module to update privileges.
9. Access Control Module grants embargo privileges in the User Management DB.
10. User Management DB updates the privileges.
11. Access Control Module confirms the update to EDRMS Core.
12. EDRMS Core displays the update confirmation to the archivist.
