1. User clicks on the garbage bin icon under "financial goal" in GUI Financial Account Details View.
2. User enters the amount to confirm deletion.
3. GUI sends a delete request for the financial goal ID to the Server.
4. Server sends a delete request for the financial goal ID to the Database.
5. Database deletes the financial goal with the specified ID.
6. Server returns the updated financial account data to the GUI.
7. GUI displays the updated financial account details to the user.
