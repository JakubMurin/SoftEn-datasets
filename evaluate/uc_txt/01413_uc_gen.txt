1. Archetype Tool generates dataset template and provides it to the Dataset Developer.
2. Dataset Developer defines the dataset type (Schemas, Configurations, Code).
3. Dataset Developer uploads the dataset package to the Build System.
4. Build System validates the package integrity and compatibility.
5. Build System stores the validated dataset package in the Dataset Repository and acknowledges the storage.
6. Build System notifies the Dataset Developer about successful packaging and storage.
7. Dataset Developer deploys the dataset type for application integration.
8. Application Ecosystem integrates the dataset type and confirms successful deployment and integration to the Dataset Developer.
