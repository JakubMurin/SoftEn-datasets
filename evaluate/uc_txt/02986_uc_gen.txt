1. User selects dataset to update.
2. Web Interface displays dataset details to User.
3. User requests to disable explore feature.
4. Web Interface makes API call to disable explore for dataset.
5. API Gateway validates user permission.
6. Dataset Management receives request for dataset update.
7. Explore Service disables explore for specified dataset.
8. Dataset Management confirms explore disabled.
9. API Gateway updates and displays success message.
10. Web Interface notifies User about successful explore disable.
