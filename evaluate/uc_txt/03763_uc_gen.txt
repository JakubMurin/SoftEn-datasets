1. Customer requests to create an account.
2. Portal UI displays Account Creation Form to Customer.
3. Customer submits account details (username, email, password).
4. Authentication System validates account details.
5. Authentication System confirms details are valid.
6. Accounts Database creates a new user account.
7. Accounts Database confirms account creation.
8. Portal UI displays Account Creation Success to Customer.
9. Portal UI initiates First-Time Login Sequence.
10. Authentication System requests authentication (login) from Customer.
11. Customer provides credentials to Authentication System.
12. Authentication System validates provided credentials.
13. Accounts Database confirms credentials are valid.
14. Authentication System notifies Customer of successful authentication.
