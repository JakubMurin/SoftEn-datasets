1. Entrant attends event by sending an Attend Event Request to Ticket.
2. Ticket receives Attend Event Request and activates.
3. Ticket sends Create Member Request to Member.
4. Member activates and receives Create Member Request.
5. Member creates a new member.
6. Member sends Create Member Response to Ticket.
7. Member deactivates.
8. Ticket creates a ticket.
9. Ticket sends Attend Event Response to Entrant.
10. Ticket deactivates.
