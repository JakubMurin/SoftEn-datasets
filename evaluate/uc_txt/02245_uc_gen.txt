1. CS receives request for help.
2. CS provides NIN to Application.
3. Application sends NIN to Signicat.
4. CS informs End-User to start Swedish BankID.
5. End-User starts BankID.
6. End-User authenticates with BankID via IDP.
7. IDP provides authentication information to Signicat.
8. Signicat processes the authentication.
9. Customer needs to get authentication result from Signicat.
10. Application polls Signicat for authentication result.
11. CS provides authentication result to Application.
12. CS helps authenticated End-User.
