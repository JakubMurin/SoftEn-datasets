1. User speaks into the device.
2. UI interface captures the audio input.
3. Speech Recognition Module processes and converts speech to text.
4. Speech Recognition Module sends the converted text to Content Query Processor.
5. Content Query Processor queries the Content Database for matching content items.
6. Content Database returns the matched content items to the Content Query Processor.
7. Content Query Processor displays the search results to the user.
8. UI interface shows the search results to the user.
