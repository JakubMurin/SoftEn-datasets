1. User selects 'Change' button.
2. User navigates to the Change Payment Terms page.
3. User inputs data and clicks 'Save changes' button on the Change Payment Terms page.
4. API call is made with Defendant Account ID and Payload (JSON) by the Fines API.
5. Defendant Account Payment Terms are created by the Fines DB.
6. The Created Defendant Account Payment Terms are displayed to the user on the Payment Terms tab.
