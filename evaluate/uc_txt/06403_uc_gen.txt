1. Archivist inputs keyword.
2. UI sends search request with keyword to Search Controller.
3. Search Controller queries keyword to Indexing Service.
4. Indexing Service fetches matching files from Database.
5. Indexing Service formats files.
6. Results Formatter sends formatted results back to Search Controller.
7. Search Controller displays search results in UI.
8. UI shows files matching the keyword to Archivist.
