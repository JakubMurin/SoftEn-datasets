1. Curator chooses the 'Move Item' option.
2. User Interface requests form to move item.
3. Collection Management System displays form.
4. User Interface submits form with Item ID, Source Collection, and Target Collection.
5. Collection Management System validates and processes the move request.
6. Collection Management System fetches item details from Source Collection.
7. Database serves item details to Collection Management System.
8. Collection Management System removes the item from Source Collection.
9. Database confirms the removal.
10. Collection Management System adds the item to Target Collection.
11. Database confirms the addition.
12. Collection Management System displays success message.
13. User Interface confirms the item moved.
