1. Customer selects "My Orders".
2. UI requests list of orders from Backend System.
3. Backend fetches customer orders from Database.
4. Database returns orders to Backend.
5. Backend displays orders on UI.
6. UI shows list of orders to Customer.
7. Customer selects a dish from an order.
8. UI requests dish details from Backend System.
9. Backend fetches dish details from Database.
10. Database returns dish details to Backend.
11. Backend displays dish details on UI.
12. UI shows dish details to Customer.
13. Customer enters comment and submits.
14. UI posts comment to Backend.
15. Backend saves comment to dish in Database.
16. Database confirms save to Backend.
17. Backend displays success message on UI.
18. UI shows confirmation of comment submission to Customer.
