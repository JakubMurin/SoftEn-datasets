1. Administrator asks to create a new type of test.
2. TestTypeUI retrieves list of Parameter Categories.
3. TestTypeController converts Parameter Categories to DTO.
4. Administrator selects a parameter category.
5. Administrator requests data (description, collectingMethod, code).
6. Administrator types requested data.
7. TestTypeController creates Test Type with provided data.
8. Administrator confirms all data.
9. TestTypeController saves Test Type.
10. Administrator is informed of operation success.
