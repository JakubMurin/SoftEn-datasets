1. Admin requests to view the list of advertisements.
2. List Advertisements View displays the list of advertisements to Admin.
3. Admin clicks on the edit button of an advertisement.
4. List Advertisements View redirects to the edit advertisement page.
5. Edit Advertise View shows the content of the advertisement to Admin.
6. Admin edits the advertisement data and saves the changes.
7. Edit Advertise View sends the information to AdsController.
8. AdsController checks the data and saves it in AdsModel.
9. AdsModel returns the saved data to AdsController.
10. AdsController returns the saved advertisement result to Edit Advertise View.
11. Edit Advertise View displays the edited advertisement to Admin.
