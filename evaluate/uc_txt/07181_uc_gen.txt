1. Researcher logs in to DMPTool.
2. Researcher navigates to project DMP.
3. DMPTool fetches DMP record details.
4. Researcher selects "Add Publication".
5. Researcher enters publication details.
6. DMPTool checks if publication is in Repository.
7. DMPTool links publication with DMP record.
8. Repository confirms linkage.
9. Researcher requests research output report.
10. DMPTool fetches all publications linked to DMP.
11. Repository returns publications data.
12. Reporting Module generates report based on publications.
13. Reporting Module confirms report generated.
14. DMPTool displays/downloads report to Researcher.
