1. Admin starts the creation of a new user.
2. EletricGo is activated.
3. EletricGo requests data (first name, last name, email, password, password confirmation, phone number, user type) from Admin.
4. Admin enters data (first name, last name, email, password, password confirmation, phone number, user type) into EletricGo.
5. EletricGo shows data and asks for confirmation from Admin.
6. Admin confirms.
7. EletricGo informs the success of the operation to Admin.
