1. User creates an instance of IncreasingOrder.
2. User creates an instance of DecreasingOrder.
3. User initializes the scanner loop.
4. Scanner sends a message to the user.
5. User inputs an integer.
6. Scanner creates a switch.
7. Switch determines whether the input is for IncreasingOrder or DecreasingOrder.
8. SortingTemplate is activated.
9. SortingTemplate applies the Template Method algorithm.
10. SortingTemplate sorts the input in IncreasingOrder.
11. SortingTemplate sorts the input in DecreasingOrder.
