1. Researcher accesses the Repository Interface.
2. Repository Interface initiates search for Faculty.
3. SearchSystem queries Faculty Data.
4. FacultyProfiles returns results to SearchSystem.
5. SearchSystem displays results to Researcher.
6. Researcher reviews Faculty Interests/Expertise.
7. Repository Interface requests detailed info from FacultyProfiles.
8. FacultyProfiles provides detailed info to Repository Interface.
9. Researcher selects Faculty Members.
10. Repository Interface saves/notes selected Faculty for Collaboration.
11. CollaborationModule confirms selection and provides next steps to Researcher.
