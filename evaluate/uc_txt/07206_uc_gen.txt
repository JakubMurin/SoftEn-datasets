1. Batch Submitter navigates to EDRMS Documentation Section.
2. EDRMS Website requests Authentication.
3. Authentication Service authenticates User.
4. Batch Submitter provides Credentials to Authentication Service.
5. User is authenticated by Authentication Service.
6. EDRMS Website requests Ingest Process Docs from Documentation Server.
7. Documentation Server retrieves Documentation from Ingest Process Documentation.
8. Documentation Server returns the Documentation to EDRMS Website.
9. EDRMS Website sends the Documentation Link to Batch Submitter.
10. Batch Submitter displays the Documentation Link received from EDRMS Website.
11. Batch Submitter clicks on the Documentation Link.
12. EDRMS Website redirects Batch Submitter to Ingest Process Documentation.
13. Ingest Process Documentation is displayed to Batch Submitter.
