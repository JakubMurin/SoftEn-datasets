1. Member requests to share a work.
2. Application retrieves file share form.
3. UI Manager shows the file share form to the Member.
4. Member views the form.
5. Member fills in content details and uploads a file.
6. Member sends the form.
7. Application analyzes the uploaded file.
8. Application displays the analysis results to the Member.
9. Application requests confirmation for submission.
10. Member confirms submission.
11. Application copies the file to the "To be moderated" directory.
12. Application generates a transaction number.
13. Application logs the transaction.
14. Application transmits the transaction and file to the server.
15. Server notifies the Librarian of the submission.
16. Application notifies the Member of the moderation wait.
