1. El usuario selecciona comprar sobre.
2. El sistema de compras solicita la compra del paquete de cartas.
3. Se inicia sesión de transacción en la base de datos.
4. El sistema de compras busca el paquete de cartas en la base de datos.
5. Se verifica la disponibilidad del paquete de cartas.
6. Se verifica el saldo del usuario.
7. El sistema de compras genera las cartas y actualiza el inventario.
8. Se actualiza el saldo del usuario en la base de datos.
9. Se registran las transacciones y se actualiza la colección del usuario.
10. Se cierra la sesión de transacción en la base de datos.
