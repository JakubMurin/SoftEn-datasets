1. O Gestor de campus inicia a criação de um novo edifício.
2. O Gestor de campus requisita os dados necessários para o novo edifício.
3. O Gestor de campus insere os dados do edifício.
4. O Container envia os dados para a API para criar o edifício.
5. A API responde com um status code.
6. Se o status code for 200:.
7. O Container devolve o edifício criado ao Gestor de campus.
8. Se o status code for 400:.
9. O Container devolve uma mensagem de erro ao Gestor de campus.
