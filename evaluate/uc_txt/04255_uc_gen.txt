1. Customer requests Reservation Page.
2. Reservation System displays Reservation Form to Customer.
3. Customer submits Reservation Details (date, time, number of people) to Reservation System.
4. Reservation System checks Availability with Database.
5. Database confirms Availability to Reservation System.
6. Reservation System shows Confirmation Message to Customer.
7. Reservation System saves Reservation Details to Database.
8. Database sends Confirmation Status to Reservation System.
9. Reservation System displays Reservation ID and Details to Customer.
10. Database shows Unavailability Message to Reservation System.
11. Reservation System suggests Different Time/Date to Customer.
