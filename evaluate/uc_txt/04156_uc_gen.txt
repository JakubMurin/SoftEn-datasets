1. Manager selects "Manage Book Categories" from UI.
2. UI validates Manager Role.
3. UI grants access to Manager.
4. UI displays Book Categories to Manager.
5. Manager selects Categories to Delete.
6. UI requests Category Deletion to Book Category Management.
7. Book Category Management deletes Category Records in the Database.
8. Database confirms Deletion to Book Category Management.
9. Book Category Management displays Success Message to UI.
10. UI shows Update to Manager.
