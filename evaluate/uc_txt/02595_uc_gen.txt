1. Operations Manager initiates policy creation process.
2. Operations Manager provides policy details: name, cloud, environment, and file name.
3. Operations Manager communicates with the CLI to create the policy.
4. The CLI triggers the policy creation process.
5. The policy creation request is sent to the system.
6. Operations Manager interacts with the Web interface to start policy creation.
7. The Web interface initiates the policy creation process.
8. The policy creation request is forwarded to the system.
