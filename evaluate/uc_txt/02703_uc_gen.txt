1. Depositor accesses deposit interface.
2. Web Interface requests authentication.
3. Web Interface authenticates depositor.
4. Web Interface shows dataset deposit form.
5. Depositor fills in form (Includes metadata, files).
6. Dataset Management System processes deposit form.
7. Dataset Management System checks if dataset metadata aligns with Pure requirements.
8. Dataset Management System confirms metadata alignment.
9. Dataset Management System deposits dataset and metadata to Bath Data Archive.
10. Bath Data Archive confirms deposit and generates DOI.
11. Dataset Management System links dataset DOI with Pure records.
12. Pure Integration Module updates Pure records.
13. Dataset Management System confirms dataset deposit and provides DOI to depositor.
14. Depositor displays confirmation (Includes DOI link).
