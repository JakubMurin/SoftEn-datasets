1. Customer selects "My Orders" from the UI.
2. UI requests the list of orders from the Backend System.
3. Backend System fetches customer orders from the Database.
4. Database returns the orders to the Backend System.
5. Backend System displays the orders on the UI.
6. UI shows the list of orders to the Customer.
7. Customer selects a dish from an order.
8. UI requests dish details from the Backend System.
9. Backend System fetches dish details from the Database.
10. Database returns dish details to the Backend System.
11. Backend System displays dish details on the UI.
12. UI shows dish details to the Customer.
13. Customer enters a comment and submits it.
14. UI posts the comment to the Backend System.
15. Backend System saves the comment to the dish in the Database.
16. Database confirms the save to the Backend System.
17. Backend System displays a success message on the UI.
18. UI shows a confirmation of the comment submission to the Customer.
