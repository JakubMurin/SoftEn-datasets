1. User initiates tagging process.
2. Web Interface requests available tags.
3. Tagging System queries controlled vocabulary.
4. Controlled Vocabulary DB returns vocabulary list.
5. Tagging System displays vocabulary tags.
6. User selects tags from controlled vocabulary.
7. Web Interface assigns selected tags to dataset/file.
8. Tagging System validates and saves tags.
9. Controlled Vocabulary DB confirms tag assignment.
10. Tagging System confirms successful tagging to user.
11. Web Interface displays tagging success message.
