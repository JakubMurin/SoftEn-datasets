1. El usuario inicia sesión en la plataforma.
2. La plataforma verifica las credenciales del usuario en la base de datos.
3. La plataforma confirma el inicio de sesión al usuario.
4. El usuario solicita contenido personalizado a la plataforma.
5. La plataforma analiza el perfil del usuario con el Sistema de Recomendación IA.
6. El Sistema de Recomendación IA obtiene el historial de aprendizaje del usuario en la base de datos.
7. El Sistema de Recomendación IA envía recomendaciones a la plataforma.
8. La plataforma muestra al usuario el contenido recomendado.
9. El usuario completa una actividad en la plataforma.
10. La plataforma guarda el progreso del usuario en la base de datos.
11. La plataforma confirma al usuario que el progreso ha sido guardado.
