1. User sends request to Controller to get courses for a specific teacher.
2. Controller retrieves the teacher details by ID.
3. If Teacher found.
4. Controller fetches the courses taught by the teacher.
5. User receives search results.
6. For each course in the list.
7. Controller retrieves the course ID.
8. Controller retrieves the course name.
9. Controller retrieves the course description.
10. Controller retrieves the course price.
11. Controller retrieves the course average rating.
12. Controller retrieves the course banner image URL.
13. User receives course card data.
