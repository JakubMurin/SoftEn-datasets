1. User runs the Reflexion Agent with input text.
2. Agent retrieves relevant memories based on input text.
3. Agent invokes agent prompt.
4. LLM responds to the agent prompt.
5. Agent decides to use a tool.
6. Registry executes the tool.
7. Tool provides observation to the Registry.
8. Agent receives observation from the Registry.
9. LLM invokes reflection prompt.
10. Agent reflects on the tool usage.
11. Agent saves episodic memory of tool usage.
12. Agent saves reflexion memory.
13. Agent decides to finish.
14. Agent saves episodic memory of the interaction.
