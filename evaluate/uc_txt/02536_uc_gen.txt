1. Manager accesses the Menu Management interface.
2. Management Interface requests current menu items from the Backend Server.
3. Backend Server fetches menu items from the Database.
4. Database provides menu items data to the Backend Server.
5. Backend Server displays menu items to the Management Interface.
6. Manager selects items to update.
7. Manager submits the updated menu items.
8. Management Interface sends an update menu items request to the Backend Server.
9. Backend Server updates menu items in the Database.
10. Database confirms the updated items to the Backend Server.
11. Backend Server confirms successful update to the Management Interface.
12. Management Interface displays a success message to the Manager.
