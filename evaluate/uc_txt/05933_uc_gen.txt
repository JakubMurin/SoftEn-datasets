1. Faculty Member enters credentials.
2. Credentials are validated by Authentication Service.
3. Search Interface displays search options.
4. Faculty Member inputs search criteria for collection.
5. Database queries with search criteria.
6. Search results are returned to Search Interface.
7. Search Interface displays list of matching collections.
8. Faculty Member selects desired collection.
9. Search Interface requests selected collection details from Database.
10. Collection details are retrieved from Database.
11. Collection content is displayed to Faculty Member.
