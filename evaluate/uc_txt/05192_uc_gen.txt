1. User fournit les informations d'identification.
2. System vérifie les informations fournies.
3. System notifie des erreurs à l'utilisateur.
4. User soumet à nouveau les informations.
5. System confirme l'authentification à l'utilisateur.
6. System informe l'utilisateur de l'échec de l'authentification.
7. Admin demande la radiation d'un compte.
8. System vérifie le compte ciblé.
9. System confirme la radiation à l'Admin.
10. System notifie l'échec de la radiation à l'Admin.
