1. Admin creates Operation Type with its attributes.
2. FrontEnd sends Operation Type creation request to Backoffice API.
3. Backoffice API validates Operation Type attributes.
4. Backoffice API saves the new Operation Type in the Database.
5. Backoffice API logs the creation of the Operation Type.
6. Backoffice API returns confirmation to FrontEnd.
7. FrontEnd returns confirmation to Admin.
