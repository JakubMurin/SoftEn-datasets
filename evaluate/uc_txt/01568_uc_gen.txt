1. User authenticates with the system.
2. Credentials are validated by the Authentication Service.
3. The Database confirms the validity of the credentials.
4. Authentication Service notifies the User of successful authentication.
5. User requests help links from the Help Link Service.
6. Help Link Service fetches contextual links from the Database.
7. Database returns the links to the Help Link Service.
8. Help Link Service displays the fetched links to the User.
