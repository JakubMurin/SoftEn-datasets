1. User opens the game and loads the map and robot position.
2. Game Controller initializes the game state.
3. Game Interface displays the initial map and robot position.
4. User clicks on the edit instructions button.
5. Game Interface enters the instruction editing interface.
6. User selects an instruction.
7. Game Interface adds the selected instruction.
8. User continues selecting and adding instructions.
9. Game Interface requests to continue selecting and adding instructions as User selects.
10. User selects and adds more instructions.
11. Game Interface allows continuing selection and addition of instructions.
12. Game Interface completes the instruction editing.
13. User clicks the start button.
14. Game Interface starts the game.
15. Game Logic begins executing the instruction sequence.
16. Game Controller updates the game status based on instruction execution.
17. User decides to exit the game.
18. Game Interface handles the game exit request.
19. Game Logic manages the game exit.
20. Game Interface confirms successful game exit to the User.
