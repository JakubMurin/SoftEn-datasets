1. The Logged-in User visits the ":GUI Fin. Acc. Details View" and goes to the "recurring transactions" section, clicking on the "+ add order" option.
2. The ":GUI Recurring Transaction Order Create Form" is opened by the user.
3. The user enters the necessary details for the recurring transaction order.
4. A new recurring transaction order, including all user inputs, is created by the system (:Server).
5. The system inserts this new recurring transaction order, along with all user inputs, into the database.
6. The ":GUI Fin. Acc. Details View" is reloaded by the system.
7. A success message is displayed to the user.
