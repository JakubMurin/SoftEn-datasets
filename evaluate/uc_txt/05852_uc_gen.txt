1. Owner requests a sales report from the Frontend Interface.
2. Frontend Interface sends an API call to the Backend System for sales data.
3. Backend System queries the Reporting Module to generate the sales report.
4. Reporting Module fetches the sales data from the Database.
5. Sales data is sent back to the Reporting Module.
6. Reporting Module generates the sales report.
7. Backend System displays the sales report data to the Frontend Interface.
8. Frontend Interface shows the sales report to the Owner.
