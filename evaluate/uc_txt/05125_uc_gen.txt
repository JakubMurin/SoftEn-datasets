1. User writes username in the input field.
2. PacManController updates GUI.
3. PacManController retrieves username from input field.
4. PacManController receives username.
5. PacManController validates the username.
6. PacManUser confirms username validation.
7. PacManController enables the start button.
8. User clicks the start button.
9. Start button click handled by PacManController.
10. PacManController sets the username.
11. PacManController sets the score.
12. Game starts.
13. Game over.
14. PacManController saves highscore.
15. PacManPersistence reads highscores from file.
16. PacManPersistence creates PacManUser object.
17. PacManUser object sent to PacManPersistence.
18. PacManPersistence writes highscores to file.
