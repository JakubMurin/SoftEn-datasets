1. User opens the application.
2. Homepage screen displays "Account Settings".
3. User navigates to Account Settings.
4. Delete Account Dialog is displayed.
5. User confirms cancellation.
6. Account cancellation is initiated.
7. API call for Account Cancellation is made.
8. Cancellation request is forwarded.
9. Account is marked for deletion in the database.
10. Account marked for deletion is confirmed.
11. Account cancellation is confirmed.
12. Cancelling confirmation is displayed.
13. User navigates back to Homepage.
14. Confirmation message is displayed to the User.
15. User logs out and session data is cleared.
