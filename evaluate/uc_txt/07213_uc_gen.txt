2. Batch Submitter requests for Authentication.
3. Batch Submitter provides Credentials to authenticate.
4. EDRMS Website authenticates the User.
5. EDRMS Website requests Ingest Process Docs.

9. Batch Submitter clicks on Documentation Link.
10. Batch Submitter is redirected to Ingest Process Documentation.
