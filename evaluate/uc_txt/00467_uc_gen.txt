1. User requests to record a new activity.
2. Client receives the request from the user.
3. Client sends the data about the new activity to the server.
4. Server receives the data from the client.
5. Server sends the data about the new activity to the database.
6. Database successfully adds the new activity data.
7. Server informs the client that the data is saved.
8. Client confirms to the user that the new activity is successfully recorded.
