1. DigitalRecords Archivist selects an embargoed item.
2. UI requests embargo details from Embargo Management System.
3. Embargo Management System retrieves embargo information from Database.
4. Database returns embargo information to Embargo Management System.
5. Embargo Management System displays embargo details to UI.
6. DigitalRecords Archivist inputs new embargo length.
7. UI sends update embargo length request to Embargo Management System.
8. Embargo Management System updates embargo information in the database.
9. Database confirms the update to Embargo Management System.
10. Embargo Management System displays success message to UI.
11. UI shows confirmation to DigitalRecords Archivist.
