1. Building Manager requests a task through the Phone App.
2. Phone App forwards the task request to the Robot Management System.
3. Robot Management System adds the request to the queue and checks robot availability.
4. If a robot is available, Robot Management System assigns the task to the Robot.
5. Robot cleans the room.
6. Robot Management System informs Phone App that the task has been assigned.
7. Phone App shows a message to the Building Manager.
