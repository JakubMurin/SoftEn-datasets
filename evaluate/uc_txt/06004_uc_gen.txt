1. Manager accesses the inventory management system.
2. UI requests the current inventory list.
3. Backend queries inventory items from the database.
4. Database returns inventory items to the Backend.
5. Backend displays inventory items to the UI.
6. Manager selects an item to update.
7. Manager enters new inventory details.
8. UI sends an update inventory request with details to the Backend.
9. Backend updates the item in the database.
10. Database confirms the update to the Backend.
11. Backend confirms the success to the UI.
12. UI displays the update confirmation to the Manager.
