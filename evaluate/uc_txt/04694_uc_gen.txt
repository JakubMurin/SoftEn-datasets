1. El usuario inicia sesión en la plataforma.
2. La plataforma verifica las credenciales del usuario con la base de datos.
3. La base de datos responde con éxito o error a la plataforma.
4. La plataforma confirma la inicio de sesión al usuario.
5. El usuario solicita contenido personalizado a la plataforma.
6. La plataforma analiza el perfil del usuario en conjunto con el Sistema de Recomendación IA.
7. El Sistema de Recomendación IA obtiene el historial de aprendizaje del usuario desde la base de datos.
8. La base de datos envía el historial al Sistema de Recomendación IA.
9. El Sistema de Recomendación IA envía las recomendaciones a la plataforma.
10. La plataforma muestra el contenido recomendado al usuario.
11. El usuario completa una actividad en la plataforma.
12. La plataforma guarda el progreso de la actividad en la base de datos.
13. La base de datos confirma el guardado a la plataforma.
