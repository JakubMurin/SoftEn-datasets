5. La aplicación solicita al usuario el código de verificación.

7. El usuario ingresa el código de verificación.
8. El sistema de autenticación envía el código de verificación a la aplicación.
9. La aplicación comprueba el código de verificación.
10. La aplicación notifica al usuario que la autenticación fue exitosa.
