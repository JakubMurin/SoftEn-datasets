1. User clicks 'Comenzar' button.
2. App handles the event.
3. StartScreen handles the event.
4. StartButtonEventHandler handles the event.
5. StartScreen sets visibility to false.
6. PlayersInputScreen is displayed to select and confirm inputs.
7. User selects and confirms inputs.
8. PlayersInputScreen handles the event.
9. ConfirmButton handles the event and requests data selection.
10. PlayersInputScreen selects data.
11. Game loads received data.
12. PlayersNamesInputScreen is created.
13. ConfirmButton shows screen for player names input.
14. User enters and confirms player names.
15. PlayersNamesInputScreen handles the event.
16. NamesButton executes event.
17. Game registers players with entered names.
18. QuestionScreen is created.
19. NamesButton shows questions and options.
20. QuestionScreen shows questions for each player.
