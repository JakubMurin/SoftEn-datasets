1. Manager accesses the Menu Management interface.
2. Manager requests the current menu items.
3. Backend Server fetches the menu items from the database.
4. Backend Server displays the menu items to the Manager.
5. Manager selects items to update.
6. Manager submits the updated menu items.
7. Management Interface updates the menu items.
8. Backend Server updates the menu items in the database.
9. Database confirms the successful update.
10. Backend Server confirms the successful update to the Manager.
11. Management Interface displays a success message to the Manager.
