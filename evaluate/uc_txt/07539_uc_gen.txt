1. O User requisita a página dos andares ao SPA.
2. O SPA retorna a página dos andares ao User.
3. O User seleciona um andar da tabela no SPA.
4. O SPA mostra os dados do andar selecionado à API.
5. O SPA faz um pedido à API para obter os edifícios disponíveis.
6. A API mostra os edifícios numa select box ao SPA.
7. O SPA mostra os dados do andar selecionado.
8. O User insere um ficheiro com o floormap do andar selecionado no SPA.
9. O SPA envia um pedido à API para atualizar o floormap.
