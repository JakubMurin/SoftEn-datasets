1. User writes the username in the username input field.
2. PacManController updates the GUI.
3. PacManController retrieves the username from the input field.
4. PacManController receives the username.
5. PacManController validates the username.
6. PacManUser confirms the username is valid.
7. PacManController enables the start button.
8. User clicks the start button.
9. Start button click event is handled by PacManController.
10. PacManController sets the username.
11. PacManController sets the score.
12. Game starts.
13. Game over.
14. PacManController saves the highscore.
15. PacManPersistence reads the highscores file.
16. FileReader retrieves the scores.
17. PacManPersistence creates a PacManUser object.
18. PacManUser is created with the username and score.
19. PacManPersistence writes the new highscore to the file.
