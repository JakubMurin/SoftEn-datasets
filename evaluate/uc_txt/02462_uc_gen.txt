1. Faculty Data Steward accesses the DMPTool through the UI Interface.
2. The UI Interface requests login information from the Authentication Service.
3. The Authentication Service successfully authenticates the Faculty Data Steward.
4. Faculty Data Steward selects "View Roles" from the UI Interface.
5. The UI Interface fetches Roles data from the Role Management Module.
6. Role Management Module requests Roles information from the DMPTool.
7. DMPTool queries the Database for Roles data.
8. The Database provides Roles data to the DMPTool.
9. DMPTool returns Roles information to the Role Management Module.
10. Role Management Module displays the Roles to the UI Interface.
11. The UI Interface shows the Roles sections to the Faculty Data Steward.
