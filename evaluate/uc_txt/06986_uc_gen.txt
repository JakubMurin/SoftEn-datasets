1. User searches and selects the restricted object.
2. Interface requests access details from ACS.
3. Interface displays the "Contact Curator" option.
4. User clicks the "Contact Curator" button.
5. Interface generates an email template to correspond with the curator.
6. User fills in and sends the email request to the curator.
7. Email system forwards the user's request to the curator.
8. Curator reivews the request received from the email system.
9. Curator updates the access permissions in ACS, if request is approved.
10. ACS notifies the User of the access decision.
11. Email system displays the access decision to the User.
