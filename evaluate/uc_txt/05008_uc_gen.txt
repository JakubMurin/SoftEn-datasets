1. Archivist uploads search image to the User Interface.
2. User Interface sends image data to the Web Server.
3. Web Server initiates image search with the Image Search Service.
4. Image Search Service queries the Database for similar images.
5. Database returns matching images to the Image Search Service.
6. Image Search Service prepares the search results using the Results Handler.
7. Results Handler sends the search results back to the User Interface.
8. User Interface displays the search results to the Archivist.
