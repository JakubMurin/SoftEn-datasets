1. User searches for a restricted object and selects it.
2. Repository Interface requests access details from Access Control System.
3. Access Control System displays "Contact Curator" option.
4. User clicks "Contact Curator".
5. Repository Interface generates an email template.
6. User fills in and sends the access request.
7. Email Notification System forwards the user request to Collection Curator.
8. Collection Curator reviews the request.
9. Collection Curator updates access permissions if the request is approved.
10. Access Control System notifies the user of the decision.
11. Email Notification System displays the access decision notification to the user.
