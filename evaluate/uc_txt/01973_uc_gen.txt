1. User initiates the delete process.
2. Confirmation to delete the holiday is shown.
3. Holiday deletion request is sent to holiday.routes.js.
4. Delete request is passed to holiday.controller.js to execute.
5. Holiday deletion action is executed in holiday.model.
6. SQL query to delete holiday data is executed by sql entity.
7. Deletion confirmed in the system.
8. Confirmation response is returned to the user.
