1. O User requisita a página dos andares.
2. O SPA retorna a página dos andares ao User.
3. O User seleciona um andar da lista de andares exibida.
4. O SPA solicita à API os dados do andar selecionado.
5. A API mostra os dados do andar selecionado ao SPA.
6. O SPA solicita a lista de edifícios à API.
7. A API retorna a lista de edifícios ao SPA.
8. O SPA exibe os dados do andar selecionado.
9. O User insere um ficheiro contendo o floormap do andar selecionado.
10. O SPA atualiza o floormap do andar na API ao enviar o ficheiro.
11. A API confirma que o floormap foi atualizado com sucesso.
12. O SPA comunica ao User que a operação foi realizada com sucesso.
