1. Request sales report from frontend interface by the owner.
2. Frontend makes an API call to backend system for sales data.
3. Backend system queries and generates the sales report via reporting module.
4. Reporting module fetches sales data from the database.
5. Database provides the sales data to the reporting module.
6. Reporting module provides the sales report to the backend system.
7. Backend system provides the sales report to the frontend interface.
8. Frontend interface displays the sales report data to the owner.
9. Owner views the displayed sales report.
