1. O GestorCampus requisita a página dos andares.
2. O Sistema de Página de Andares retorna a página dos andares para o GestorCampus.
3. O GestorCampus seleciona um andar na tabela de andares.
4. O Sistema de Página de Andares mostra os dados do andar selecionado.
5. O Sistema de Página de Andares mostra os edifícios disponíveis numa caixa de seleção.
6. O GestorCampus insere um ficheiro com o floormap do andar selecionado.
7. O Sistema de Página de Andares envia o ficheiro para a API para atualização.
