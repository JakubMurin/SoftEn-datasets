1. View landing page.
2. Navigate to Training Reports.
3. Perform Filtering or Category selection as necessary.
4. Display Training Reports.
5. Select Event ID.
6. Display Event Content.
7. Select Edit Event From ... Menu on Event ID.
8. Display Event in Editable View.
9. Make changes to Event By Section.
10. Save As Draft.
11. Save And Continue.
12. Submit Event.
13. Return to Event List.
14. Apply Filtering.
15. Choose Event State.
16. Choose Create session under the ellipsis of an event.
17. Display Training Report - Session page.
18. Make changes to Session By Section.
