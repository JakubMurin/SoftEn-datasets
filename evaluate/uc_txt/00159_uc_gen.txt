1. Caregiver Requests to Enable Step Counter feature for User.
2. Alfred Activates Step Counter.
3. User Receives Message: "Step Counter feature is now available.".
4. User Enables Step Counter.
5. Alfred Initiates Step Counting.
6. User Walks, Alfred Counts Steps.
7. Software Sends Daily Step Data to Health Monitoring Service, which Acknowledges Receipt.
8. Caregiver Requests Step Count Data from Alfred.
9. Alfred Retrieves Step Data from Software.
10. Software Requests User's Step Data from Health Monitoring Service.
11. Health Monitoring Service Sends Step Data to Software.
12. Alfred Displays Step Data.
13. Alfred Shows Step Data to Caregiver.
