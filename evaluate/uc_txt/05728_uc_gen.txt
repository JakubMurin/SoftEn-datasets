1. 用户 opens the mini-program.
2. Mini-program loads the system message module.
3. Mini-program requests system messages.
4. System message module queries unread messages.
5. System message module returns unread message list to mini-program.
6. Mini-program displays unread message notification.
7. User clicks to view messages.
8. User views message details.
9. Mini-program requests message content from system message module.
10. System message module queries message content.
11. System message module returns message content to mini-program.
12. Mini-program displays message content.
