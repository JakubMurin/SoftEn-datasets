1. Curator requests the Dashboard view.
2. Dashboard fetches recent submissions from the SubmissionService.
3. SubmissionService queries the database for submissions by date.
4. Database returns submission data to SubmissionService.
5. SubmissionService processes submission summary.
6. Dashboard prepares the collection widget.
7. CollectionWidget is displayed on the Dashboard with recent submissions.
8. Dashboard shows the dashboard with the widget to the Curator.
