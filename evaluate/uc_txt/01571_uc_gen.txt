1. User authenticates by providing credentials.
2. Authentication Service validates the provided credentials with the Database.
3. Database confirms the validity of the credentials.
4. Authentication Service notifies User of successful authentication.
5. User requests help links from Help Link Service.
6. Help Link Service fetches contextual links from the Database.
7. Database returns the fetched links to Help Link Service.
8. Help Link Service displays the help links to the User.
