1. DataManager requests data type information from DMPTool Interface.
2. DMPTool Interface queries Metadata Repository for stored projects metadata.
3. Metadata Repository returns metadata including data types to DMPTool Interface.
4. DMPTool Interface displays data types to DataManager.
5. DataManager selects a data type.
6. DMPTool Interface queries Support Services Module for required support services.
7. Support Services Module returns a list of basic support services and functionalities to DMPTool Interface.
8. DMPTool Interface displays support services and functionalities to DataManager.
9. DataManager confirms understanding and requirements.
10. DMPTool Interface logs the interaction and requirements confirmation with Metadata Repository.
