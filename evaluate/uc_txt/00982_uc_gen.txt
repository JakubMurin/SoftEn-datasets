1. Faculty member selects the data to upload using the Upload Interface.
2. Data file is uploaded to the Repository System.
3. Repository System stores the data file in the Database.
4. Metadata Service asks for metadata entry form from Repository System.
5. Metadata form is displayed to the Faculty member.
6. Faculty member enters metadata, including publication details in the form.
7. Metadata Service updates metadata in the Database.
8. Repository System confirms data and publication association to Faculty member.
