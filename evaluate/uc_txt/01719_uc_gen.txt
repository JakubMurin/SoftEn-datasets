1. Customer selects "My Orders" from the UI.
2. UI requests list of orders from Backend.
3. Backend fetches customer orders from DB.
4. DB returns orders to Backend.
5. Backend displays orders on UI.
6. UI shows list of orders to Customer.
7. Customer selects a dish from an order on UI.
8. UI requests dish details from Backend.
9. Backend fetches dish details from DB.
10. DB returns dish details to Backend.
11. Backend displays dish details on UI.
12. UI shows dish details to Customer.
13. Customer enters comment and submits on UI.
14. UI posts comment to Backend.
15. Backend saves comment to dish in DB.
16. DB confirms the save.
17. Backend displays success message on UI.
18. UI shows confirmation of comment submission to Customer.
