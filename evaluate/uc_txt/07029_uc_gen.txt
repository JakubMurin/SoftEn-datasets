1. Gracz clicks on a board position.
2. Board processes the move including the current color.
3. Board validates if the selected position is valid.
4. Logic retrieves the color present at the selected position.
5. Logic processes the move by adding a stone at the position for the current color if valid.
6. Server is notified with the move coordinates.
7. Server validates the move and responds to the board with the move acknowledgement.
8. Board updates the state after the move.
