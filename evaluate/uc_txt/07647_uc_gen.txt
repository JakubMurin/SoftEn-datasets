1. Alice adds a new edge agent.
2. Edge Agent 2 provides instructions to Alice for adding Agent 2 and selecting authorizations.
3. Alice adds the agent with the selected authorizations.
4. Edge Agent 1 displays the provisioning code for adding Agent 2.
5. Alice shows the code to Edge Agent 2, who scans it.
6. Edge Agent 2 sends the add request, including authentication and validation keys.
7. Edge Agent 1 stores the validation key of Agent 2 in the wallet.
8. Edge Agent 1 confirms the addition of Agent 2.
9. Agent 1 adds keys and authorizations for Agent 2 to the policy registry.
10. Edge Agent 2 receives and stores data in the wallet from Agent 1.
