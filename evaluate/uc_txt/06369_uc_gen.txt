1. Enhanced Research Data Management Platform prompts the Depositor to link with DMP.
2. Depositor provides the DMP reference to the Enhanced Research Data Management Platform.
3. Data Management Plan Tool requests DMP details from ERDMP.
4. DMP details queries by reference in DMP Repository.
5. DMP details are returned to the Data Management Plan Tool.
6. Data Management Plan Tool provides the DMP details to ERDMP.
7. Dataset is linked with the DMP in Dataset Repository.
8. DMP is updated with dataset link in DMP Repository.
9. ERDMP confirms the linkage between the dataset and DMP.
