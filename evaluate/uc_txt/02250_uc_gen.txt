1. CS receives request for help from End-User.
2. CS collects and forwards NIN from End-User to Application.
3. Application processes NIN and instructs End-User to start Swedish BankID process.
4. End-User starts BankID process.
5. End-User authenticates with BankID via IDP.
6. IDP sends authentication information to Signicat.
7. Signicat processes the authentication information.
8. Application instructs Customer to get authentication result from Signicat.
9. Application polls Signicat for authentication result.
10. Signicat sends authentication success response to Application.
11. CS provides the authentication result to End-User.
12. CS assists the authenticated End-User as needed.
