1. Moderator navigates to Account Settings.
2. Web Interface receives the request to view current account details.
3. Backend Server fetches account details from the Account Database.
4. Account Database returns the requested account details to the Backend Server.
5. Backend Server displays the current account details to the Web Interface.
6. Web Interface shows the account details form to the Moderator.
7. Moderator inputs updated account details.
8. Web Interface submits the updated account details to the Backend Server.
9. Backend Server updates the account details in the Account Database.
10. Account Database confirms the details have been successfully updated to the Backend Server.
11. Backend Server acknowledges the successful account details update to the Web Interface.
12. Web Interface displays the update success message to the Moderator.
