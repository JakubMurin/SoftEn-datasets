1. Member navigates to the login page.
2. Web Interface prompts for credentials to authenticate.
3. Member enters the login credentials.
4. Authentication System validates the credentials in Database.
5. Authentication System notifies Web Interface about access grant.
6. Web Interface displays the dashboard to the Member.
7. Member navigates to reserve a book through Web Interface.
8. Web Interface presents the search interface of Reservation System.
9. Member inputs search parameters.
10. Reservation System queries the Database.
11. Database returns the query results to Reservation System.
12. Reservation System displays the available books using Web Interface.
13. Member selects a book to reserve through the Web Interface.
14. Web Interface starts processing the reservation request with Reservation System.
15. Database updates the reservation status based on the request.
16. Reservation System confirms the successful reservation.
17. Web Interface displays the reservation confirmation to the Member.
18. Member receives a success message for the reservation.
