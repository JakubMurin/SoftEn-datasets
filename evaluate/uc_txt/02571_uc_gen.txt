1. User initiates tagging process.
2. Web Interface requests available tags.
3. Tagging System queries Controlled Vocabulary DB for tags.
4. Controlled Vocabulary DB returns vocabulary list to Tagging System.
5. Tagging System displays available vocabulary tags in Web Interface.
6. User selects tags from controlled vocabulary list.
7. Web Interface assigns selected tags to dataset/file.
8. Tagging System validates and saves assigned tags.
9. Controlled Vocabulary DB confirms tag assignment.
10. Tagging System confirms successful tagging to the user.
11. Web Interface displays tagging success message to the user.
