1. Content Editor enters search query (keywords/tags).
2. Search Interface queries for content.
3. Search Interface displays search results to Content Editor.
4. Content Editor selects content from results.
5. Search Interface requests selected content from CMS Database.
6. CMS Database retrieves content details.
7. Search Interface displays content in editor (Content Page).
8. Content Editor edits content on Content Page.
9. Content Page saves changes to CMS Database.
10. CMS Database confirms changes saved.
11. Content Page displays confirmation of successful edit.
