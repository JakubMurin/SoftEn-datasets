1. User selects option to create new pipeline.
2. Web UI displays pipeline creation form.
3. User fills in pipeline details and specifies new dataset instance.
4. Pipeline initiates pipeline and dataset creation.
5. Pipeline requests dataset type check.
6. Dataset Management Service verifies dataset type and version.
7. Version Control provides dataset type specifications.
8. Dataset Management Service confirms dataset type.
9. Pipeline requests creation of new dataset instance.
10. Dataset Instance Creator creates new dataset instance.
11. Version Control registers new dataset instance version.
12. Dataset Instance Creator confirms dataset instance creation.
13. Pipeline confirms successful pipeline and dataset instance creation.
14. Web UI displays success message and pipeline details.
