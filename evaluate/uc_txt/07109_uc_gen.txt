1. Depositor triggers access to repository information through Web Interface.
2. Web Interface receives the request and sends it to EDRMS for accepted items and formats retrieval.
3. EDRMS receives the request from the Web Interface and starts interacting with Format and Type Management.
4. Format and Type Management fetches the list of item types and recommended formats from EDRMS.
5. EDRMS relays the acquired data to the Web Interface for display purposes.
6. Web Interface presents the fetched information to Depositor showcasing the accepted item types and formats.
