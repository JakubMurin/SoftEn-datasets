1. Faculty Member selects the data file to upload.
2. Faculty Member initiates the file upload process.
3. The data file is stored in the digital repository.
4. File metadata, including uploader information, is saved in the database.
5. The file upload success is displayed to the Faculty Member.
6. The file ID and upload confirmation are provided to the Faculty Member.
7. Faculty Member chooses the licensing option (CC-By or CC0).
8. The license choice and file ID are sent to the Licensing Service.
9. The chosen license is applied to the file by the DRM Service.
10. The file metadata is updated with the license information by the Repository.
11. File record is updated with license metadata in the database.
12. License application to the file is confirmed by the Repository.
13. The license application success is displayed to the Faculty Member.
14. Confirmation that the license has been set is provided to the Faculty Member.
