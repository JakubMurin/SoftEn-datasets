1. User selects the file version revert option.
2. User requests to revert the file by providing fileID and version.
3. Backend service fetches the older file version based on the provided fileID and version.
4. Backend service updates the current file with the older version.
5. File Storage confirms the update.
6. Backend service confirms the successful revert to the user.
7. User interface displays a success message for the file revert operation.
