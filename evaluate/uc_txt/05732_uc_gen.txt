1. 用户 opens the mini program.
2. Mini program loads the system message module.
3. Mini program requests to retrieve system messages.
4. System message module queries unread messages.
5. System message module returns the list of unread messages to the mini program.
6. Mini program displays the unread message notification.
7. User clicks to view the messages.
8. User views message details.
9. Mini program requests to get the message content.
10. System message module queries the message content.
11. System message module returns the message content to the mini program.
12. Mini program displays the message content.
