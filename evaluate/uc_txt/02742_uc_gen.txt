1. User A initiates a transaction on the Polkadot Ledger through API Client.
2. API Client forwards the transaction request to API Server.
3. API Server invokes the transact() method on the Polkadot Ledger Connector.
4. Polkadot Ledger Connector processes the transaction.
5. API Server sends back a response to API Client.
6. API Client formats and presents the response to User A.
