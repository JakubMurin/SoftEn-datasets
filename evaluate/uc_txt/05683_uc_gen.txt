1. Waiter selects order to review.
2. Waiter requests order details from the Waiter Interface.
3. Order Management System fetches the order items from the Menu Database.
4. Menu Database returns the order items to the Order Management System.
5. Order Management System requests allergy warnings for the order items from the Allergy Information System.
6. Allergy Information System looks up allergy information in the Menu Database.
7. Menu Database returns the allergy data to the Allergy Information System.
8. Allergy Information System compiles allergy warnings for the order items.
9. Order Management System sends the compiled allergy warnings to the Waiter Interface.
10. Waiter Interface displays the allergy warnings to the Waiter.
