1. Developer requests to create dataset package.
2. Archetype Tool generates dataset template and provides it to the developer.
3. Developer defines dataset type (Schemas, Configurations, Code).
4. Developer uploads dataset package to the Build System for validation.
5. Validate package integrity and compatibility.
6. Store validated dataset package in the Dataset Repository and acknowledge storage to the developer.
7. Notify successful packaging and storage to the developer.
8. Return error message to the developer if validation fails.
9. Deploy dataset type for application integration in the Ecosystem.
10. Integrate dataset type in the Application Ecosystem and confirm successful deployment and integration to the developer.
