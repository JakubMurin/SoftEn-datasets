1. Building Manager requests cleaning task through Phone App.
2. Phone App forwards request to Robot Management System.
3. Robot Management System adds request to queue and checks robot availability.
4. If robot is available:.
5. Robot Management System assigns task to Robot.
6. Robot cleans the room.
7. Phone App receives task assigned message from Robot Management System.
8. If robot is not available:.
9. Robot Management System estimates wait time and informs Phone App.
10. Phone App shows message to Building Manager.
