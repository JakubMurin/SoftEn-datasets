1. User sends an HTTP request to the Finance App.
2. Finance App forwards the SQL request to the Database.
3. Database processes the SQL request and sends a response to the Finance App.
4. Finance App sends an HTTP response back to the User.
5. User sends an Authentication request to the Finance App.
6. If valid credentials:.
7. Finance App responds with an Authentication success message.
8. If invalid credentials:.
9. Finance App responds with an Invalid credentials message.
10. Finance App sends an Authentication response to the User.
