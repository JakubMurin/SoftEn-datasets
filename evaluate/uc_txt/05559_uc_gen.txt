1. User configures image resolution for the device.
2. Mobile App sets the image resolution for the device.
3. Mobile App passes the CID of the device.
4. Server finds active TCP connection.
5. Server notifies Mobile App of device unavailability.
6. Mobile App displays information about device unavailability to User.
7. Server sets configuration on ESP32.
8. ESP32 presents the configuration.
9. Server sends the current device configuration to Mobile App.
10. Mobile App updates the view.
11. Mobile App displays information to User about successful settings update.
