1. Librarian selects "Late Returns" option.
2. Web Interface queries overdue books from Database.
3. Database returns list of overdue books to Web Interface.
4. Web Interface displays the list of overdue books to Librarian.
5. Librarian requests to send reminders.
6. Web Interface triggers overdue notices to Notification System.
7. Notification System looks up borrower contact info from Database.
8. Database returns contact info to Notification System.
9. Notification System confirms notifications sent to Librarian.
