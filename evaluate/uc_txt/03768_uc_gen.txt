1. Customer requests to create an account.
2. Portal UI displays account creation form to the customer.
3. Customer submits account details (username, email, password).
4. Authentication System validates account details.
5. Authentication System confirms details are valid.
6. Portal UI creates a new user account in the database.
7. Accounts Database confirms account is created.
8. Portal UI displays account creation success to the customer.
9. Portal UI initiates first-time login sequence with Authentication System.
10. Authentication System requests authentication (login) from the customer.
11. Customer provides credentials to the Authentication System.
12. Authentication System validates customer credentials.
13. Accounts Database confirms customer credentials are valid.
14. Authentication System informs customer of successful authentication.
