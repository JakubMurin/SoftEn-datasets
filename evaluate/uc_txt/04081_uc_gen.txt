1. Access Update Center Form - Actor Admin interacts with the UI to access the form for updating Recycling Center information.
2. Display Form - Admin interface displays the form for entering Recycling Center data to Admin.
3. Input Recycling Center Data - Admin inputs the Recycling Center data.
4. Submit New Center Info - Admin submits the new Recycling Center information via UI to the Server.
5. Validate and Add Info - Server validates and adds the new Recycling Center information to the Database.
6. Confirmation - Database sends a confirmation to the Server.
7. Display Success Message - Server displays a success message to the UI.
8. Show Confirmation - UI shows the confirmation message to the Admin.
