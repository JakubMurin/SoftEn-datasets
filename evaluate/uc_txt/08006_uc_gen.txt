1. Depositor accesses the repository information through the Web Interface.
2. Web Interface requests accepted items and formats from the EDRMS.
3. EDRMS retrieves item types and formats from the Format and Type Management system.
4. Format and Type Management system returns types and recommended formats to EDRMS.
5. EDRMS displays the information.
6. Web Interface shows accepted item types and formats to the Depositor.
