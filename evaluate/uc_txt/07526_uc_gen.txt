1. Student selects exam to take.
2. System displays list of available exams.
3. Student selects an exam from the list.
4. System validates if selected exam is open for submission.
5. If yes, system builds the exam for the student to take.
6. Student progresses through exam by answering questions.
7. System validates each question submitted by the student.
8. System calculates exam grade percentage.
9. System saves the completed exam grade.
10. System provides feedback and grade to student.
