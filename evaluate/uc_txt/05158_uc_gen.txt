1. El usuario selecciona iniciar la construcción de un edificio.
2. El sistema verifica si se puede costear la construcción.
3. El sistema crea una construcción tentativa.
4. El usuario visualiza la construcción tentativa.
5. El usuario decide si se puede construir o no.
6. El usuario mueve el centro de la construcción a una nueva posición.
7. El usuario mueve nuevamente el centro de la construcción a otra posición.
8. El sistema verifica si se puede construir la edificación.
9. El sistema construye el edificio.
