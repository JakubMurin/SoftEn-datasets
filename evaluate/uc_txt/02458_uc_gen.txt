1. Faculty Data Steward accesses DMPTool through the UI Interface.
2. UI Interface requests login from Authentication Service.
3. Authentication Service authenticates the user successfully.
4. Faculty Data Steward selects "View Roles" in the UI Interface.
5. UI Interface fetches Roles data from Role Management Module.
6. Role Management Module requests Roles information from DMPTool.
7. DMPTool queries Roles data from the Database.
8. Database provides Roles data to DMPTool.
9. DMPTool returns Roles information to Role Management Module.
10. Role Management Module displays Roles information in the UI Interface.
11. UI Interface shows the Roles sections to the Faculty Data Steward.
