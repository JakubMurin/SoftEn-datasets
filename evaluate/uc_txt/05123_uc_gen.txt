1. Publisher requests preview of Data Package from UI Interface.
2. UI Interface verifies Publisher's identity with Publisher Account.
3. Publisher Account confirms Publisher's identity.
4. UI Interface requests Data Package preparation from Data Package Service.
5. Data Package Service validates the Data Package with Validation Service.
6. Validation Service provides validation results to Data Package Service.
7. Data Package is validated and ready for Preview.
8. Data Package Service sends validated Data Package to Preview Service.
9. Preview Service generates Preview Link for the Data Package.
10. Preview Link is provided to UI Interface by Data Package Service.
11. UI Interface displays Preview Link to Publisher.
