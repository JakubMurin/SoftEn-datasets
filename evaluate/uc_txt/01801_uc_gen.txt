1. Building Manager requests a task through the Phone App.
2. Phone App forwards the task request to the Robot Management System.
3. Robot Management System adds the task request to the queue.
4. Robot Management System checks the availability of the robot.
5. If the robot is available:.
6. Robot Management System assigns the cleaning task to the Robot.
7. Robot cleans the room.
8. Robot Management System sends a task assigned message to the Phone App.
9. Phone App shows the task assigned message to the Building Manager.
10. If the robot is not available:.
11. Robot Management System estimates the wait time.
12. Robot Management System sends an estimated wait time message to the Phone App.
