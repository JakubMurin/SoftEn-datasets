1. El Sistema o Usuario inicia la construcción.
2. El Sistema o Usuario inicia la construcción de un elemento construible.
3. La Fase de Construcción verifica si puede costear el elemento construible.
4. La Fase de Construcción confirma que puede costear el elemento construible.
5. La Fase de Construcción crea una construcción tentativa con el elemento construible.
6. La Construcción Tentativa confirma la creación de la construcción.
7. La Fase de Construcción informa al Sistema o Usuario sobre la construcción tentativa.
8. El Sistema o Usuario verifica si la construcción tentativa se puede realizar.
9. La Construcción Tentativa informa al Sistema o Usuario que la construcción no se puede realizar.
10. El Sistema o Usuario muestra un mensaje de error al usuario.
11. El Sistema o Usuario mueve el centro de la construcción a una posición.
12. El Sistema o Usuario mueve el centro de la construcción a otra posición.
13. El Sistema o Usuario verifica si la construcción tentativa se puede realizar.
14. La Construcción Tentativa informa al Sistema o Usuario que la construcción se puede realizar.
15. El Sistema o Usuario ordena la construcción del elemento.
