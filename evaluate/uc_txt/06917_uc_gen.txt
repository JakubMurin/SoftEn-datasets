1. Content Editor enters search query (keywords/tags).
2. Search Interface queries for content from the CMS Database.
3. CMS Database retrieves search results.
4. Search Interface displays search results to Content Editor.
5. Content Editor selects content from the search results.
6. Search Interface requests selected content details.
7. CMS Database retrieves and sends content details back to Search Interface.
8. Search Interface displays the selected content in a Content Page editor.
9. Content Editor edits the content in the editor.
10. Content Page saves changes, which are confirmed by the CMS Database.
11. Content Editor receives a confirmation that the edit was successful.
