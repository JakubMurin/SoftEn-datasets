1. Staff identifies missing documents on the application detail page.
2. Application detail page fetches missing documents from the backend server.
3. Backend server queries the database for missing documents.
4. Database returns the missing documents to the backend server.
5. Backend server notifies the staff user system about the missing documents.
6. Notification system confirms the notifications have been sent.
7. Backend server confirms the notifications have been sent to the application detail page.
8. Application detail page displays a confirmation to the staff user.
