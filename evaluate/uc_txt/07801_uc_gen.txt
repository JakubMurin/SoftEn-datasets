1. FAE User selects 'Change' button on the View Account (Payment Terms tab) UI Page.
2. View Account (Payment Terms tab) navigates to Change Payment Terms UI Page.
3. FAE User inputs data and clicks 'Save changes' button on the Change Payment Terms UI Page.
4. Change Payment Terms UI Page calls API with Defendant Account ID and Payload (JSON).
5. API creates Defendant Account Payment Terms in Payment Terms DB.
6. View Account (Payment Terms tab) displays Payment Terms tab.
