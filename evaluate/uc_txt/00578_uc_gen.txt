1. User decides to pay with USDT.
2. UI requests payment information from Gateway.
3. Gateway provides USDT deposit address and amount to User, including QR code.
4. User transfers USDT to the Blockchain.
5. Blockchain detects the transaction.
6. The Blockchain requests transaction confirmation from the Monitor.
7. Blockchain updates transaction status upon Monitor's request.
8. Monitor sends a notification of final transaction approval to the Gateway.
9. Gateway stores transaction and user information in Database.
10. Database confirms payment and converts it to user points.
