1. User initiates the tagging process through Web Interface.
2. Web Interface requests available tags from Tagging System.
3. Tagging System queries Controlled Vocabulary DB for the required tags.
4. Controlled Vocabulary DB returns the vocabulary list to Tagging System.
5. Tagging System displays the vocabulary tags to Web Interface.
6. User selects tags from the controlled vocabulary.
7. Web Interface assigns the selected tags to the dataset/file using Tagging System.
8. Tagging System validates and saves the tags via Controlled Vocabulary DB.
9. Controlled Vocabulary DB confirms the successful tag assignment to Tagging System.
10. Tagging System confirms the successful tagging to the User.
11. Web Interface displays a success message to the User.
