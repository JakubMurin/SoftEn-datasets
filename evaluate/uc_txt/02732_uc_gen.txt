1. The CLI Tool validates the JSON schema of the data package.
2. If validation fails, the Validation Service returns validation errors to the CLI Tool.
3. If validation succeeds, the Validation Service informs the CLI Tool of the successful validation.
4. The CLI Tool generates a preview of the data package views.
5. The Preview Service displays the generated preview to the CLI Tool.
6. The CLI Tool shows the preview output to the Publisher.
